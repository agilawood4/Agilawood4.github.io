组合导航

由于笔者将来读研可能要做导航定位方向，遂补充该方面相关知识，将会不定期整理更新（）

导航定位原理分类：

  • 空间交汇测量（距离，角度）：全站仪，GPS
  • 数据库匹配导航：地标定位，地形匹配
  • 航位推算（Dead Reckoning）：磁罗盘 + 里程计，惯性导航

惯性测量单元 IMU

  • IMU = 三轴加速度计 + 三轴陀螺，加速度计测量相互正交的三个方向上的比力，陀螺测量相互正交的三个方向上的角速度
  • 通常三轴陀螺和三轴加速度计的轴向是平行对齐的
  • 传感器与载体固定连接，载体系原点为三周加速度计的交点

IMU，ISA，INS 概念区分：

  • ISA，Interial Sensor Assembly：三轴陀螺 + 三轴加速度计，输出原始传感器数据
  • IMU，Interial Measurement Unit：ISA 经误差标定补偿（零偏和比力因子误差等）和数据转换，输出补偿后的数据
  • INS，Interial Navigation System：IMU + 惯性导航算法，输出位置，速度，姿态角等导航状态量，以及补偿后的传感器数据

惯性导航中常见的四个坐标系

实用惯性坐标系（i-frame）：各轴方向不随地球自转而改变

  • 原点：地球质心
  • z 轴：沿地球自转轴方向，从地心指向北极
  • x 轴：在赤道平面内，从地心指向春分点（宇宙中一个固定位置）
  • y 轴：和 x，z 轴构成右手系

地心地固坐标系（ECEF，e-frame）：各轴方向随地球自转而改变

  • 原点：地球质心
  • z 轴：沿地球自转轴方向，从地心指向北极点
  • x 轴：在赤道平面内，从地心指向赤道与本初子午线的交点
  • y 轴：位于赤道平面内，与 x，z 轴构成右手坐标系

导航坐标系（n-frame）：又称为当地水平坐标系，地理坐标系

  • 原点：载体中心或参考点
  • z 轴：沿参考椭球的法线方向向下
  • x 轴：参考椭球北向
  • y 轴：参考椭球东向

常用的导航坐标系有北-东-下（N-E-D）和东-北-天（E-N-U）两种，本课程选择前者

IMU 坐标系（b-frame）

  • 原点：IMU 测量中心
  • x 轴：x 轴加速度计和陀螺仪正方向
  • y 轴：y 轴加速度计和陀螺仪正方向
  • z 轴：z 轴加速度计和陀螺仪正方向

常用的 IMU 坐标系有前-右-下（F-R-D，对应 N-E_D）和右-前-上（R-F-U，对应 E-N-U）两种，本课程选择前者

从左到右展示的误差类型分别为：

  • Quantization Noise：量化噪声，斜率为 -1
  • Rate White Noise：角速率白噪声，斜率为 -1/2
  • Correlated Noise：相关噪声，斜率结余 -1/2 到 0 之间的过渡区
  • Sinusoidal：正弦噪声，曲线上出现局部凸起的小鼓包
  • Bias Instability：零偏不稳定性，斜率为 0，曲线最低点
  • Rate Random Walk：速率随机游走，斜率为 +1/2
  • Rate Ramp：速率斜坡，斜率为 ±1

Allan 方差只关心时间为 rrr 尺度时误差的变化：

  • rrr 以内误差的变化通过对每个块内取平均被消除
  • rrr 以外误差的变化通过仅对相邻时间块作差被消除（相当于仅关心每一个 2r2r2r 区间内前半段和后半段误差的变化）

惯导的初始对准

姿态角定义

姿态角定义：

  • yaw 航向角 / 偏航角（ψ\psiψ）：描述载体头朝向哪个方向，当地正北为 0°，北偏东为正方向，绕 z 轴旋转
  • pitch 俯仰角（θ\thetaθ）：描述载体抬头与低头的程度，上仰为正，下俯为负，绕 y 轴旋转
  • roll 横滚角（ϕ\phiϕ）：描述载体左右倾斜的程度，右翼下压为正，绕 x 轴旋转

姿态角描述的是 b-frame 相对于 n-frame 的旋转关系，即载体本体相对于当地的北东地（N-E-D）坐标系旋转的角度，标准旋转顺序为：

Cnb=Rx(ϕ)Ry(θ)Rz(ψ)C_n^b=R_x(\phi)R_y(\theta)R_z(\psi) Cnb​=Rx​(ϕ)Ry​(θ)Rz​(ψ)

即先绕 z 轴转航向角（yaw），再绕旋转后新的 y 轴转俯仰角（pitch），最后绕旋转后新的 x 轴转横滚roll）

和航向角 yaw 近似的两个概念：

  • Heading：航向角：飞行器机头指向相对于正北的夹角，北偏东（顺时针）方向为正
  • Azimuth：方位角：和 Heading 含义接近，二者都近似等于 yaw
角度 描述对象 参考系 常见范围
Roll 左右倾斜 b vs n ±180°
Pitch 抬头低头 b vs n ±90°
Yaw 机头水平转向 b vs n ±180° 或 0~360°
Heading 载体朝向/真北或磁北 水平面 0~360°
Azimuth 目标方位或初始方位 水平面/真北 0~360°

静态粗对准

水平面确定

加速度计调平（Accelerometer Leveling）：通过感应重力加速度找到水平面

以俯仰角为例（横滚角同理）：假设载体头上仰角为 θ\thetaθ

ax=g⋅sin⁡θ⇒θ=arcsin⁡axg≈axga_x = g\cdot\sin\theta\Rightarrow \theta = \arcsin\frac{a_x}{g}≈\frac{a_x}{g} ax​=g⋅sinθ⇒θ=arcsingax​​≈gax​​

这是理想情况，而实际测量时，除去加速度在 xxx 轴的分量，还会额外引入一个测量误差 δax\delta a_xδax​

a^x=g⋅sin⁡θ+δax⇒θ^=arcsin⁡a^xg≈a^xg=θ+δaxg\hat{a}_x = g\cdot \sin\theta + \delta a_x\Rightarrow \hat{\theta} = \arcsin\frac{\hat{a}_x}{g}≈\frac{\hat{a}_x}{g}=\theta+\frac{\delta a_x}{g} a^x​=g⋅sinθ+δax​⇒θ^=arcsinga^x​​≈ga^x​​=θ+gδax​​

因此则得到

θ^=θ+δaxg\hat{\theta}=\theta+\frac{\delta a_x}{g} θ^=θ+gδax​​

即对于 θ\thetaθ 的测量误差 δθ\delta\thetaδθ 满足

δθ=δaxg\delta\theta=\frac{\delta a_x}{g} δθ=gδax​​

这是一个通用的结论，即姿态测量角的误差 δθ\delta\thetaδθ 就等于所使用的加速度计的测量误差 δax\delta a_xδax​，除以当地的重力加速度 ggg，更抽象来讲，某个姿态角的误差，就等于传感器测量的误差，除以对应参考的向量的幅度

航向角确定

陀螺罗盘（Gyro Compassing）：通过感应地球自转角速度找到航向

地球自转的角速度 ω\omegaω 和地轴平行，假设当地的纬度为 ϕ\phiϕ，则其投影到正北方向的分量为 ω⋅cos⁡ϕ\omega\cdot\cos \phiω⋅cosϕ，其投影到 Down 方向的分量为 ω⋅sin⁡ϕ\omega\cdot \sin \phiω⋅sinϕ，投影到正东方向的分量为 000，这些分量都可以由陀螺仪测量得到

ωxb=ωe⋅cos⁡ϕ⋅cos⁡Aωyb=−ωe⋅cos⁡ϕ⋅sin⁡A\omega_x^b=\omega_e\cdot\cos \phi\cdot\cos A\\ \omega_y^b=-\omega_e\cdot\cos \phi\cdot \sin A ωxb​=ωe​⋅cosϕ⋅cosAωyb​=−ωe​⋅cosϕ⋅sinA

其中 ωe\omega_eωe​ 是地球自旋转的角速度，ψ\psiψ 是载体所在地的纬度

陀螺仪角度误差分析

（1）由于陀螺仪零偏误差 bbb 而带来的角度测量误差

δA=bωecos⁡ψ\delta A=\frac{b}{\omega_e\cos\psi} δA=ωe​cosψb​

（2）由于陀螺仪白噪声误差而带来的角度测量误差

陀螺仪在工作时会产生一种在频域上功率均匀分布的噪声，成为白噪声，记作 ε(t)\varepsilon(t)ε(t)，白噪声的均值为 0，不同时刻之间完全不相关

RMS，Root Mean Square 为均方根，是用于描述随机噪声平均幅度的统计量：

RMS=E{ε2(t)}RMS=\sqrt{E\{\varepsilon^2(t)\}} RMS=E{ε2(t)}​

陀螺仪测量的真实角速率是 ω(t)\omega(t)ω(t)，而实际的输出待噪声 ε(t)\varepsilon(t)ε(t)，因此陀螺仪实际显示的测量结果为：

ω~(t)=ω(t)+ε(t)\tilde{\omega}(t) = \omega(t) + \varepsilon(t) ω~(t)=ω(t)+ε(t)

要得到角度，则需要对角速率积分：

θ(t)=∫otω~(τ)dτ=∫0tω(τ)dτ+∫0tε(τ)dτ\theta(t)=\int_o^t\tilde{\omega}(\tau)d\tau=\int_0^t\omega(\tau)d\tau + \int_0^t\varepsilon(\tau)d\tau θ(t)=∫ot​ω~(τ)dτ=∫0t​ω(τ)dτ+∫0t​ε(τ)dτ

真实角度加上噪声的积分，其中噪声积分项为：

Δθ=∫0tε(τ)dτ\Delta\theta=\int_0^t\varepsilon(\tau) d\tau Δθ=∫0t​ε(τ)dτ

白噪声的积分称为随机游走，即每一步随机往正或者负方向走一点，时间越长偏离越远，其标准差为：

σΔθ(t)=q⋅t=ARW⋅t\sigma_{\Delta\theta}(t)=\sqrt{q\cdot t}=\text{ARW}\cdot\sqrt{t} σΔθ​(t)=q⋅t​=ARW⋅t​

其中 ARW=q\text{ARW}=\sqrt{q}ARW=q​（q 是功率谱密度），单位是 °h°\sqrt{h}°h​，因此 ARWARWARW 的本质就是白噪声积分后角度误差随时间增长的速率，比如 ARW=0.002°/h\text{ARW}=0.002°/\sqrt{h}ARW=0.002°/h​ 则意味着：

  • 积分 1 小时，角度误差标准差为 0.002°
  • 积分 4 小时，角度误差标准差为 0.004°
  • 积分 t 小时，角度误差标准差为 0.002t\sqrt{t}t​°

寻北对准需要再积分的基础上，对静止段陀螺的输出取时间均值：

计算这个均值噪声的 RMS\text{RMS}RMS：

RMSωˉ=E{δωˉ2}\text{RMS}_{\bar{\omega}}=\sqrt{E\{\delta\bar{\omega}^2\}} RMSωˉ​=E{δωˉ2}​

代入上面的公式：

σδθ(Ta)=q⋅Ta\sigma_{\delta\theta}(T_a) = \sqrt{q\cdot T_a} σδθ​(Ta​)=q⋅Ta​​

则有：

RMSωˉ=qTa=ARWTa\text{RMS}_{\bar{\omega}} = \sqrt{\frac{q}{T_a}} = \frac{\text{ARW}}{\sqrt T_a} RMSωˉ​=Ta​q​​=T​a​ARW​

这是一个相当重要的结论，从直观上理解就是：平均时间越长，均值估计越稳定，剩余噪声越小，且按 1Ta\frac{1}{\sqrt{T_a}}Ta​​1​ 衰减，也就是说，器件的 ARW\text{ARW}ARW 是固定的，通过延长平均时间 TaT_aTa​ 来压低 RMS\text{RMS}RMS

而 RMS 可以作为陀螺仪的动态误差项，代入角度误差计算公式，即得到因为白噪声误差而带来的角度测量误差：

δA=RMSωecos⁡ψ=ARWTaωecos⁡ψ\delta A = \frac{\text{RMS}}{\omega_e\cos\psi}=\frac{\text{ARW}}{\sqrt T_a\omega_e\cos\psi} δA=ωe​cosψRMS​=T​a​ωe​cosψARW​

静基座航向角对准影响因素

静基座对准的航向角精度取决于：

  • 陀螺东向轴的零偏
  • 纬度 ψ\psiψ
  • 陀螺随机角度游走（ARW，影响 RMS 的分子）
  • 对准时间（影响 RMS 的分母），延长对准时间，可以使得动态误差的量级对齐静态误差的量级

姿态矩阵原始解

求解静态粗对准的姿态矩阵，首先需要用到两个已知向量：重力向量 ggg 和地球自转角速度 ωie\omega_{ie}ωie​，它们在 b 系和 n 系下都有已知表达

  • 在 b 系：gbg^bgb 是加速度计的实际输出，三个轴都有分量
