дёҠиҝ°д»Јз ҒеҚіе®ҡд№үдәҶдёҖдёӘиҫ“е…Ҙдёә 5 дёӘзҘһз»Ҹе…ғпјҢиҫ“еҮәдёә 3 дёӘзҘһз»Ҹе…ғзҡ„е…ЁиҝһжҺҘеұӮпјҢйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢеҰӮжһңжҲ‘д»¬жү“еҚ° linear.weightпјҢдјҡеҸ‘зҺ°жү“еҚ°еҮәжқҘзҡ„жҳҜдёҖдёӘ [3, 5] зҡ„зҹ©йҳөпјҢиҝҷжҳҜз”ұдәҺ Pytorch еә•еұӮеӯҳеӮЁжңәеҲ¶жүҖеҜјиҮҙзҡ„пјҢе®һйҷ…дёҠеңЁеҸӮдёҺи®Ўз®—ж—¶пјҢlinear.weight зҡ„еҪўзҠ¶е°ұжҳҜ [5, 3]пјҢе’ҢжҲ‘д»¬зҡ„е®ҡд№үдёҖиҮҙ

Pytorch дёӯжүҖжңүеҲқе§ӢеҢ–ж–№жі•йғҪеңЁ nn.init еҢ…дёӯпјҢеҪ“жҲ‘д»¬з”ҹжҲҗдёҖдёӘ nn.Linear ж—¶пјҢдјҡеҸ‘зҺ°е…¶ weight е’Ң bias еҸӮж•°д№ҹжҳҜиў«еҲқе§ӢеҢ–иҝҮзҡ„пјҢиҝҷйҮҢй»ҳи®ӨйҮҮз”Ёзҡ„жҳҜ nn.init.kaiming_uniform_ еҲқе§ӢеҢ–ж–№жі•

еңЁзҺ°жңү Pytorch жһ¶жһ„дёӯпјҢеҺҹжңүзҡ„ ones(), zeros(), constant() зӯүж–№жі•е·Із»ҸеәҹејғпјҢеҸ–иҖҢд»Јд№Ӣзҡ„жҳҜеёҰжңүдёӢеҲ’зәҝзҡ„ж–№жі•пјҢеҠҹиғҪе®Ңе…ЁзӣёеҗҢ

жіЁж„Ҹпјҡе°ҶжқғйҮҚеҲқе§ӢеҖји®ҫдёә 0 е°Ҷж— жі•жӯЈзЎ®иҝӣиЎҢеӯҰд№ пјҢдёҘж јең°иҜҙпјҢдёҚиғҪе°ҶжқғйҮҚеҲқе§ӢеҖји®ҫжҲҗдёҖж ·зҡ„еҖјпјҢеӣ дёәиҝҷж„Ҹе‘ізқҖеҸҚеҗ‘дј ж’ӯж—¶жқғйҮҚе…ЁйғЁйғҪдјҡиҝӣиЎҢзӣёеҗҢзҡ„жӣҙж–°пјҢиў«жӣҙж–°дёәзӣёеҗҢзҡ„еҖјпјҲеҜ№з§°зҡ„еҖјпјүпјҢиҝҷдҪҝеҫ—зҘһз»ҸзҪ‘з»ңжӢҘжңүи®ёеӨҡдёҚеҗҢзҡ„жқғйҮҚзҡ„ж„Ҹд№үдё§еӨұдәҶпјҢдёәдәҶйҳІжӯў вҖңжқғйҮҚеқҮдёҖеҢ–вҖқпјҲз“Ұи§ЈжқғйҮҚзҡ„еҜ№з§°з»“жһ„пјүпјҢеҝ…йЎ»йҡҸжңәз”ҹжҲҗеҲқе§ӢеҖј

Xavier еҲқе§ӢеҢ–пјҲGlorot еҲқе§ӢеҢ–пјү

Xavier еҲқе§ӢеҢ–ж №жҚ®иҫ“е…Ҙе’Ңиҫ“еҮәзҡ„зҘһз»Ҹе…ғж•°йҮҸи°ғж•ҙжқғйҮҚзҡ„еҲқе§ӢиҢғеӣҙпјҢзЎ®дҝқжҜҸдёҖеұӮзҡ„иҫ“еҮәж–№е·®дёҺиҫ“е…Ҙж–№е·®зӣёиҝ‘пјҢйҖӮз”ЁдәҺ Sigmoid е’Ң Tanh зӯүжҝҖжҙ»еҮҪж•°пјҢиғҪжңүж•Ҳзј“и§ЈжўҜеәҰж¶ҲеӨұжҲ–зҲҶзӮёй—®йўҳ

Xavier жӯЈжҖҒеҲҶеёғеҲқе§ӢеҢ–пјҡеқҮеҖјдёә 0пјҢж ҮеҮҶе·®дёә 2nin+nout\sqrt{\frac{2}{n_{in}+n_{out}}}ninвҖӢ+noutвҖӢ2вҖӢвҖӢ зҡ„жӯЈжҖҒеҲҶеёғгҖӮ

Xavier еқҮеҢҖеҲҶеёғеҲқе§ӢеҢ–пјҡ[вҲ’6nin+nout,2nin+nout][-\sqrt{\frac{6}{n_{in}+n_{out}}},\sqrt{\frac{2}{n_{in}+n_{out}}}][вҲ’ninвҖӢ+noutвҖӢ6вҖӢвҖӢ,ninвҖӢ+noutвҖӢ2вҖӢвҖӢ] еҢәй—ҙеҶ…еқҮеҢҖеҲҶеёғгҖӮ

He еҲқе§ӢеҢ–пјҲKaiming еҲқе§ӢеҢ–пјү

He еҲқе§ӢеҢ–ж №жҚ®иҫ“е…Ҙзҡ„зҘһз»Ҹе…ғж•°йҮҸи°ғж•ҙжқғйҮҚзҡ„еҲқе§ӢиҢғеӣҙпјҢдё»иҰҒйҖӮз”ЁдәҺ ReLU еҸҠе…¶еҸҳдҪ“пјҲеҰӮ Leaky ReLUпјүжҝҖжҙ»еҮҪж•°

He жӯЈжҖҒеҲҶеёғеҲқе§ӢеҢ–пјҡеқҮеҖјдёә 0пјҢж ҮеҮҶе·®дёә 2nin\sqrt{\frac{2}{n_{in}}}ninвҖӢ2вҖӢвҖӢ зҡ„жӯЈжҖҒеҲҶеёғ

HeеқҮеҢҖеҲҶеёғеҲқе§ӢеҢ–пјҡеҢәй—ҙ [2nin,2nin][\sqrt{\frac{2}{n_{in}}},\sqrt{\frac{2}{n_{in}}}][ninвҖӢ2вҖӢвҖӢ,ninвҖӢ2вҖӢвҖӢ] еҶ…еқҮеҢҖеҲҶеёғ

е…¶дёӯ ninn_{in}ninвҖӢ иЎЁзӨәиҫ“е…Ҙж•°

BatchNormal еҸӮж•°еҪ’дёҖеҢ–

Pytorch дёӯзҡ„еҸӮж•°еҪ’дёҖеҢ–ж–№жі•д»ҘзҪ‘з»ңеұӮзҡ„еҪўејҸе®һзҺ°пјҢйҖҡиҝҮи°ғз”Ё torch.nn.BatchNormal1D() ж–№жі•еҸҜд»ҘеҲқе§ӢеҢ–дёҖдёӘзҪ‘з»ңеұӮпјҢиҜҘж–№жі•жҺҘж”¶дёҖдёӘеҸӮж•° num_features иЎЁзӨәеҫ…еҪ’дёҖеҢ–зҡ„зү№еҫҒж•°

еёёи§Ғзҡ„еҸҳз§Қиҝҳжңү torch.nn.BatchNormal2D() е’Ң torch.nn.BatchNormal3D() ж–№жі•пјҢзӣ®еүҚжҲ‘д»¬жңҖеёёз”Ёзҡ„е°ұжҳҜ 1DпјҢиҜҘж–№жі•жүҖжҺҘж”¶зҡ„иҫ“е…Ҙеј йҮҸзҡ„еҪўзҠ¶еҝ…йЎ»жҳҜдәҢз»ҙжҲ–иҖ…дёүз»ҙпјҢеҚіеҪўзҠ¶дёә [N, C] жҲ–иҖ… [N, C, L]пјҢе…¶дёӯпјҡ

  • C дёәзү№еҫҒж•°пјҢеҝ…йЎ»е’Ң num_features еҖјзӣёеҜ№еә”
  • N дёәдёҖдёӘжү№ж¬Ў batch зҡ„еӨ§е°ҸпјҢеҚідёҖз»„еҶ…ж•°жҚ®зҡ„жқЎж•°
  • L дёәеҸҜйҖүйЎ№пјҢиЎЁзӨәеәҸеҲ—й•ҝеәҰжҲ–ж—¶й—ҙз»ҙеәҰ

еҜ№дәҺ 3D еј йҮҸ [N, C, L]пјҢе…¶еҪ’дёҖеҢ–и§„еҲҷжҳҜеҜ№дәҺжҜҸдёҖдёӘзү№еҫҒ CпјҢеҚіжүҖжңүзҹ©йҳөзҡ„з¬¬ C иЎҢдёҠзҡ„жүҖжңүеҲ—пјҢжұӮи§Јиҝҷдәӣж•°жҚ®зҡ„еқҮеҖјж–№е·®пјҢз”ЁжқҘеҜ№жүҖжңүзҹ©йҳөиҝҷдёҖиЎҢдёҠжүҖжңүеҲ—иҝӣиЎҢеҪ’дёҖеҢ–пјҢеҒҮи®ҫж•°жҚ®дёәпјҡ

  • д»Һж ·жң¬1пјҢйҖҡйҒ“0пјҡ[13, 14, 15, 16]

  • д»Һж ·жң¬1пјҢйҖҡйҒ“1пјҡ[17, 18, 19, 20]
  • жүҖжңүж•°жҚ®пјҡ[5, 6, 7, 8, 17, 18, 19, 20]пјҢе…ұ 8 дёӘеҖјжұӮеқҮеҖје’Ңж–№е·®пјҢз”ЁжқҘеҪ’дёҖеҢ–иҝҷ 8 дёӘж•°жҚ®

  • д»Һж ·жң¬0пјҢйҖҡйҒ“2пјҡ[9, 10, 11, 12]
  • д»Һж ·жң¬1пјҢйҖҡйҒ“2пјҡ[21, 22, 23, 24]
  • жүҖжңүж•°жҚ®пјҡ[9, 10, 11, 12, 21, 22, 23, 24]пјҢе…ұ 8 дёӘеҖјжұӮеқҮеҖје’Ңж–№е·®пјҢз”ЁжқҘеҪ’дёҖеҢ–иҝҷ 8 дёӘж•°жҚ®

  • еҪўеҰӮ__init__()иҝҷз§ҚеҠ еҸҢдёӢеҲ’зәҝзҡ„ж–№жі•еңЁ Python дёӯз§°дёәйӯ”жңҜж–№жі•пјҢдёҚдјҡиў«жҳҫејҸи°ғз”ЁпјҢиҝҷйҮҢзҡ„__init__()еңЁеҠ иҪҪжЁЎеһӢ model=Model() зҡ„ж—¶еҖҷе°ұдјҡиў«зі»з»ҹиҮӘеҠЁи°ғз”ЁпјҢе®ҢжҲҗеҜ№еҸӮж•°зҡ„еҲқе§ӢеҢ–
  • иҝҷйҮҢжҲ‘д»¬е®ҡд№үзҡ„ self.linear1 зӯүйғҪжҳҜзұ»еҶ…еұһжҖ§пјҢдҪҶжҳҜеңЁ forward ж–№жі•дёӯиў«еҪ“еҒҡеҮҪж•°и°ғз”ЁпјҢе®һйҷ…дёҠжҳҜи°ғз”ЁдәҶ Linear зұ»еҶ…зҡ„йӯ”жңҜж–№жі•__call__()пјҢжң¬иҙЁдёҠиҝӣиЎҢзҡ„е°ұжҳҜ xw+b зҡ„ж“ҚдҪң
  • еҰӮжһңжҲ‘д»¬жғіи®©ж•ҙдёӘзҪ‘з»ңеұӮе®ҡд№үеңЁ GPU жҳҫеҚЎдёҠпјҢйңҖиҰҒеҗҢж—¶дҝ®ж”№иҫ“е…Ҙ x е’ҢжҜҸдёҖдёӘзәҝжҖ§еұӮ nn.Linear() зҡ„ device еҸӮж•°дёә cudaпјҢжҲ–иҖ…еңЁе®ҡд№үе®ҢжҲҗеҗҺпјҢи°ғз”Ёе®ғд»¬зҡ„ to(вҖҳcudaвҖҷ) ж–№жі•пјҢе°Ҷе…¶иҪ¬з§»еҲ°жҳҫеҚЎдёҠ

зӣҙжҺҘи°ғз”Ё summary() ж–№жі•пјҢдј е…Ҙеҫ…и§ӮжөӢжЁЎеһӢ modelпјҢиҫ“е…Ҙж•°жҚ®зҡ„зү№еҫҒж•°йҮҸ input_sizeпјҢиҫ“е…Ҙж•°жҚ®зҡ„жү№ж¬ЎеӨ§е°Ҹ batch_sizeпјҢд»ҘеҸҠжЁЎеһӢжүҖеңЁзҡ„и®ҫеӨҮ deviceпјҲй»ҳи®Өдёә cpuпјҢеҸҜжҢҮе®ҡдёә cudaпјү

иҜҘж–№жі•еҚіеҸҜиҫ“еҮәз»“жһ„еҢ–еӯ—ж®өпјҢеҸҚжҳ жЁЎеһӢзҡ„з»“жһ„е’ҢеҸӮж•°дҝЎжҒҜпјҡ

еҪ“жЁЎеһӢеҸӮж•°йҮҸдёҚеӨ§ж—¶пјҢеҰӮжһңжҲ‘д»¬жғіиҜҰз»ҶжҹҘзңӢжҜҸдёӘзәҝжҖ§еұӮе…·дҪ“зҡ„еҸӮж•°еҖјжҳҜеӨҡе°‘пјҢиҝҳеҸҜд»ҘдҪҝз”Ёд»ҘдёӢдёӨз§Қж–№жі•пјҡ

еҸҜд»ҘйҖҡиҝҮ torch.nn.Sequential жқҘжһ„е»әжЁЎеһӢпјҢе°Ҷеҗ„еұӮжҢүйЎәеәҸдј е…Ҙ

иҝҷйҮҢйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢеҜ№дәҺ Pytorch дёӯзҡ„жҝҖжҙ»еҮҪж•°пјҢжҲ‘д»¬зҺ°еңЁжңүдёӨз§Қи°ғз”Ёж–№ејҸпјҡ

  • и°ғз”Ё torch.softmax()пјҢtorch.tanh()пјҢtorch.relu() зӯүж–№жі•пјҢеӯ—жҜҚйғҪжҳҜе°ҸеҶҷпјҢеҪ“еҮҪж•°дҪҝз”ЁпјҢеңЁ forward дёӯеҚ•зӢ¬и°ғз”Ё
  • и°ғз”Ё nn.Softmax()пјҢnn.Tanh()пјҢnn.ReLU() зӯүж–№жі•пјҢйҰ–еӯ—жҜҚеӨ§еҶҷпјҢеҪ“зҪ‘з»ңеұӮдҪҝз”ЁпјҢеҸҜд»Ҙе’Ң nn.Linear() зӯүзәҝжҖ§еұӮпјҢдёҖиө·еңЁеҲқе§ӢеҢ–зҪ‘з»ңз»“жһ„ж—¶е®ҡд№ү

иҝҷдёӨз§Қи°ғз”Ёеә•еұӮи°ғз”Ёзҡ„жҝҖжҙ»еҮҪж•°е®Ңе…ЁзӣёеҗҢ

Sequential зұ»дҪҝжЁЎеһӢжһ„йҖ еҸҳеҫ—з®ҖеҚ•пјҢдёҚеҝ…иҮӘе®ҡд№үзұ»е°ұеҸҜд»Ҙз»„еҗҲж–°зҡ„жһ¶жһ„пјҢз„¶иҖҢе№¶дёҚжҳҜжүҖжңүзҡ„жһ¶жһ„йғҪжҳҜз®ҖеҚ•зҡ„йЎәеәҸжһ¶жһ„пјҢеҪ“йңҖиҰҒжӣҙејәзҡ„зҒөжҙ»жҖ§ж—¶иҝҳжҳҜйңҖиҰҒиҮӘе®ҡд№үжЁЎеһӢ

д»ҘдёҠжҲ‘д»¬йҖҡиҝҮеҜ№ Model жЁЎеқ—зҡ„йҮҚеҶҷпјҢе®ҢжҲҗдәҶзҪ‘з»ңжЁЎеһӢзҡ„е®ҡд№үе’ҢеүҚеҗ‘дј ж’ӯиҝҷдёӨдёӘиҝҮзЁӢпјҢиҖҢеҰӮжһңжғізңҹжӯЈеҫ—еҲ°жўҜеәҰпјҢеҲҷйңҖиҰҒеҸҚеҗ‘дј ж’ӯзҡ„иҝҮзЁӢпјҢеҸҚеҗ‘дј ж’ӯзҡ„иө·зӮ№еҚіжҚҹеӨұеҮҪж•°пјҢдёӢж–ҮжҲ‘д»¬и®Іи§ЈжҚҹеӨұеҮҪж•°еңЁ Pytorch дёӯзҡ„е®һзҺ°ж–№ејҸ

  • е’ҢжЁЎеһӢйў„жөӢиҫ“еҮәеҪўзҠ¶ [N, 2] зӣёеҗҢзҡ„зӢ¬зғӯзј–з ҒпјҢжҜҸдёҖиЎҢйғҪжҳҜ [0, 1] жҲ–иҖ… [1, 0]пјҢд№ҹеҸҜд»Ҙи®ӨдёәжҳҜзұ»еҲ«ж ҮзӯҫпјҢж Үи®°еұһдәҺ 0 зұ»иҝҳжҳҜ 1 зұ»

Pytorch е®һзҺ°зӨәдҫӢеҰӮдёӢпјҡ

еңЁжұӮеҫ—жҚҹеӨұеҮҪж•°д»ҘеҗҺпјҢжңҖеҗҺдёҖжӯҘе°ұжҳҜйҖүжӢ©еҗҲйҖӮзҡ„еҸӮж•°жӣҙж–°ж–№жі•пјҢиҝӣиЎҢеҸҚеҗ‘дј ж’ӯпјҢеңЁеӨҡиҪ®иҝӯд»Јдёӯжӣҙж–°еҸӮж•°

еҸӮж•°жӣҙж–°ж–№жі•

Pytorch дёӯжңүе…іеҸӮж•°жӣҙж–°зҡ„ж–№жі•д»ҘдјҳеҢ–еҷЁпјҲoptimizerпјүзҡ„еҪўејҸе®һзҺ°пјҢеёёз”Ёзҡ„дјҳеҢ–ж–№жі•йғҪйҖҡиҝҮ torch.optim еҢ…иҝӣиЎҢи°ғз”Ё

Pytorch дёӯеҸҚеҗ‘дј ж’ӯжӣҙж–°еҸӮж•°зҡ„жөҒзЁӢеӨ§иҮҙеҲҶжҲҗд»ҘдёӢдёүжӯҘпјҡ

еңЁеӨҡиҪ®иҝӯд»Јдёӯпјҡ

  • жҜҸж¬ЎеҠ иҪҪдёҖдёӘе°Ҹжү№йҮҸ batch зҡ„ж•°жҚ®пјҢдҪңдёәжЁЎеһӢ model иҫ“е…Ҙеҫ—еҲ°жЁЎеһӢиҫ“еҮә
  • е°ҶжЁЎеһӢиҫ“еҮәе’Ңж Үзӯҫдј е…Ҙи®ҫе®ҡзҡ„жҚҹеӨұеҮҪж•°пјҢеҫ—еҲ° loss_value
  • жҚҹеӨұеҮҪж•°иҝӣиЎҢеҸҚеҗ‘дј ж’ӯеҫ—еҲ°жЁЎеһӢеҸӮж•°жўҜеәҰ
  • дҪҝз”Ёи®ҫзҪ®еҘҪзҡ„дјҳеҢ–еҷЁжӣҙж–°жЁЎеһӢеҸӮж•°
  • жё…з©әжң¬иҪ®жўҜеәҰпјҢйҳІжӯўеңЁдёӢдёҖиҪ®еҸӮж•°жӣҙж–°ж—¶жң¬иҪ®жўҜеәҰзҙҜеҠ 

йҒөеҫӘиҝҷдёӘжөҒзЁӢпјҢжҲ‘д»¬жң¬иҠӮйңҖиҰҒеҒҡзҡ„е°ұжҳҜйҖүжӢ©дёҚеҗҢзҡ„дјҳеҢ–еҷЁеҚіеҸҜ

йҡҸжңәжўҜеәҰдёӢйҷҚ & еҠЁйҮҸжі•

жўҜеәҰдёӢйҷҚжі•е’ҢеҠЁйҮҸжі•йғҪеҸҜд»ҘйҖҡиҝҮ torch.optim.SGD() ж–№жі•е®һзҺ°пјҢеҗҺиҖ…йҖҡиҝҮи®ҫзҪ® momentum еҸӮж•°пјҢжқҘи®ҫзҪ®еҺҶеҸІжўҜеәҰжқғйҮҚпјҢд»ҺиҖҢе®һзҺ°еҠЁйҮҸжі•

иҜҘж–№жі•жҺҘж”¶еҰӮдёӢеҮ дёӘеёёи§ҒеҸӮж•°пјҡ

  • paramsпјҡдјҳеҢ–еҷЁеҫ…жӣҙж–°зҡ„еҸӮж•°еҲ—иЎЁпјҢеҰӮжһңжңүзҪ‘з»ңжЁЎеһӢ ModelпјҢйҖҡеёёдј е…Ҙ Model.parameters() еҚіеҸҜпјҢеҰӮжһңеҸӘжҳҜжӣҙж–°еҚ•дёӘеј йҮҸ XпјҢеҲҷйңҖиҰҒе°ҒиЈ…жҲҗеҲ—иЎЁеҪўејҸ
  • lrпјҡеӯҰд№ зҺҮпјҢй»ҳи®Өдёә 1e-3
  • momentumпјҡеҠЁйҮҸжі•дёӯзҡ„еҺҶеҸІжўҜеәҰжқғйҮҚ

еӯҰд№ зҺҮиЎ°еҮҸ

еӯҰд№ зҺҮиЎ°еҮҸдјҳеҢ–еҷЁ lr_scheduler е№¶дёҚзӣҙжҺҘжӣҙж–° Model.parameters()пјҢиҖҢжҳҜжӣҙж–°жҹҗдёӘе·ІжңүдјҳеҢ–еҷЁзҡ„еҸӮж•°пјҢеӣ жӯӨе…¶жҺҘж”¶зҡ„еҸӮж•°жҳҜ optimizer иҖҢдёҚжҳҜ paramsпјҢеңЁжӣҙж–°дјҳеҢ–еҷЁеҸӮж•°ж—¶пјҢд№ҹйңҖиҰҒйўқеӨ–и°ғз”ЁдёҖж¬Ў scheduler_lr.step() ж–№жі•жүҚиғҪе®ҢжҲҗеҜ№еә”дјҳеҢ–еҷЁеӯҰд№ зҺҮзҡ„иЎ°еҮҸ

  • optimizerпјҡиҰҒе®һзҺ°еӯҰд№ зҺҮиЎ°еҮҸзҡ„дјҳеҢ–еҷЁ
  • sizeпјҡеҸ‘з”ҹиЎ°еҮҸзҡ„й—ҙйҡ”
  • gammaпјҡиЎ°еҮҸзҡ„жҜ”дҫӢ

дҫӢеҰӮи®ҫзҪ®еӯҰд№ зҺҮжҜҸй—ҙйҡ” 20 epoches иЎ°еҮҸдёәе…ҲеүҚзҡ„ 0.7пјҡ

  • optimizerпјҡиҰҒе®һзҺ°еӯҰд№ зҺҮиЎ°еҮҸзҡ„дјҳеҢ–еҷЁ
