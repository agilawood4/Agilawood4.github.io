物理层

在 OSI 参考模型中，物理层指的是物理信道，而不是物理介质

基本概念

物理层解决如何在连接各种计算机的传输媒体上传输比特流，而不是指具体的传输媒体，和具体的介质没有关系

物理层的主要任务描述为：确定与传输媒体的接口的一些特性，即

  • 机械特性：例如接口形状，大小，引线数目
  • 电气特性：例如规定电压范围（-5V 到 +5V）
  • 功能特性：例如规定 -5V 表示 0，+5V 表示 1
  • 过程特性：也称规程特性，规定建立连接时各个相关部件的工作步骤

数据通信的基础知识

典型的数据通信模型：

相关术语：

  • 数据（data）：运送消息的实体（通信的目的是传送消息）
  • 信号（signal）：数据的电气或电磁的表现
  • 模拟信号：代表消息的参数的取值是连续的
  • 数字信号：代表消息的参数的取值是离散的
  • 码元（code）：在使用时间域的波形表示数字信号时，代表不同离散数值的基本波形称为码元
    • 在数字通信中常用时间间隔相同的符号来表示一个二进制数字，这样的时间间隔内的信号称为二进制码元，间隔被称为码元长度，一个码元可以携带 n 个 bit 的信息量

有关信道的基本概念：

  • 单向通信（单工通信）：只能有一个方向的通信而没有反方向的交互，比如电视台无线电广播
  • 双向交替通信（半双工通信）：通信的双方都可以发送信息，但不能双方同时发送（也不能同时接收），比如对讲机
  • 双向同时通信（全双工通信）：通信的双方可以同时接收和发送信息，比如电话

基带信号（baseband）和带通信号（band pass）：

  • 基带信号，即基本频带信号，指来自信源的信号，像计算机输出的代表各种文字或图像文件的数据信号都属于基带信号，基带信号就是发出的直接表达了要传输的信息的信号，比如我们说话的声波就是基带信号
  • 带通信号：把基带信号经过载波调制后，把信号的频率范围搬移到较高的频段以便在信道中传输，即仅一段频率范围内的信号能通过信道
  • 近距离传输（比如计算机到监视器、打印机等外设之间信号的传输），使用基带信号；远距离传输，使用带通信号

几种基本的调制方法：

  • 调幅（AM）：载波的振幅随基带数字信号而变化
  • 调频（FM）：载波的频率随基带数字信号而变化
  • 调相（PM）：载波的初始相位随基带数字信号而变化（正弦波 / 余弦波）

常见编码方式：

（1）单极性不归零码和双极性不归零码

  • 单极性：用 0.5 作为判决门限，小于 0.5 看做 0，大于 0.5 看做 1
  • 双极性：用 -1 表示 0，用 1 表示 1
  • 所谓不归零的含义为：如果有连续的 1 或者 -1 信号出现，相邻信号之间保持 1 或 -1，间隔位置不会变为 0

（2）单极性归零码和双极性归零码

  • 所谓归零的含义：在连续的 1 和 -1 之间，信号会归零
  • 其余和不归零码无区别

（3）以上两种编码方式，无法区分是信号传输流结束，还是信号全部都是 0，由此提出了曼彻斯特编码

  • 曼彻斯特编码：一个时钟周期只表示一个 bit，且必须经过两次采样才能得到一个 bit，该编码能够携带时钟信号，且可以区分没有数据传输与信号全部为 0 的情况
  • bit 内部从低信号跳变到高信号，表示 0；bit 内部从高信号跳变到低信号，表示 1，如果不发生跳变，则表示没有数据传输

（4）差分曼彻斯特编码

  • bit 与 bit 之间有信号跳变，表示下一个 bit 为 0
  • bit 与 bit 之间没有信号跳变，表示下一个 bit 为 1
  • 差分曼彻斯特编码的 bit 内部也有信号跳变，一般认为其抗干扰能力强于曼彻斯特编码

奈氏准则与香农准则

信号传输过程中有可能造成失真：

奈氏准则：该准则给出了在假定的理想条件下，为了避免码间串扰，码元的传输速率的上限值

在任何信道中，码元传输的速率是有上限的，否则就会产生码间串扰的问题，使接收端对码元的判决 / 识别成为不可能

如果信道的频带越宽，也就是能够通过的信号高频分量越多，那么就可以用更高的速率传送码元而不出现码间串扰

奈氏准则

理想低通信道（没有噪声）的最高码元传输速率为：2WBaud

  • W 是理想低通信道的带宽，单位为 Hz
  • Baud 是波特，是码元传输速率的单位

Baud 与 Bit 的区别：一个 Baud 等于多少 Bit 是由码元所能表示的信号的二进制位数所决定的

  • 波特：在调制解调器中经常用到这个概念
  • BIt：信息量
  • 如果一个码元中含有 X 个 Bit 的信息量，那么则有 1 Baud = X Bit

比如在本图中，一个码元表示 3 位二进制，因此 1 Baud = 3 Bit

奈氏准则考虑的是没有噪声的情况，香农公式考虑的是有噪声的情况

香农公式

信噪比：香农用信息论的理论推导出了带宽受限且有高斯白噪声噶绕的信道的极限、无差错的信息传输速率

信道的迹象信息传输速率 C\mathcal{C}C 可以表示为：

C=Wlog⁡2(1+SN)  b/s\mathcal{C} = W\log_2(1+\frac{S}{N})\; b/s C=Wlog2​(1+NS​)b/s

其中：

  • W：信道的带宽，单位为 Hz
  • S：信道内所传信号的平均功率
  • N：信道内部的高斯噪声功率

香农公式表明：

  • 信道的带宽或信道中的信噪比越大，则信息的极限传输速率越高
  • 只要信息传输速率低于信道的极限信息传输速率，就一定可以找到某种办法来实现无差错的传输
  • 若信道带宽 W 或信噪比 S/N 没有上限（实际信道不可能发生），则信道的极限传输速率 CCC 也没有上限
  • 实际信道能够到达的信息传输速率远低于香农公式的极限传输序速率

物理层下面的传输媒体

电信领域下使用的电磁波的频谱：频率越高，波长越短

导向传输媒体

导向传输媒体：电磁波沿着固体媒体传播

  • 双绞线
    • 屏蔽双绞线 STP，一般用于远程传送
    • 无屏蔽双绞线 UTP，一般用于近程传送
  • 同轴电缆
    • 50Ω 同轴电缆，用于数字传输，也多用于基带传输，因此也被称为基带同轴电缆
    • 75Ω 同轴电缆，用于模拟传输，即宽带同轴电缆

  • 光缆
    • 多模光纤：光在内部发生全反射
    • 单模光纤：光在内部沿直线传输

非导向传输媒体

非导向传输媒体：非导向传输媒体即自由空间，其中的电磁波传输被称为无线传输

  • 无线传输所使用的频段很广

短波通信主要是靠电离层的反射，单短波通信的通信质量较差

微波在空间中沿直线传播，因此需要为其建立基站，或者借助卫星完成通信

集线器

物理层设备—集线器

集线器工作特点：在网络中只起到信号放大和重发作用，其目的是扩大网络的传输范围，而不具备信号的定向传送能力

假设单条网线最大传输距离 100m，使用集线器，两台设备间最大传输距离变成 200m

集线器是一个大的冲突域（半双工通信），连接在集线器上的设备，一次只允许其中两台设备进行单向通信，且每台设备得到的平均带宽为集线器的总带宽 / 设备数

集线器不安全，A 设备发送给 B 设备的信息，不会进行过滤，A 会把信息发送给所有设备，只有 B 会选择接收，其他设备如果想接收也可以得知信息内容

频分复用技术

频分复用（Frecuency Division Multiplexing，FDM）

用户在分配到一定的频带后，在通信过程中自始至终都占用这个频带，频分复用则指的是所有用户在同样的时间占用不同的带宽资源（这里的带宽是指频率带宽而不是数据的发送速率）

频分复用的核心是把不同用户的信号调制到不同频率，再把不同频率的波进行叠加，由于波的形式，叠加后的波相互之间不会干扰，最后再按照频率进行过滤，分别解调即可

频分复用的例子：多台电话之间的连接，其中 A 给 B 打电话的同时，C 给 D 也可以打电话，使用的即为频分复用原理，其中能够复用的线路称为干道

在频分复用场景下，带宽是可以叠加的，比如每台电话占用 4kHz 的带宽，如果是 12 台电话同时工作，带宽就会变成 48kHz

时分复用技术

时分复用（Time Division Mulyiplexing，TDM）

时分复用是将时间划分为一段段等长的时分复用帧（TDM 帧），每一个时分用户的用户在每一个 TDM 帧中占用固定需要的时隙，每一个用户所占用的时隙周期性出现，其周期就是 TDM 帧的长度对应的时间

TDM 信号也称为等时（isochronous）信号，时分复用的所有用户是在不同的时间占用同样的频带宽度

每一台设备都会对应分配一个时间片，各个设备的时间段的顺序是固定的，比如下图中都是按照先 1 后 2 再 3 再 4 的顺序分配时间片，接收的时候也是按照这个顺序对应接收端设备，不能打乱

另一种示意图：

把信号具象化为二进制，则按照从上到下依次分配时间片的顺序，第一个 TDM 帧传输的信号为 1010，第二个 TDM 帧传输的信号为 1000，第三个 TDM 帧传输的信号为 1011，依次类推

可以认为有一个按照固定方向匀速旋转的转盘，以固定的顺序从不同设备每次取出一个信号放到流水线上，同时另一个转盘从流水线取信号给接收端设备

时分复用技术的缺点：可能造成线路资源的浪费，如果某些设备在对应的时间片没有给出信号，原本这个信号对应的位置也无法分配给其它设备，会被空置，由此造成带宽浪费

使用时分复用系统传送计算机数据时，由于计算机数据的突发性质，用户对分配到的子信道的利用率一般是不高的

统计时分复用

统计时分复用（Statistic TDM，STDM）

统计时分复用不再为每个设备预留固定的时间片，只会按照固定顺序扫描每个设备，在读取某设备信号的时候，会给其打上一个标签，标记它来自哪个设备

接收端可以读取这个标签，然后将其去掉，通过标签来判断这个信号该给哪个接收设备

波分复用

波分复用（Wavelenth Division Multiplexing，WDM）

波分复用本质上就是光的频分复用，因为按照波长进行区分，在光速一定的前提下，本质上就是按照频率进行区分

码分复用

码分复用（CDM，Code Division Multiplexing）

常用的名词是码分多址（CDMA，Code Division Multiple Access），各个用户使用经过特殊挑选的不同码型，因此彼此不会造成干扰

这种系统发送的信号有很强的抗干扰能力，其频谱类似于白噪声，不容易被敌人发现，每一个比特时间划分为 m 个短的间隔，称为码片（chip ）

其工作原理如下：

  • 每个设备都会有一个码片，比如 S 站的码片序列为 S，在码片序列中只有 1 和 -1
  • 该设备在发送信号时，如果想发送信号 1，就发送码片序列本身；如果想发送 0，就发送码片序列的反码（比如 S 的码片序列如果是 -1-1-111-111，其反码就是 111-1-11-1-1）
  • 不同设备选取的码片序列不仅各不相同，相互之间必须是正交的
  • 任何一个码片向量和该码片向量自己的规格化内积都是 1；任何一个码片向量和该码片向量反码的规格化内积都是 0

在上图中，S 站和 T 站发送的 110 信号，每一位各自会用其码片序列或码片序列的反码表示，假设一个码片序列为 8 位，则这个 3 位信号会被扩展成 24 位的信号并且进行叠加（按位相加）

在接收端解码时，对应基站会把自己的码片向量和传送过来的信号做规格化内积，也就是向量按位相乘求和再除以向量长度，根据码分复用的最后一条性质，只有对应基站之间的规格化内积才会得到 1 或者 -1，其中 1 表示传输过来一个 1 信号，-1 表示传输过来一个 0 信号；而其他基站做规格化内积的结构都是 0

码分复用的缺点是：要传输 1 bit 的信息，现在需要发送 m bit（m 是码片数量）才能完成，且设备越多，为了区分不同设备的码片，需要划分的码片数量 m 也随之增加

手机通信用的就是码分复用技术，所有手机打电话用的信号频率都是相同的

数字传输系统

脉码调制 PCM 体制最初是为了在电话局之间的中继线上传送多路电话

由于历史原因，PCM 有两个互不兼容的国际标准，即北美的 24 路 PCM（称为 T1 标准），和欧洲的 30 路 PCM（称为 E1 标准），我国采用的是 E1 标准

E1 的速率是 2.048 Mb/s，而 T1 的速率是 1.544 Mb/s

E1：每秒钟采样 8000 次，每个采样为 8bit，每个 bit 中包含 32 位信息（32 个信道）

T1：每秒钟采样 8000 次，每个采样为 8bit，每个 bit 包含 24 位置信息（24 个信道），此外还有一个控制信道
