物理层

物理层的概念

物理层的主要任务：实现传输介质上的（透明）比特流传输

物理层涉及的四个特性：

  • 机械特性：接口所用接线器的形状和尺寸，引线数目和排列，固定和锁定装置等
  • 电气特性：在接口电缆的各条线上出现的电压的范围
  • 功能特性：某条线上出现的某一电平的电压表示何种意义
  • 过程特性：对于不同功能的各种可能事件的出现顺序

数据通信的基础知识

数据通信系统的模型：

计算机产生的信号为数字信号（数字比特流），在公共电话网上传输的信号是模拟信号，数字信号转换为模拟信号的过程称为调制，模拟信号转换为数字信号的过程称为解调

数据通信的基础知识

信道（channel）：向某一方向传送信息的介质

通信的目的：传送消息（message），如话音，文字，图像等

数据（data）：消息的实体

信号（signal）：数据的电气或电磁表现，信号可以分为模拟信号和数字信号

码元：在使用时间域（简称为时域）的波形表示数字信号时，代表不同离散数值的波形称为码元

  • 码元速率：单位时间内通过信道的码元个数，单位为波特（baud）
  • 比特速率：单位时间内通过信道的信息量（比特数），单位为 bit/s 或 bps（bit per second）
  • 比特速率和码元速率是两个不同的概念，对于同一信道而言，如果一个码元可以携带 n 比特的信息，则比特速率即为 n 倍的码元速率，换言之，仅当使用二进制编码时，二者才相等
  • 我们希望一个码元能尽可能携带更多的比特，从而可以提高信道的数据速率

通信双方的交互方式：

  • 单向通信（单工通信，simplex）：只有一个方向的通信而没有反方向的交互
  • 双向交替通信（半双工通信，half-duplex）：通信的双方都可以发送信息，但不能双方同时发送，当然也不能双方同时接收
  • 双向同时通信（全双工通信，duplex）：通信的双方可以同时发送和接收信息

基带信号和带通信号：

  • 基带信号（baseband）：又称为基本频带信号，指来自信号源的信号，计算机输出的代表各种文字或图像文件的数据信号都属于基带信号
    • 基带信号往往包含较多的低频成分，甚至有直流成分，而许多信道并不能传输这种低频分量或直流分量，因此必须对基带信号进行调制（modulation）
  • 带通信号（band pass）：基带信号经过载波调制后，信号的频率范围搬移到较高的频段，以便在信道中传输（即仅在一段频率范围内能够通过信道）

基本的二元调制方法：

  • 调幅（AM）：载波的振幅随基带数字信号而变化
  • 调频（FM）：载波的频率随基带数字信号而变化
  • 调相（PM）：载波的初始相位随基带数字信号而变化

为了获得更高的传输速率（也就是让每个码元携带更多的比特信息），通常采用多元制的混合调制方法，常见的是振幅和相位相结合的正交振幅调制（QAM，Quadrature Amplitude Modulation）

不同振幅的波和不同相位的波都可以携带不同的信息，比如我们使用 4 种振幅和 4 种相位进行组合，即可实现每个码元都有 16 种不同的状态，即携带 16 个比特的信息，而因为频率可能会和信道传输信息的能力有关，因此我们一般不会轻易改变波的频率

信道的极限容量：

信号存在失真问题，即任何实际的信道都不是理想的，在传输信号时会产生各种失真并且带来各种干扰

码元的传输速率越高，或者信号传输的距离越远，在信道的输出端的波形的失真就越严重，因此一个码元所能携带的比特数不能无限制增加

一个信道所能通过的频率范围总是有限的，高频分量往往不能通过信道，因为它们会导致接收端收到的信号波形失去码元之间的清晰界限，即码间串扰现象

奈奎斯特定理（Nyquist）：理想（无噪声，和香农公式对比）低通信道最大数据传输率可以表示为

V=2Hlog⁡2V      (bps)V=2H\log_2V\;\;\;(bps) V=2Hlog2​V(bps)

其中 H 表示信道带宽（单位 Hz，这里 1Hz 对应最大传输速率的 1bps），V 表示信号电平的级数，即信号的状态数

在任何信道中，码元传输的速率都是有上限的，否则就会出现码间串扰的问题，使接收端对码元的判别称为不可能，由奈奎斯特定理可知，信道的频带越宽，能够通过的信号高频分量越多，即可使用更高的速率传输码元而不出现码间串扰

奈奎斯特定理应用：信道带宽为 3kHz，信号状态数为 4×4=16，因此最大比特传输速率为 24 bps

信噪比

信噪比以分贝 (dB)(dB)(dB) 作为计量单位，其具体公式为：

信噪比=10log⁡10(S/N)      (dB)\text{信噪比} = 10\log_{10}(S/N)\;\;\;(dB) 信噪比=10log10​(S/N)(dB)

其中 S/NS/NS/N 为信号的平均功率和噪声的平均功率之比，例如：S/N=10 时，信噪比为 10dB，S/N=1000 时，信噪比为 30dB

香农公式：有噪声的信道的极限信息传输速率 CCC 可表示为

C=Wlog⁡2(1+S/N)      (bps)C=W\log_2(1+S/N)\;\;\;(bps) C=Wlog2​(1+S/N)(bps)

其中 WWW 为信道带宽，单位 HzHzHz，SSS 为信道内部所传信号的平均功率，NNN 为信道内部高斯噪声的功率

香农公式表明：信道的带宽或信道中的信噪比越大，则信息的极限传输速率就越高

物理层下面的传输介质

传输介质：又称为传输媒体或传输媒介，指数据传输系统中发送器和接收器之间的物理通路

传输媒体分为两类

  • 导引型传输介质（有线介质）
  • 非导引型传输介质（无线介质）

导引型传输介质

双绞线（twisted pair）：把两根绝缘铜线绞合，减少相互干扰

  • 屏蔽双绞线 STP：用于抗干扰要求比较高的环境
  • 无屏蔽双绞线 UTP：计算机网络中大部分双绞线都是无屏蔽的

无屏蔽双绞线：双绞线在网络时代来临前就已经广泛用于电话系统，优点是价格便宜，便于安装使用，缺点是传输距离较短

局域网布线系统常使用 8 芯非屏蔽双绞线，标准 ETA/TIA-568 中规定了非屏蔽双绞线的类别和带宽

同轴电缆（coaxial cable）：内导体铜质芯线外包裹网状编织的屏蔽层

同轴电缆的抗干扰能力和传输速率都高于双绞线，但因为成本高，安装复杂，目前在计算机网络中应用并不广泛

常见的同轴电缆分为以下两类：

  • 50Ω 同轴电缆
  • 75Ω 同轴电缆

光纤（fiber optics）：由非常透明的石英玻璃拉成细丝，由纤芯和包层构成双层通信圆柱体

光纤分为两类：

  • 多模光纤（multi-mde）：多条不同入射角度的光线在同一条光纤中传输，距离短，成本低（短距离传输，比如机房之间的通信）
  • 单模光纤（single-mode）：光纤直径等于广播波长，光纤成为波导，此时光在光纤内部沿直线传播，不会反射（干道传输，比如国家之间的通信）

光纤通信的优点：

  • 通信容量大
  • 传输损耗小，距离长
  • 抗干扰能力强
  • 保密性好
  • 体积小，重量轻

计算机内部产生的电信号需要经过**光纤模块（收发器）**转换成光信号，从而才能够在光纤内进行传输

光纤目前已经广泛应用于长途干线传输，局域网/城域网的干线，高带宽且高可靠的网络连接（如服务器）

光纤通信的缺点：

  • 安装复杂，光纤连接需要光纤熔接器
  • 需要考虑光电信号的转换
  • 价格高

非导引型传输介质

非导引型传输介质指的是可传播无线电波的自由空间

无线通信的适用场合：

  • 偏远地区通信
  • 城市中敷设线缆较为困难的场合
  • 移动设备

主要分类：

  • 短波通信：通过电离层反射，传输距离长，通信质量较差，传输速率低
  • 微波通信和卫星通信：微波频率范围 300MHz∼300GHz300MHz\sim300GHz300MHz∼300GHz，频率大于 100MHz100MHz100MHz 的电磁波几乎沿着直线传播，可以使用抛物面状天线将能量聚集成束，从而获得较高的信噪比

地面微波通信：长距离传输时，需要每间隔一定距离建立中继站，进行接力转发

  • 优点：信道频段范围宽，容量大；在无线通信中抗干扰能力较强；与线缆通信相比，建设速度快，成本低
  • 缺点：相邻站需直视，易受地理或建筑物影响；容易受气候影响（雷电，太阳因子爆发等）；隐蔽性和保密性较差；中继站维护需要人力物力

无线频谱政策：

大多数无线频段都由政府进行管理和分配，需要获得许可才能使用

  • 相同频率的电磁波在空间中传播时相互影响，且无线频段范围有限，因此是稀缺资源，不能随意使用

政府也预留了无需许可即免费的频段，即 IMS 频段（Industrial Scientific Medical）：

  • 限制：发射功率小于 1 Watt
  • 极大的促进了短距离无线电的应用
  • 使用 ISM 频段的设备：WLAN，Bluetooth，无线鼠标，无绳电话

WIFI 信号也属于 ISM，由于发射功率的限制，大部分 WIFI 信号的覆盖范围都不大

未来可用 ISM 频率范围升高带来的问题：频率越高，信号穿透能力越差，在小范围内效果可能提升，但远距离传输质量变低

卫星通信：通过卫星进行微波信号转发

  • 优点：通信距离远，覆盖范围广，通信容量大
  • 缺点：传播时延极大，对于地球同步卫星而言，单向时延可达 250∼300ms250\sim300ms250∼300ms

除地球同步卫星以外，低轨道卫星在数据通信领域发展较快，低轨道卫星的优势如下：

  • 卫星造价和发射成本低
  • 传输距离短，时延较小，信号衰减小，可与地面手持 / 移动设备通信
  • 可以组建卫星互联网：StarLink，星网

其他无线通信手段：

  • 红外：易受可见光影响，只用于室内场景
  • 激光：在大气层内易受云雾遮挡，在太空应用前景广阔

信道复用技术

复用（Multiplexing）：在一个信道上传输多路信号

最基本的复用方法：频分多路复用（FDM，Frequency Division Multiplexing）和时分多路复用（TDM，Time Division Multiplexing）

频分多路复用：不同用户占用不同的频带资源

时分多路复用：把时间划分成一段段等长的时隙，每个用户使用不同的时隙，便于数字传输

示意图：

波分复用（WDM，WaveLength Division Multiplexing）：波分复用实质上是光的频分复用（因为光的频率一般都很高，因此用波长代替频率表示光载波），专门用于光纤通信

码分复用（CDM，Code Division Multiplexing）：也称为码分多址，即 CDMA，Code Division Multiple Access，常用于无线电通信

其基本原理为：每一个比特时间被划分为 m 个短的间隔，称为码片（chip），每个主机（发送端 / 接收端）都分配一个唯一的 m bit 码片序列，各个主机之间的码片序列必须正交（orthogonal）

  • 码片正交：代表码片的向量做规格化内积为 0
  • 向量规格化内积：向量内积除以向量长度

码分复用会使用扩频通信（Spread Spectrum）方法，把每 bit 信号转换为 m bit 的码片进行传输

CDMA 应用案例：

  • 假设我们使用 8 bit 的码片序列，即 1 bit 的信号会扩展为 8 bit 的码片
  • 每个基站 S 和 T 都会相应给出一个码片序列，比如 S 站为 (-1 -1 -1 +1 +1 -1 +1 +1)，T 站为 (-1 -1 +1 -1 +1 +1 +1 -1)，需要保证两者的码片序列是正交的，即 S⋅T=0,S⋅S=1S\cdot T=0,S\cdot S=1S⋅T=0,S⋅S=1
  • 假设 S 站发出的码元序列为 110，经过扩频后得到的发射信号为 Sx，T 站发出的码元序列为 110，经过扩频后得到的发射信号为 Tx
  • 两个信号叠加，即 Sx+TxS_x+T_xSx​+Tx​，得到混合的信号序列，在接收端解析时，S 站接收端使用 S 的码片序列，做 S(Sx+Tx)S(S_x+T_x)S(Sx​+Tx​)，即可还原出 S 站发送的信号，T 站接收端使用 T 的码片序列，做 T(Sx+Tx)T(S_x+T_x)T(Sx​+Tx​)，即可还原 T 站发送的信号
  • 同时有 S·T~X~=0 和 T·S~X~=0 成立，也就是每个接收端的码片和与之不对应的发送端传来的信号做规格化内积，得到的结果都是 0

码分复用发送的信号具有很高的抗干扰能力（因为使用了扩频通信，信号的频率从单值聚集扩展成一个频段的范围），其频谱类似于白噪声

数字传输系统

脉码调制系统（PCM，Pulse Code Modulation）用于电话交换系统，在数字传输系统中进行多路话音的变换与传输

话音由模拟到数字的变换：对模拟话音按周期进行采样和 A/D 变换

  • 采样频率 8000 次每秒，采样数据用 8bit 表示
  • 话音数据速率 64kb/s

数字化的话音数据（数字电话）进行时分多路复用，主要是欧洲体制和北美体制两种分类，中国采用的是欧洲体制

宽带接入技术

宽带没有严格定义，宽带接入泛指超越传统的 modem+ 电话线拨号上网的技术

典型的宽带接入技术：

  • 以家庭现有线路为基础，比如基于电话线的 xDSL，基于有线电视的 HFC
  • 基于光纤传输：典型的 FTTH

（1）xDSL 技术（Digital Subscribe Line）：解决最后一英里的问题，即完成从通信运营商局端到家庭用户之间的本地回路

xDSL 的几种常见类型：

  • ADSL：仍然使用现有电话线，无需对庞大的电话布线系统进行改造，把 0~4kHz 的低端频谱留给传统电话使用，把高端频谱作为网络接入（用户上网），宽带非对称，即上行（用户→ISP）和下行（ISP→用户）带宽不同，适应普通用户上网特点
  • HDSL
  • SDSL
  • VDSL

（2）HFC 技术（Hybrid Fiber Coax）：光纤混合同轴电缆，以有线电视网 CATV 为基础的宽带接入技术

（3）基于光纤的带宽接入：FTTx

  • FTTH，Fiber To The Home：光纤到户
  • FTTB，Fiber To The Building：光纤到大楼
  • FTTC，Fiber To The Curb：光纤到路边
