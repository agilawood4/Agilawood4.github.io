神经网络学习

神经网络的主要特点即可以从数据中进行 “学习”，这个学习的过程，就是让训练数据自动决定最优的权重参数

神经网络（深度学习）也是机器学习的一种，和传统的机器学习方法相比，神经网络不需要人工设置特征量（如 SIFT，HOG 等），这样就可以用同样的流程处理所有的问题

损失函数

在神经网络中，需要以某个指标为线索来寻找最优权重参数，这个指标即为损失函数（Loss Function），损失函数通常可以写成 L(f(x),y)L(f(x),y)L(f(x),y)，其中 xxx 为原始输入，yyy 为数据标签，f()f()f() 代表神经网络中的参数 w,bw,bw,b，由于对于一组确定的训练数据而言，原始输入和标签都是确定值，因此损失函数本质上是权重参数的函数

均方误差（MSE）

均方误差（Mean Squared Error，MSE）也称为 L2 Loss，通常用于回归问题：

L=1n∑i=1n(yi−ti)2L=\frac{1}{n}\sum_{i=1}^n (y_i-t_i)^2 L=n1​i=1∑n​(yi​−ti​)2

其中，yiy_iyi​ 表示神经网络的输出，tit_iti​ 表示监督数据的标签（正确的解标签），nnn 则是数据的维度，对于固定维度的网络，前面的系数 nnn 不重要，因此为了求导方便，有时公式也写成：

L=12∑i=1n(yi−ti)2L=\frac{1}{2}\sum_{i=1}^n (y_i-t_i)^2 L=21​i=1∑n​(yi​−ti​)2

L2 Loss 对异常值敏感，遇到异常值时容易发生梯度爆炸

交叉熵误差（CE）

交叉熵误差（Cross Entropy Error，CE）通常用作分类问题的损失函数：

L=−1n∑i=1ntilog⁡yiL=-\frac{1}{n}\sum_{i=1}^n t_i\log y_i L=−n1​i=1∑n​ti​logyi​

其中，log⁡\loglog 表示自然对数，yiy_iyi​ 表示神经网络的输出，tit_iti​ 表示正确解标签，且 tit_iti​ 为 one−hotone-hotone−hot 表示，其中只有正确解标签对应的值为 1，其他均为 0，因此交叉熵损失也可写成：

Loss=−1n∑i=1n∑c=1CI(yi=c)log⁡P(yi=c∣xi)Loss=-\frac{1}{n}\sum_{i=1}^n\sum_{c=1}^CI(y_i=c)\log P(y_i=c|x_i) Loss=−n1​i=1∑n​c=1∑C​I(yi​=c)logP(yi​=c∣xi​)

二者是等价的表述形式

二分类任务损失函数

二分类任务常用二元交叉熵损失函数（Binary Cross-Entropy Loss，BCE）

L=−1n∑i=1n(yilog⁡y^i+(1−yi)log⁡(1−y^i))L=-\frac{1}{n}\sum_{i=1}^n (y_i\log \hat{y}_i + (1-y_i)\log(1-\hat{y}_i)) L=−n1​i=1∑n​(yi​logy^​i​+(1−yi​)log(1−y^​i​))

其中 yiy_iyi​ 为真实值，通常设为 0 或 1，y^i\hat{y}_iy^​i​ 为预测值（单个标量值），表示样本 iii 为 1 的概率

多分类任务损失函数

多分类任务常用多类交叉熵损失函数（Categorical Cross-Entropy Loss），它是对每个类别的预测概率与真实标签之间差异的加权平均

L=−1n∑i=1n∑c=1Cyt,clog⁡y^t,cL=-\frac{1}{n}\sum_{i=1}^n\sum_{c=1}^C y_{t,c}\log \hat{y}_{t,c} L=−n1​i=1∑n​c=1∑C​yt,c​logy^​t,c​

其中 CCC 是类别数，yt,cy_{t,c}yt,c​ 为真实值，表示 yiy_iyi​ 是否为类别 ccc，取值通常为 0 或 1，y^i,c\hat{y}_{i,c}y^​i,c​ 为预测值，表示样本 iii 为类别 ccc 的概率

MAE

平均绝对误差（Mean Absolute Error，MAE），也称为 L1 Loss：

L=1n∑i=1n∣yi−y^i∣L=\frac{1}{n}\sum_{i=1}^n|y_i-\hat{y}_i| L=n1​i=1∑n​∣yi​−y^​i​∣

L1 Loss 对异常值适应性较好，但在 0 点位置不可导

MSE

均方误差（Mean Squared Error，MSE），也称为 L2 Loss：

L=1n∑i=1n(yi−y^i)2L=\frac{1}{n}\sum_{i=1}^n(y_i-\hat{y}_i)^2 L=n1​i=1∑n​(yi​−y^​i​)2

L2 Loss 对异常值敏感，遇到异常值时容易发生梯度爆炸

Smooth L1

平滑 L1（Smooth L1）是 MAE 和 MSE 二者结合的版本，单个样本的 Smooth L1 函数表达式如下：

Smooth    L1={12(yi−y^i)2,    ∣yi−y^i∣<1∣yi−y^i∣−12,    ∣yi−y^i∣≥1Smooth\;\; L1= \left\{\begin{array} {l} \frac{1}{2}(y_i-\hat{y}_i)^2,\;\;|y_i-\hat{y}_i|<1\\ |y_i-\hat{y}_i|-\frac{1}{2},\;\;|y_i-\hat{y}_i|\ge1 \end{array}\right. SmoothL1={21​(yi​−y^​i​)2,∣yi​−y^​i​∣<1∣yi​−y^​i​∣−21​,∣yi​−y^​i​∣≥1​

当误差较小时（∣yi−y^i∣<1|y_i-\hat{y}_i|<1∣yi​−y^​i​∣<1）使用 L2 Loss，使得损失函数平滑可导；当误差较大时（∣yi−y^i∣≥1|y_i-\hat{y}_i|\ge1∣yi​−y^​i​∣≥1）使用 L1 Loss 降低异常值的影响

数值微分

损失函数的值越小，代表我们选取的参数越合适，而如果想求得损失函数的最小值，最基本的想法就是对函数进行求导，找到导数为 0 的点，并判断其是否为极小 / 最小值

实际情况中的损失函数往往比较复杂，直接求导很难得到解析解，此时我们考虑用数值微分的方式求解函数在某一点的导数

导数和数值微分

在数学上，导数被定义为：

f′(x)=df(x)dx=lim⁡h→0f(x+h)−f(x)hf'(x)=\frac{df(x)}{dx}=\lim_{h\to 0}\frac{f(x+h)-f(x)}{h} f′(x)=dxdf(x)​=h→0lim​hf(x+h)−f(x)​

这个定义中表达出了导数的本质，当 xxx 发生微小变化 hhh 或者 Δx\Delta xΔx 时，函数值 f(x)f(x)f(x) 也会发生变化，而当 hhh 无限趋近于 0 时，此时 f(x)f(x)f(x) 的变化率即为函数在 xxx 这一点处的导数值

利用这个定义，我们可以直接以数值计算的方式，利用微小的差分来求函数某点处的导数值，这种方法就称为数值微分

在实际应用中，由于函数都具有一定的凹凸性，我们无论是计算 x+h∼xx+h\sim xx+h∼x 还是 x−h∼xx-h\sim xx−h∼x 之间的割线斜率，都和真实的导数值之间存在一定的误差，因此我们考虑以 xxx 为中心，计算它两边各自发生微小变化后的差分，可以避免只计算单向增大或减少时产生的误差，这种方法称为中心差分，即：

f′(x)=df(x)dx=lim⁡h→0f(x+h)−f(x−h)2hf'(x)=\frac{df(x)}{dx}=\lim_{h\to 0}\frac{f(x+h)-f(x-h)}{2h} f′(x)=dxdf(x)​=h→0lim​2hf(x+h)−f(x−h)​

此外，取微小值 hhh 不能太小，否则会导致计算机浮点数表示的精度不够，出现舍入误差

偏导数

如果函数 fff 的自变量并非单个元素，而是多个元素，例如：

f(x,y)=x2+xy+y2f(x,y)=x^2+xy+y^2 f(x,y)=x2+xy+y2

可将其中一个元素 xxx 看作参数，此时 fff 可看作另一元素 yyy 的函数，即：

fx(y)=x2+xy+y2f_x(y)=x^2+xy+y^2 fx​(y)=x2+xy+y2

在 x=ax=ax=a 固定的情况下，可计算 fxf_xfx​ 关于 yyy 的导数：

fa′(y)=a+2yf_a'(y)=a+2y fa′​(y)=a+2y

这种导数就称为偏导数，一般记作：

δfδy(x,y)=x+2y\frac{\delta f}{\delta y}(x,y)=x+2y δyδf​(x,y)=x+2y

更一般地来说，一个多元函数 f(x1,x2,…,xn)f(x_1,x_2,\dots,x_n)f(x1​,x2​,…,xn​) 在 (a1,a2,…,an)(a_1,a_2,\dots,a_n)(a1​,a2​,…,an​) 处对 xix_ixi​ 的偏导数定义为：

δfδxi(a1,a2,…,an)=lim⁡Δxi→0f(a1,a2,…,ai+Δxi,…,an)−f(a1,a2,…,an)Δxi\frac{\delta f}{\delta x_i}(a_1,a_2,\dots,a_n)=\lim_{\Delta x_i\to0}\frac{f(a_1,a_2,\dots,a_i+\Delta x_i,\dots,a_n)-f(a_1,a_2,\dots,a_n)}{\Delta x_i} δxi​δf​(a1​,a2​,…,an​)=Δxi​→0lim​Δxi​f(a1​,a2​,…,ai​+Δxi​,…,an​)−f(a1​,a2​,…,an​)​

偏导数同样也可以用数值微分的方式来求，即只改变一个自变量，其他变量不变，作差分考察函数的变化率

梯度

多元函数 f(x1,x2,…,xn)f(x_1,x_2,\dots,x_n)f(x1​,x2​,…,xn​) 关于每个变量 xix_ixi​ 都有偏导数 δfδxi\frac{\delta f}{\delta x_i}δxi​δf​，在 α=[a1,a2,…,an]\alpha=[a_1,a_2,\dots,a_n]α=[a1​,a2​,…,an​] 位置处，这些偏导数定义了一个向量：

∇f(α)=[δfδx1(α),δfδx2(α),…,δfδxn(α)]\nabla f(\alpha)=[\frac{\delta f}{\delta x_1}(\alpha),\frac{\delta f}{\delta x_2}(\alpha),\dots,\frac{\delta f}{\delta x_n}(\alpha)] ∇f(α)=[δx1​δf​(α),δx2​δf​(α),…,δxn​δf​(α)]

这个向量称为 fff 在 α\alphaα 位置处的梯度

例如 f(x,y)=x2+xy+y2f(x,y)=x^2+xy+y^2f(x,y)=x2+xy+y2 在 (3,3)(3,3)(3,3) 位置的梯度为：

∇f(x,y)=[2x+y,x+2y]\nabla f(x,y)=[2x+y,x+2y] ∇f(x,y)=[2x+y,x+2y]

因此代入 x=3,y=3x=3,y=3x=3,y=3 则有：

∇f(3,3)=[9,9]\nabla f(3,3)=[9,9] ∇f(3,3)=[9,9]

此时我们就记：

δfδx1(3,3)=9,    δfδx2(3,3)=9\frac{\delta f}{\delta x_1}(3,3)=9,\;\;\frac{\delta f}{\delta x_2}(3,3)=9 δx1​δf​(3,3)=9,δx2​δf​(3,3)=9

即 fff 关于 x1,x2x_1,x_2x1​,x2​ 在 (3,3)(3,3)(3,3) 位置的偏导数分别是 9,99,99,9

显然，在函数的极小值，极大值或者鞍点处，梯度为 0

值得注意的是，梯度代表的其实是函数值增大最快的方向，在实际应用中，我们需要寻找损失函数的最小值，所以应该选择损失函数减小最快的方向，即负梯度方向

神经网络的梯度计算

神经网络中的梯度，值得就是损失函数关于权重参数的梯度

我们以一个单层的简单网络为例，其形状为 2×32\times32×3，权重参数为 WWW，损失函数记为 LLL，其权重矩阵为：

W=(w11w12w13w21w22w23)W=\left(\begin{matrix}w_{11}&w_{12}&w_{13}\\w_{21}&w_{22}&w_{23}\\\end{matrix}\right) W=(w11​w21​​w12​w22​​w13​w23​​)

则对应的梯度为：

∂L∂W=(∂L∂w11∂L∂w12∂L∂w13∂L∂w21∂L∂w22∂L∂w23)\frac{\partial L}{\partial W}=\left(\begin{matrix}\frac{\partial L}{\partial w_{11}}&\frac{\partial L}{\partial w_{12}}&\frac{\partial L}{\partial w_{13}}\\\frac{\partial L}{\partial w_{21}}&\frac{\partial L}{\partial w_{22}}&\frac{\partial L}{\partial w_{23}}\\\end{matrix}\right) ∂W∂L​=(∂w11​∂L​∂w21​∂L​​∂w12​∂L​∂w22​∂L​​∂w13​∂L​∂w23​∂L​​)

这里梯度 δLδW\frac{\delta L}{\delta W}δWδL​ 也是一个 2×32\times 32×3 的矩阵，其中各元素由 LLL 关于 WWW 中各元素的偏导数构成，即 (1,1)(1,1)(1,1) 位置为 LLL 关于 w11w_{11}w11​ 的偏导数，(1,2)(1,2)(1,2) 位置为 LLL 关于 w12w_{12}w12​ 的偏导数，以此类推

随机梯度下降法 SGD

梯度下降法

梯度下降法（Gradient Decent）是一种用于最小化目标函数的迭代优化算法，核心是沿着目标函数（如损失函数）的负梯度方向逐步调整参数，从而逼近函数的最小值，梯度方向指示了函数增长最快的方向，因此负梯度方向则是函数下降最快的方向

具体而言，我们初始时刻找到函数 f(x1,x2)f(x_1,x_2)f(x1​,x2​) 中自变量的一组初始值 (x1,x2)(x_1,x_2)(x1​,x2​)，作为梯度下降的起点，按下式进行更新：

x1′=x1−ηδfδx1x2′=x2−ηδfδx2x_1'=x_1-\eta\frac{\delta f}{\delta x_1}\\ x_2'=x_2-\eta\frac{\delta f}{\delta x_2}\\ x1′​=x1​−ηδx1​δf​x2′​=x2​−ηδx2​δf​

这里代入 δfδx1\frac{\delta f}{\delta x_1}δx1​δf​ 和 δfδx2\frac{\delta f}{\delta x_2}δx2​δf​ 的自变量值都是 (x1,x2)(x_1,x_2)(x1​,x2​)，但是由于它们分别是 fff 对 x1,x2x_1,x_2x1​,x2​ 求偏导得到的不同表达式，因此 x1,x2x_1,x_2x1​,x2​ 减去的偏移量也不同，这样我们就沿着梯度的负方向，找到一个新的点 (x1′,x2′)(x_1',x_2')(x1′​,x2′​)，使得函数值更小

这里的 η\etaη 表示每次的更新量，在神经网络的学习过程中，代表一次学习的步长（每次学习多少，用多大的程度去更新参数），称为学习率（learning rate）

学习率需要作为超参数预先设定好，过大或过小都会导致学习效果不佳：

  • 学习率过小，导致收敛缓慢，可能训练结束还未收敛到最优解
  • 学习率过大，导致发散，可能出现左右横跳的情况

模型训练相关概念

（1）Epoch

一个 Epoch 表示模型完整遍历一次整个训练数据集的过程，例如，训练 10 个 Epoch 表示模型将整个数据集反复学习 10 次，模型需要多次遍历数据集（训练多个 Epoch）才能逐步学习数据中的模式，单次遍历数据集（训练 1 个 Epoch）通常不足以让模型收敛，多次遍历可以逐步优化模型参数

（2）Batch Size

Batch Size 是每次训练时输入的样本数量，例如Batch Size=32表示每次使用 32 个样本计算一次梯度并更新模型参数

小批量数据计算比单样本（Batch Size=1）通常更稳定，比全批量（Batch Size=全体数据）通常更高效，并且较小的Batch Size可能带来更多噪声，有助于模型泛化

（3）Iteration

一次 Iteration 表示完成一个 Batch 数据的正向传播（预测）和反向传播（更新参数）的过程

例如，数据集现在有 2000 个样本，对其训练 10 个 Epoch，选择 Batch Size 为 64，则本次训练中：

  • 总的 Batch 数为 2000/64+1=32（最后一个 Batch 中只有 16 个样本）
  • 每个 Epoch 的迭代次数 Iteration 为 32 次
  • 总迭代次数为 10×32=320 次
  • 总训练样本数为 2000×10=20000 条

SGD

在神经网络的学习中，可以使用梯度下降法来更新参数，目标即减小损失函数的值

在实际操作中，一般会从训练数据中随机选择一个小批量数据（mini-batch），然后用梯度下降法迭代多个轮次（Iteration），这种对随机选择的数据进行的梯度下降法，就称为随机梯度下降法（Stochastic Gradient Descent，SGD）

随机梯度下降法的过程如下：

（1）随机选择批数据（mini-batch）

  • 从训练数据中随机选出一部分数据，学习的目标就是减少这个 mini-batch 中的数据对应的损失函数的值

（2）计算梯度

  • 对当前的各权重参数，计算出梯度的值，负梯度就表示了损失函数减小最多的方向

（3）更新参数

  • 根据梯度下降公式，对权重参数沿负梯度方向进行微小更新

（4）重复迭代

  • 重复上述（1-3）的步骤，直到完成预定的总迭代次数
