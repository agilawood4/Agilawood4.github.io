Matplotlib 基础

面向对象绘图

面向对象绘图适合复杂的自定义绘图场景（比如需要在一张画布内绘制多张子图），其包含两个核心对象：

  • figure 对象：画布，负责整个图表的全局属性和布局
  • axes 对象：子图，负责具体某一张图表的内容，相当于画布上的一个绘图区域

一个 figure 可以包含多个 axes

创建对象的方法：

  • 分别创建 figure 和 axes
  • 同时创建 figure 和 axes

在创建画布的时候，可以对画布进行尺寸设计，传入参数 figsize=(W, H)，分别表示画布的宽度和高度，以英寸为单位，默认尺寸为 6.4 × 4.8 英寸^2^

标题、网格线与轴标签

画布标题 suptitle

作为整个图形的主标题，通常位于顶部中央，当一个图形包含多个子图时，用于概括所有子图的主题，有两种实现方式：

  • fig.suptitle()：面向对象的实现方式
  • plt.suptitle()：单张子图的实现方式

这两个方法接收常见参数如下：

  • 第一个 str 参数：标题字符串
  • fontsize：字号大小，通常为 10 到 15
  • fontweight：字体粗细，通常为 100 到 900 取整百

注意后两个参数只有设置 rcParams 中字体添加思源黑体以后才能够使用

子图标题 title

描述单个图表的具体内容，位于每张图表的上方，有两种实现方式：

  • ax.set_title()：面向对象的实现方式
  • plt.title()：单张子图的实现方式

这两个方法接收常见参数和画布标题对应的函数相同，除此之外，我们还可以设置一个 y 参数，用于控制标题的高度（距离图表上缘的距离）

轴标签 xylabel

用来指明 x 轴和 y 轴上坐标的含义，有两种实现方式：

  • ax.set_xlabel()，ax.set_ylabel()：面向对象的实现方式
  • plt.xlabel()，ply.ylabel()：单张子图的实现方式

除了传入 str 作为轴标签的内容，我们还可以额外设置一个 labelpad 参数，用于控制轴标签和坐标轴之间的距离

网格线 grid

在图表中添加水平和 / 或竖直的参考线，帮助用户更准确地读取数据点的值，有两种实现方式：

  • plt.grid()：面向对象的实现方式
  • ax.grid()：单张子图的实现方式

这两个方法接收如下常见参数：

  • linestyle：网格线的类型，常见的有 –，-，-. 等
  • axis：both 表示横轴和纵轴都绘制网格线，x 表示只绘制横轴网格线，y 表示只绘制纵轴网格线
  • alpha：网格线透明度，越接近 0 越偏透明，越接近 1 越偏实线

线条样式与图例

线条样式修改

在使用 ax.plot() 或 plt.plot() 进行绘图时，我们可以传入以下参数，控制绘图所用的线条样式：

  • linestyle：线型，- 表示实线，– 表示虚线，-. 表示点线
  • linewidth：线宽，默认为 1
  • color：线的颜色，可以传常见颜色单词 red，blue，wihte 等，也可以传十六进制颜色编码
  • marker：线上关键节点的类型，常见设置为 o 或者 .

图例 legend

标识不同数据系列的含义，比如不同颜色或线型所代表的类别，有两种实现方式：

  • ax.legend()：面向对象的实现方式
  • plt.legend()：单张图表的实现方式

注意，在使用 plot 进行绘图时，传入作为图例的 label 标签默认不会显示，只有调用 legend 方法以后才会显示，此外，该方法接收两个常见的位置参数，用于控制图例在图中的位置：

  • loc：best 默认，upper right 右上，upper left 左上，lower right 右下，lower left 左下，center 图标中心，center right 右侧中心等
  • bbox_to_anchor：传入一个元组 (x,y)，把图标视作坐标系，(0,0) 对应右下角，(1,1) 对应右上角，实际传入的坐标范围可以超过 [0,1]，即图例布置在图表之外
  • fontsize，fontweight，color 等和文字有关的设置

从底层来看，legend 方法会对我们绘图时所传入的所有 label 参数进行收集，最后统一展示

轴范围、轴刻度与水平垂直线

轴范围 xylim

用于设置 x 轴和 y 轴坐标范围的最小值和最大值：

  • ax.set_xlim()，ax.set_ylim()：面向对象的实现方式
  • plt.xlim()，plt.ylim()：单张图表的实现方式

这些方法都接收一个元组作为参数，即 (min_value, max_value)，分别表示坐标轴的最小坐标和最大坐标

轴刻度 xyticks

设置坐标轴上显示的刻度：

  • ax.set_xticks()，ax.set_ytiks()：面向对象的实现方式
  • plt.xticks()，plt.yticks()：单张图表的实现方式

后者支持先后传入两个列表，分别表示要保留显示在坐标轴上的刻度，以及这个刻度可能对应的标签，比如：

如果从传入标签的角度来看，先传入的一个列表相当于指定了标签在坐标轴上对应的刻度，比如plt.xticks([1,2,3], ['A','B','C'])则表示在原图刻度为 1 2 3 的位置分别生成标签 A B C，plt.xticks() 和 plt.yticks() 支持二者同时设置

而 ax.set_xticks() 和 ax.set_yticks() 则只支持设置在原图上要保留的刻度，而设置这些刻度对应的标签则需要通过 ax.set_xtickslabel() 和 ax.set_ytickslabel() 方法实现，即需要分两步实现：

  • 注意，ax.set_xticks() 并不支持接收 rotation 参数，对应 y 轴操作的函数都同理

水平垂直线 axvline，axhline，hlines，vlines

在图表中标记特定的 Y/X 值的位置，用户标记关键时间点或事件：

  • ax.axvline()，ax.axjline()：面向对象的实现方式
  • plt.axvline()，plt.axhline()：单张图标的实现方式

其中 v 表示 vertical，即绘制垂直线，h 表示 horizontal，即绘制水平线，上述方法接收以下参数：

  • 位置参数 x 或 y：坐标轴上的位置，基于某个坐标进行绘制
  • 线型 linestyle，线宽 linewidth，透明度 alpha，颜色 color 等
  • 标记 label，用于在图例中进行展示

这是相对于整张子图绘制水平、垂直线的方法，如果我们仅想在某段区间内进行绘制，则对应需要使用 ax.hlines 和 ax.vlines 方法：

以水平线为例：

  • y（必填）：线的高度，对齐 y 轴的哪个刻度
  • xmin（可选）：线的左端点横坐标
  • xmax（可选）：线的右端点横坐标

注意：Matplotlib 中所有和线条绘制有关的函数，都接收类似线型 linestyle，线宽 linewidth，透明度 alpha，颜色 color 等参数，用于调整绘图曲线的风格

文本 text

用于在图表中添加自定义注释或解释：

  • ax.text()：面向对象的实现方式
  • plt.text()：单张图表的实现方式

该方法接收如下的常见参数：

  • x, y, s：分别表示要添加的文字所处的横坐标，纵坐标，以及文字内容
  • transform：决定 x, y 用什么参考系来定位文本
    • ax.transData：数据坐标系（默认），即 x, y 对应图表中实际的数据刻度
    • ax.transAxes：子图百分比坐标系，x, y 取值范围为 0~1，代表文本所处在子图百分比的位置
  • ha：horizontal-alignment，水平对齐方式，以传入的 x 坐标为基准，left 表示文本左侧和 x 对齐，right 表示文本右侧和 x 对齐，center 表示文本中部和 x 对齐
  • va：vertical-alignment：竖直对齐方式，以传入的 y 坐标为基准，同理
  • 字号 fontsize，子型 fontweight，字体颜色 color 等参数

注意：Matplotlib 中所有和文字绘制有关的函数，都接收类似字号 fontsize，子型 fontweight，字体颜色 color 等参数，用于调整文字的风格

注释与箭头 annotate

注释用于在 Matplotlib 图表中标记重要数据点，可以通过带箭头的文本，清晰突出关键信息（比如特定时间点，峰值等），使用如下方法实现：

各参数含义如下：

  • string：用于显示的文本
  • datapoint coordinate：对应 xy 参数，传入一个 (x, y) 元组，表示被注释的数据点的坐标，即箭头指向的位置
  • text coordinate：对应 xytext 参数，传入一个 (x, y) 元组，表示注释文本的坐标，即箭头起始的位置
  • arrow style dictionary：箭头样式设置，建议直接复制官网模版，比如arrowprops=dict(arrowstyle="->", connectionstyle="angle3,angleA=0,angleB=-90"))
  • text formatting：文本格式设置，用于定义字号，字型，颜色等

边框隐藏与轴关闭

边框隐藏 spines

边框用于控制图表四周的边界线（top，right，left，bottom），移除特定边框可以简化图标外观，使得图标更加简洁：

  • ax.spines[side]：面向对象实现方式
  • plt.spines[side]：单张图表实现方式

可以通过设置 ax.spines[边框位置].set_visible(False) 实现边框隐藏

轴关闭 axis

通过设置 ax.axis(‘off’)* 即可关闭坐标轴，即坐标轴上不会显示任何刻度和标签
