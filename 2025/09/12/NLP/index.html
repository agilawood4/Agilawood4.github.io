词嵌入和word2vec

词嵌入和词向量指的是同一个概念，指的是把一个单词（word）转化为一个能够表示这个单词的向量，词嵌入是许多深度学习模型的基础

one-hot 编码

one-hot 编码（独热编码）是一种对分类数据进行编码的方法，独热编码给每个类别分配了一列，属于该类别的该列为 1，其他列为 0

独热编码的缺陷：

  • 独热编码属于词袋模型，丢失了词原本的顺序信息（无法表示上下文关系）
  • 任意两个词向量之间都是正交关系，难以表示词义
  • 如果待表示的词很多，则需要很高维的向量，同时每个向量只有一个位置为 1，构成稀疏矩阵浪费空间

基于独热码的缺陷，我们提出了两种改进的 word2vec 方法，这两种方法能够同时学习到一个词在语法和语义上的特征：

  • CBOW：连续词袋模型
  • Skip-gram：跳元模型

连续词袋模型

CBOW（Continuious Bag of Words）连续词袋模型：通过上下文（周围的词）来预测中心词（target word），给定一个串口大小为 C，我们选择中心词 yjy_jyj​ 的前 C/2 和后 C/2 个词作为上下文

模型要学习的就是，在给定上下文的条件下，预测中心词的概率最大，即目标是让 P(yj∣x1k,x2k,…,xCk)P(y_j|x_{1k}, x_{2k},\dots,x_{Ck})P(yj​∣x1k​,x2k​,…,xCk​) 最大

输入层到隐藏层

输入层 x 是上下文中词语的独热编码，也就是 x~1k~，x~2k~，…一直到 x~Ck~，其中每一个 x~ik~ 都是 V 维的向量（假设词表大小为 V，每个向量只有一个位置为 1）

对于每一个输入，我们设置共同的 V×N 维的权重矩阵 W~V×N~，通过 xik∗WV∗Nx_{ik}*W_{V*N}xik​∗WV∗N​ 矩阵乘法，把它转化层一个 N 维的向量，然后我们把这 C 个输入进行加权平均，也就是：

h=Wx=W1C(x1k+x2k+⋯+xCk)=1C∑t=1CWxtkh=Wx=W\frac{1}{C}(x_{1k}+x_{2k}+\cdots+x_{Ck})=\frac{1}{C}\sum_{t=1}^CWx_{tk} h=Wx=WC1​(x1k​+x2k​+⋯+xCk​)=C1​t=1∑C​Wxtk​

得到一个中间层 N 维的向量 h，其中 V 表示词表大小，N 表示词向量维度，这里 h 就是隐藏层 / 中间层的输出，表示上下文的综合语义表示

隐藏层到输出层

我们设计一个 N×VN×VN×V 维的矩阵 WWW，该矩阵的第 j 列 wjw_jwj​ 表示词表中第 j 个词的输出向量（output embedding），通过 h∗W′h*W'h∗W′ 得到最终的输出向量 yyy，其中 yyy 的每个分量就对应词表中一个词的得分，而我们的目标函数选用 softmax 函数，也就是将 yyy 输入 softmax，得到对整个词表中每个词的*概率分布：

P(yj∣x1k,x2k,…,xCk)=softmax(yj)=eyj∑i=1VeyiP(y_j|x_{1k},x_{2k},\dots,x_{Ck})=softmax(y_j)=\frac{e^{y_j}}{\sum_{i=1}^V e^{y_i}} P(yj​∣x1k​,x2k​,…,xCk​)=softmax(yj​)=∑i=1V​eyi​eyj​​

目标函数

训练目标就是让真实中心词的概率最大，即最大化似然：

max⁡∏P(yj∣x1k,x2k,…,xCk)\max \prod P(y_j|x_{1k},x_{2k},\dots,x_{Ck}) max∏P(yj​∣x1k​,x2k​,…,xCk​)

跳元模型

Skip-Gram（跳元模型）的思想与 CBOW 恰好相反：它是根据中心词的输入预测周围一组上下文的 C 个词出现的概率

以句子中第 k 个词为中心词 xkx_kxk​（独热编码表示，词表大小为 V），以窗口大小 C 取它左右的上下文集合 Sk=−c,⋯−1,1,…,cS_k = {-c,\dots -1,1,\dots,c}Sk​=−c,⋯−1,1,…,c，目标是在给定中心词时，每个上下文词出现的条件概率尽量大，也就是考虑最大化P(y1j,y2j,…,yCJ∣xk)P(y_{1j},y_{2j},\dots,y_{CJ}|x_k)P(y1j​,y2j​,…,yCJ​∣xk​)

这个条件概率可以看成近似计算：

P(y1j∣xk)∗P(y2j∣xk)∗⋯∗P(yCj∣xk)=∏P(yjt=wk+t∣xk)P(y_{1j}|x_k)*P(y_{2j}|x_k)*\cdots*P(y_{Cj}|x_k)=\prod P(y_{j_t}=w_{k+t}|x_k) P(y1j​∣xk​)∗P(y2j​∣xk​)∗⋯∗P(yCj​∣xk​)=∏P(yjt​​=wk+t​∣xk​)

上式把联合概率近似为独立相乘，能够在保证精度的基础上简化模型计算

输入层到隐藏层

我们设计 V×NV×NV×N 的向量矩阵 W，通过 h=WTxk=vwkh=W^Tx_k=v_{w_k}h=WTxk​=vwk​​，直接抽取中心词的向量

我们设计 N×VN×VN×V 的输出矩阵（输出嵌入），对词表红每个词得到打分 y=WThy=W^Thy=WTh，取 softmax 得到某个词是当前要预测的上下文词的概率：

P(y=j∣xk)=eyj)∑i=1Vexp⁡(uiTvwk)P(y=j|x_k)=\frac{e_{y_j)}}{\sum_{i=1}^V\exp(u_i^Tv_{w_k})}P(y=j∣xk​)=∑i=1V​exp(uiT​vwk​​)eyj​)​​
