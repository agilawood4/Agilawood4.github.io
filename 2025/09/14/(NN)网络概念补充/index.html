神经网络补充概念

通过阅读本文，您可以在神经网络的基础模型上，大致了解和网络的欠拟合与过拟合，正则化，Dropout，梯度消失与梯度爆炸等有关的内容

训练常见问题

万能近似定理

万能近似定理（Universal Approximation Theorem）：一个具有足够多的隐藏节点的多层前馈神经网络，可以逼近任意连续的函数（前提是网络中至少有一层具有非线性空间变换性质的激活函数，如 Relu，Sigmoid 等）

这也是后面，我们的网络能够学习各种类型输出的基础

网络深度的影响

大量实验表明，增加模型深度可以提高模型的泛化能力，这是基于下面的猜想：无论是多么复杂的函数，都可以拆分成若干个简单函数的组合表示，而我们每增加网络的一层结构，即可捕获函数的部分特征信息，多层结构进行叠加，即可较好拟合出目标映射函数的全部信息

过拟合与欠拟合问题

过拟合问题（Overfitting）：过拟合问题指的是模型在训练数据上表现良好，但是在测试数据上表现不佳的现象

泛化能力：训练后的模型应用到新的，未知的数据上的能力

当模型出现过拟合时，即说明模型的泛化能力不足，一般都是由于模型的复杂度过高所导致的

欠拟合问题（Underfitting）：欠拟合问题指的是模型的学习能力不足，无法学习到数据集中的一般规律

产生愿意：一般是由于模型学习能力较弱，而数据复杂度较高所导致

模型复杂度与过拟合、欠拟合之间的关系：

过拟合与欠拟合问题

过拟合与欠拟合问题本质上都是数据和模型之间的匹配问题（过拟合 - 模型匹配训练数据过强，欠拟合 - 模型匹配训练数据过弱），我们可以从以下几个角度考虑解决：

  • 数据集
  • 模型本身（损失函数）
  • 训练过程

数据集复杂度

数据集过小时，非常容易出现过拟合问题，而数据集过大则可能导致训练效率降低，因此我们需要适当增强数据集的大小

数据增广

实际训练中，我们很难找到充足的数据，因此就需要对有限的数据做数据增广，也就是对原有训练数据进行变换（旋转，剪裁，引入噪声等），增加数据数量和多样性

通过数据增强，可以有效解决过拟合问题，提高模型在新数据上的泛化能力

引入验证集

验证集（validation set）也属于训练过程，此时我们可以把整个数据集划分为三部分：

  • Training set：用于训练出一个完整的模型
  • Validation set：用于在训练阶段即对模型进行对比评估，调整超参数，得到较优的模型
  • Test set：在新数据上，测试模型的性能

模型选择

过于简单的模型可能出现欠拟合问题，而过于复杂的模型可能出现过拟合问题，根据奥卡姆剃刀法则，我们选择简单、合适的模型解决复杂问题即可

实验数据表名，一定程度上增加模型中网络层的层数（模型深度），有助于提高模型的性能

K折交叉验证

我们可以把训练数据均分为 K 份，选择其中的一份作为验证集，剩下的 K-1 份作为训练集，因此我们可以进行 K 轮训练（分别选择第一份，第二份，…，第 K 份作为验证集），对这 K 轮验证中，模型的表现结果得分取平均值，即可作为当前模型验证的得分

早停机制

提前终止（Early stopping）：模型对训练数据集迭代收敛之前就停止迭代来防止过拟合

此时验证集承担一种预警的机制，随着我们在训练集上训练轮次的增加，Training loss 逐渐减少，但 Validation loss 可能先减少后增加，当我们在验证集上验证时，如果发现验证损失开始增加，我们就应当及时停止训练，因为此时再训练下去很可能就会发生过拟合

正则化

通过在训练目标（损失函数）中加入某些约束项或者偏好项，可以有效抑制过拟合，提升模型泛化能力的方法，统称为正则化方法（Regularization），通常以在经验风险后添加惩罚项，或者改造训练过程的形式实现

添加正则项后损失函数的一般形式为： $$ \min L_{train}(\theta)+\lambdaΩ(\theta) $$ 其中 L 为训练损失（经验误差或标准误差），λ 为正则强度，Ω 为正则项，某些意义上，数据增强也可以认为是一种正则化方法

正则化是根据实际情况对学习算法的一种修改，目的是减少泛化误差，而不是训练误差

从狭义上来讲，正则化指的是通过在损失函数中添加正则项的方法（L1，L2正则），而从广义上来讲，所有能够减小训练误差，提高泛化能力的方法都可以认为是正则化方法

没有免费午餐定理：没有一种算法或者模型能够在所有的场景中都表现良好，正则化只是一种在特定数据集上，用于权衡过拟合和欠拟合的手段

L2正则化

L2 正则化也称为岭回归，通过给模型的损失函数添加一个模型参数的平方和的惩罚项来实现正则化，比如假设模型中用到的参数为 ω1, ω2, …, ω3，则 L2 正则化的表达式如下： $$ Loss = Loss_{original}+\lambda\sum_{i=1}^n\omega_i^2 $$ L2 正则化用于惩罚大权重，因为我们使用的参数越大，正则化项的代价就会越高，即 L2 正则化会引导模型使用更小，更平滑的参数去解释数据，从而抑制过拟合现象

从几何视角来看，L2 正则化相当在经验风险的等高线（椭圆）处于一个以原点为球形的 L2 球（θ : ∥θ∥22 < r）在交点处取解，下图是以两个参数为例的二维可视化：

L1正则化

L1正则化又称为 LASSO 回归，通过在损失函数中加入对模型中所用到的各个元素的绝对值之和的惩罚项，来限制模型参数的值，L1 正则化的表达式如下： $$ Loss = Loss_{original}+\lambda\sum_{i=1}^n(|\omega_i|) $$ L1 正则化的主要目的是使得模型参数矩阵中为 0 的位置尽可能增加，也就是减少可能的参数数量，从而实现参数矩阵的稀疏化，L1 正则化有时也被称为 L1 范数正则化

几何解释：L1正则项（∥θ∥1 < t）的可行域在二维中是一个菱形（四个尖角在坐标轴上），在高维是交叉多面体（cross-polytope），有大量尖角，每个尖角对应某些坐标恰好为 0；训练损失 L(θ) 的等高线通常是光滑的椭圆

在 L1 正则化约束下的最优解就是椭圆等高线向外扩，与 L1 菱形边界首次相切的点，由于边界有很多尖角（拐点），二者常常在尖角处相切，故而自然产生稀疏解（参数落到坐标轴上变为 0），而 L2 正则化约束的便捷是圆 / 高维球，边界处处光滑，因此对参数只收缩而不稀疏

模型对比：

  • L1 正则化更倾向于产生稀疏解，适于特征选择
  • L2 正则化更倾向于小的非零权值，更适用于优化问题

范数惩罚

L1正则和L2正则可以扩展到一般情况，其中 Ω 函数可以设置为任意正则化函数： Loss = Lossoriginal + λΩ(ω) 在神经网络模型中，一般参数有 ω 和 b 两种（y = f(wx+b)）我们一般只对 ω （权重矩阵）做正则化，而不对 b（偏置）进行处理，因为前者拟合所需要的数据量往往远大于后者，对于 b 的拟合不需要额外处理，一般就能获得很好的拟合效果

权重衰减

权重衰减（Weight Decay）也可以认为是一种正则化方法，只不过它不是作用于损失函数，而是直接修改最优化过程中参数迭代的方程： ωt = (1−λ)ωt − 1 − αΔL 其中 ωt 是当前时刻的参数矩阵，ωt − 1 是上一轮训练得到的参数矩阵，ΔL 则为损失函数（Loss function）关于权重参数 ω 的偏导数，我们通过在当前参数矩阵的更新过程中，加入对上一轮参数矩阵的考虑，此时我们称这种正则化方法为权重衰减

可以证明，在使用 SGD（随机梯度下降）作为 Loss function 时，权重衰减最终效果等价于 L2 正则化（二者只是最终表达式相同，前者添加的是 (1−λ)ωt − 1，后者修改的是损失函数，二者有本质区别）

Dropout方法

工作原理：在训练过程中随机删除（即将其权重设置为 0）一些神经元，从而使得模型不能依赖某些特定的特征，这样可以降低模型对训练集的依赖程度，从而提高模型泛化能力

注意 dropout 只能使用在训练期间，不能使用在测试期间

主要步骤

进行 Dropout 处理时，我们一般需要遵循以下步骤：

  • 制定一个保留比例 p
  • 对于每一层的每一个神经元而言，以 p 的概率进行保留，以 1-p 的概率把权重设置为 0
  • 训练过程中，仅使用被保留的神经元进行前向、反向传播
  • 在测试过程中，把每个神经元的权重都乘以 p，这里主要是从期望的角度进行考虑，在训练的时候，每个神经元都会有 p 的概率被保留，那么总体参数贡献的期望就是 $\sum_{i=1}^n w_iE(m_i)h=\sum_{i=1}^npw^Th$，而测试的时候，总体参数贡献的期望为 $\sum_{i=1}^n w^Th$，为了保证测试时的激活幅度和训练时的期望一致，我们需要给测试也乘以一个对应的权重 p，避免分布漂移

从直观上理解，Dropout 相当于把一个网络进行拆分，每次都取一个子网络（由原网络部分节点构成的网络）进行训练，然后让这些子网络进行集成学习

为什么能减少过拟合

Dropout 本质上是又放回的集成学习，可以起到平均化作用（节点随机选择，使得上一轮训练中依赖的关系，可能在下一轮训练中不会再次存在，会驱动网络学习更加全局的信息）；同时减少了神经元之间的复杂关系；类似于性别在生物进化中的角色（奇妙の比喻）

优点：

  • 可以有效减少过拟合
  • 简单方便
  • 实用有效

缺点（损失都在可接受范围内）：

  • 降低训练效率
  • 损失函数不够明确

代码实现

在 Pytorch 中已经为我们封装好了现成的dropout函数，我们只需要引入from torch import nn模块，直接调用nn.Dropout(p)传入每个神经元节点被保留的概率值 p 即可

需要注意的是，我们通常把nn.Dropout()层放在全连接层或者线性层nn.Linear()层的后边，这样能起到最好的训练效果

完整代码见文件：dropout.ipynb

梯度消失与梯度爆炸

### 梯度的重要性

实验表明，在处理复杂问题时，一定程度上增加网络层的深度，可以提高模型的性能，而目前优化神经网络的策略大都是基于反向传播的思想，即根据损失函数计算误差通过梯度的反向传播，来指导神经网络权值参数的更新和优化

整个深度神经网络可以视为一个非线性多元函数，优化模型就是需要找到合适的权重，从而最小化损失函数 Loss = ∥g(x) − f(x)∥22，其中 g(x) 为实际输出，f(x) 为网络模型的预测输出

如上图所示，我们需要找到损失函数的最小值，对于这种在局部 / 全局寻找函数最小值的的问题，非常适合使用梯度下降法进行解决

反向传播的内在问题

反向传播的核心在于链式求导法则，链式求导法则本质就是激活函数的偏导数的连乘，比如： $$ \frac{\delta L}{\delta w_2}=\frac{\delta L}{\delta f_4}\frac{\delta f_4}{\delta f_3}\frac{\delta f_3}{\delta f_2}\frac{\delta f_2}{\delta w_2} $$ 其中 f1 f2 f3 f4 即为各个隐层的激活函数，如果存在多项偏导数的值都较大，则他们相乘的结果就会呈指数级别增长，也就是出现梯度爆炸的现象，而如果存在多项偏导数的值都较小，则他们相乘的结果就会无限接近于 0，也就是已出现梯度消失的现象

梯度消失

选择不合适的激活函数（比如 sigmoid 函数）就极有可能造成梯度消失的现象：

如左图所示，sigmoid 函数的导数在自变量范围稍大的时候，就会无限接近于 0，从而导致多项极小项连乘，造成梯度消失，一种改进的思路是使用 tanh 激活函数，即如右图所示

梯度爆炸

梯度在更新中会发生累积，可能会变得非常大，导致网络不稳定，具体原因有二：

  • 网络层数过深，多个损失函数较大的偏导值相乘
  • 初始化权重的值过大

模型无法在训练过程中获得进一步的更新，模型在训练过程中损失函数值变为 NaN（损失溢出），模型变得不稳定，这些都是可能发生梯度爆炸的信号

应对策略

为解决梯度消失或梯度爆炸问题，我们可以由以下几种常见的应对策略：

  • 预训练+微调（笨办法，现在大多已经弃用）
  • 梯度裁剪、正则
  • ReLU 激活函数
  • Batchnorm
  • 残差结构

梯度裁剪

设置一个梯度裁剪的阈值，超过这个阈值则将其强制限制在这个范围之内

ReLU激活函数

我们从链式求导法则的表达式角度来看，如果每个激活函数的偏导数值都恒为 1，那么若干个偏导数相乘，永远不会发生梯度消失或者爆炸的现象，为此我们提出了 ReLU 激活函数

该函数的导数在整数部分恒定为 1，无梯度消失

Batch Normalization

Batch Normalization（简称 BN）又称为批规范化，其核心思想在于将输出信号规范化到均值为 0，方差为 1，从而减少数据的分散程度，保证网络的稳定性

在多层神经网络中，不同层参数的更新可能会改变后续层输入的分布情况，从而使得训练过程出现抖动，BN 能够把每一层的输入向左标准化，从而使得梯度尺度更加稳定

同时，从现代深度学习观点来看，BN 改变了网络的参数化方式，让损失曲面更加平滑，尺度不敏感，优化更加容易

批归一化操作的对象是输入在某个特定维度上取出的标量或者向量：

  • 对于全连接层（MLP）而言，输入形状为（N, D），其中 N 为 batch_size，D 是特征维度，对于每一个特征维度 d，BN 会在 batch 维上做标准化，此时的 xi 就是第 i 个样本在该特征维度上的标量
  • 对于卷积层（CNN）而言，输入形状为（N, C, H, W），BN 按照通道 C 进行归一化，也就是对同一通道中的所有像素和 batch 样本统计，此时 xi 可以看成是某个通道上取到的一个像素值，依然是标量

残差结构

残差网络能够通过跨层连接结构，从而解决梯度消失问题，其核心结构在于给不同的网络层之间连接残差捷径

如下图所示，在从 x 传递到最终输出的时候有两条路径，一条是经过正常的网络层权重矩阵，另一条则是通过侧面的残差捷径，则 $\frac{\delta x_L}{\delta x_l}$ 则可以拆分成两项，一项是正常网络层求偏导的结果，一项是对残差捷径求偏导的结果，也就是常数 1： $$ \frac{\delta x_L}{\delta x_l}=(1+\frac{\delta }{\delta x_L}\sum_{i=1}^{L-1}F(x_i, W_i)) $$ 残差结构在目前神经网络中应用即为广泛，几乎所有的网络都离不开残差结构，残差网络可以轻松构建成百上千层的神经网络，并且不需要担心梯度消失问题
