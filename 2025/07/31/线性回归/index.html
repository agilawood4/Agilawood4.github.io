线性算法

通过阅读本文，您可以大致了解机器学习中和线性算法相关的知识

智若禹行水，道如丁解牛

线性模型的关键就是找到一条线，这条线可以是直线也可以是曲线：

  1. 如果想让尽可能多的数据点都分布在这条线上 -> 回归任务
  2. 如果想尽可能把不同的点分在线的两边 -> 分类任务

线性回归思想与原理

线性模型和 KNN 模型的区别：

在分类层面：KNN 算法一般所分的都是离散的特征，比如 A 类 B 类 C 类三个不同的类别，因此我们可以用形如颜色这样的标记进行类别划分；而线性算法所处理的一般都是连续的特征，比如房价（浮点数价格），因此，没法用打标签这样的方式进行划分，必须单独占用一个坐标轴，也就是右图中的输出向量，我们可以认为，在可视化层面上，线性算法比 KNN 算法少了一个维度，如果右图的线性算法也想像左图的 KNN 算法一样以两个特征维度进行处理，则需要用到三维坐标系

在回归层面：KNN 算法偏重于以空间位置关系作为考虑，而线性算法则更能体现顺序关系（比如随着时间的推移，或者随着某个特征参数的线性变化），这更体现出一种顺势而为的思想，所以线性模型一般更多的用于回归任务中

一元线性回归

所谓一元线性回归，就是所有数据点只有一个特征的情况（有多个特征值时称为多元线性回归），此时用二维坐标即可表示

在一元线性回归问题中，我们希望找到一条最优的直线，最优性的评判规则如下：对于每一个点 p，我们把 p 的横坐标代入这条直线得到纵坐标 y1，这个点真实的纵坐标为 y，则我们用 (y - y1)^2^ 表示这一个点 p 的误差，我们求出所有的节点对应的误差 Σ(y - y1)^2^，使得这个误差值最小的直线就是最优的直线

我们把直线方程 y1 = kx + b 代入到 y1 的表达式中，则我们要求解的参数 k 和 b 就是使得目标求和式最小的参数，这个求和式一般也称为损失函数（loss function）

接下来就是训练模型，找到最优的 k 和 b 参数的过程，这一过程可以通过编程简单实现，称为参数学习，而我们找到的最优化参数一般是凸优化情况下的参数

多元线性回归

当数据有多个特征的时候，我们可以把这些特征组合成一个向量

X = (x1, x2, x3, ……, xn)

这里每一个 xi 都是数据的一个特征，标签 y 需要由这些特征共同表示

y = w0 + w1x1 + x2x2 + …… + wnxn

仍然可以用二维坐标系表示，只不过横坐标成为一个向量 X，斜率系数 K 变成一个矩阵，也就是

y = w0 + x × w^T^，其中 x 和 w 是 n 维向量，如果我们给特征向量 X 中加入一个恒为 1 的 x0，把这个 x0 和我们前面的 w0 进行组合，那么最终的表达式就可以简化成：

y = x × w^T^，此时 x 和 w 都成为 n+1 维的向量

这里需要注意的是，我们之所以在多元线性回归中把横轴 X 看成是一个包含了 k 维特征的向量，主要是为了方便可视化，因为当数据有 > 2 个特征的时候，我们是无法在更高维空间中绘图的，进行特征压缩，可以保证我们始终保持在二维坐标系中

逻辑回归思想与原理

线性逻辑回归

线性回归（Liner Regression）一般用于解决回归任务（预测），而逻辑回归（Logistic Regression）一般用于解决分类任务

逻辑回归能解决的一般都是二分类问题（只有两个类别的分类问题）

我们所建立的逻辑回归模型的核心就是 f 函数，它会把每个数据的特征向量 X 映射到一个概率值 [0, 1]，我们判断的标准为 p >= 0.5 时，就划分为标签 1 的集合，p < 0.5 时，就划分到标签 2 的集合

现在的问题是，我们怎么保证 f 映射之后，值域的范围一定是 0 到 1 之间，其实 f 本身是没法保证的，这里我们需要额外套一层逻辑函数，用于实现分类至于的二次映射，把 [-inf, inf] 的范围再次映射回 [0, 1] 如下图所示：

能够实现这种值域映射的函数，我们就称之为逻辑函数（Logistic Function），最常见的逻辑函数就是 sigmoid 函数（S型函数），其表达式为：

g(t) = 1 / ( 1 + e^-t^ ) ->

逻辑回归的损失函数

使用逻辑回归需要处理预测值和真实值的统一性问题，数据真是的标签就是 0 和 1 两类标签，而我们模型预测的结果是一个 [0, 1] 之间的概率值，二者不具有可比性，我们没法像线性回归那样用真实值 - 预测值直接作差的方式求解

这里我们考虑根据真实值分类：

  1. 真实值 y = 1 时，预测概率 p 越小，误差越大；预测概率 p 越大，误差越小
  2. 真实值 y = 0 时，预测概率 p 越大，误差越大；预测概率 p 越小，误差越小

同时我们希望模型分类错误的时候，给出较大的惩罚（比如 y = 1，p = 0 或者 y = 0，p = 1 的情况），因此最终找到的损失函数为：-ylog(p) - (1-y)log(1-p)

其中 y 是某个数据的真实分类标签，p 是通过 f 分类，g 二次映射得到的概率值，y 值其实只能取得 0 或 1

上面的石子只是我们设计的模型，对于单个数据而言的损失函数，而模型整体的损失函数和全部的 m 个数据有关，因此我们需要对每个数据的损失先求和再取平均，得到最终的损失函数：

在逻辑回归中，通过损失函数求解模型参数的过程，无法像线性回归那样通过方程求解的方法实现（对表达式求导，求最值这种），只能通过梯度下降的方式实现，而该损失函数是只有一个极值点的下凸函数，因此使用梯度下降的求解效果一般会很好

多项式逻辑回归

在之前的逻辑回归中，我们都假设数据按线性分布，如果数据本身分布就是非线性的，我们需要把分类的线从直线，变成曲线（甚至有可能是闭合曲线）

此时我们找到的分类曲线就接近于圆的方程：x^2^ + y^2^ = r^2^，对于这类方程的求解，我们同样采用变量替换的方式：

  • x^2^ -> t1
  • y^2^ -> t2

这样我们再次把高维的逻辑回归降维成为线性的逻辑回归，其实只要我们能写出分类 / 拟合曲线表达式的回归问题，都可以通过变量替换的方式，把曲线降幂到线性问题进行求解

线性回归的模型评价方法

MSE 和 RMSE

MSE 称为均方误差（Mean Square Error），RMSE 称为均方根误差（Root Mean Square Error），二者表达式分别如下：

这里用的是真实值减去预测值的平方和取平均，其中 MSE 因为有平方操作，可能会改变原始数据的量纲，RMSE 则可以保持数据的量纲不变，当数据对量纲敏感的时候，推荐使用 RMSE 方法计算

MAE

MAE（Mean Absolute Error）被称为平均绝对值误差，其表达式如下：

MAE 和 RMSE 相比，对于同一组数据和模型，前者的误差一般会小一点，因为 RMSE 是先对差值取平方和后再开方，相当与放大了最大误差

R^2^

R^2^（coefficient of determination）称为比较判定系数，其表达式如下：

该评价方法的特点是：

  1. 消除了量纲的影响（比值形式会抵消量纲）
  2. 一定程度上能够反映模型对数据的拟合程度

R^2^ 越大，表示模型效果越好，误差越小；R^2^ 越小，表示模型效果越差，误差越大

R^2^ 还有另一种定义，我们给其表达式的分子分母都各自乘上一个 1/n，那么分子变成 MSE 指标，分母变成 y 值真实值的方差，所以 R^2^ 也可以被定义成MSE 和 真实 y 值的方差之比，这个比值本身就体现了归一化的思想（MSE 除以方差，可以消除数据本身大小带来的影响）
