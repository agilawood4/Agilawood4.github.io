控件

在 PyQt 中，控件（Widget）是构建用户界面的基本元素，可以理解为可视化程序中的各种”零件”，先前介绍的button就属于按钮控件，接下来我们考虑引入更多控件来丰富我们的布局，同时为空间触发功能，来实现和用户的交互

QPushButton 控件

button控件就是常见的按钮控件，用户可以通过点击按钮实现某些功能，该控件可以通过QPushButton()方法来实现

这里需要注意，button1 有 self 前缀，而 button2 没有，表示两者作用域不同，button1 在整个类中都可用，而 button2 则仅在当前的方法中（__init__初始化）使用，如果我们需要再类的其他方法中访问该按钮，如绑定事件，修改文本等，则需要用 self.button1 的方式实现

信号与槽

如果我们想为按钮添加和用户交互的功能，比如用户点击按钮时，用户按下按钮的阶段，用户松开按钮时，我们可以通过信号和槽实现相关的功能

按钮可选中 / 可切换

注意在 PyQt 中，我们的按钮被按下后有两种情况，一种是按下后松开鼠标，按钮扔保持被按下的状态，一种是按下后松开鼠标，按钮立刻回弹（默认情况），前者每次点击按钮都会发射toggled信号，后者每次点击按钮只会发射clickd信号，可以通过设置setCheckable()函数控制两种状态切换

只有setCheckable()选项设置为 True 的按钮才可以添加信号和槽，信号和槽可以用来捕获按钮状态的变化，比如用户按下按钮的瞬间，我们可以捕获这个操作，并且通过这个操作关联某个函数，对用户的操作做出反应

信号和槽实现

每个按钮都有以下几个状态，可以通过connect()方法把它们与特定的函数联系起来，当触发某个状态时，就会执行相应的函数：

  1. clicked：按钮经历完整点击动作（用户按下按钮并释放后触发）
  2. pressed：按钮被按下（用户按下按钮时触发）
  3. released：按钮被松开（用户松开按钮时触发）
  4. toggled：按钮状态发生切换（比如从松开到按下，从按下到松开时都会触发该信号）

QLineEdit 控件

该控件被称为文本编辑控件，用来实现页面中的输入框（比如用户可以通过该控件输入账号和密码），该控件通过QLineEdit()方法进行初始化

常用效果

文本编辑控件常用的效果设置有以下三个，通过调用setEchoMode()方法可以实现：

  1. setEchoMode(QLineEdit.EchoMode.Password)：密码遮挡（一般会显示为小圆点或星号）
  2. setEchoMode(QLineEdit.EchoMode.NoEcho)：取消回显
  3. setEchoMode(QLineEdit.EchoMode.PasswordEchoOnEdit)：只在输入时显示，输入完成移走光标以后就隐藏

  2. 检测输出完成，通过回车触发：editingFinished
  3. 检测光标选区变化：selectionChanged
  4. 检测文字编辑变化：textChanged
