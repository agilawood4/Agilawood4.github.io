python编程基础

通过阅读本文，您可以大致了解 python 编程中的常用知识和技巧

全局变量

全局变量可以在函数内部直接被访问，但是不能直接被修改，如果想在函数内部修改全局变量的值，需要加global声明

列表排序

如果想对列表排序，首先需要保证列表中的元素类型相同

如果列表中的元素是常用的数据类型，比如整型，浮点型等，可以直接调用 sort 方法对列表进行原地排序，而如果列表中的元素是字典等复杂类型，就需要我们自定义排序规则，否则排序可能会报错

利用 key 排序

sort 方法中支持传入 key 参数，可以通过一个 lamda 表达式自定义排序规则，其中 lambda 匿名函数的形参是列表中的一个元素，对于下面的 my_list 而言，其中的一个元素就是一个字典，因此 lambda 函数的形参就是一个字典类型

sort 方法是默认升序排序的，如果按照指定的某规则降序排序，则可以传入参数reverse = True

多关键字排序

如果列表中的元素也是列表，想根据列表的某一维进行排序，可以通过 x [ i ] 实现，表示按照列表元素 x 的第 i 维进行排序

如果想制定多个排序规则，比如先按照第二维升序，再按照第一维降序等，可以根据排序规则中各个维度的优先级，把各个维度写在括号里，也就是写成元组的形式，其中降序排序的维度前面加负号即可

如果是字典或者其他数据类型，我们也可以把需要排序的各个维度，按照优先级顺序依次写入一个元组中

列表推导式

列表推导式可以用来快速生成列表，可以对标 C++ 中对 vector 类数组初始化的过程，只不过在 python 中的实现会更加灵活

规则一

变量 = [ 数据生成规则 for 临时变量 in 范围 / 容器 ]，依据此规则，每循环一次就会创建一个列表元素

这里的数据生成规则可以是满足语法规定的任意规则

规则二

变量 = [ 数据生成规则 for 临时变量 in 范围 / 容器 if 判断条件 ]，依据此规则，每循环一次且满足 if 条件时，就会创建一个列表元素

列表去重

借助集合 set 的去重特性，先用 set() 方法把列表转化成集合，再用 list() 方法把集合转化回列表即可

print 换行

python 中的 print 函数在输出时会自带一个换行，我们如果想取消掉这个换行，让其在同一行打印，可以通过设置 print 的 end 参数来控制，默认条件下end = '\n'，我们将其设为空即可

高阶函数

把函数当做参数传递，这个函数就称为高阶函数，python 中有很多已经写好的高阶函数

map 函数

通过调用 map() 函数可以遍历传入的可迭代集合中的每一个元素，并对其进行指定的操作，并且返回一个迭代器，使得调用者可以通过该迭代器逐个获取操作后的每一个元素，原集合不会被改变

语法：iterator = map(function, list)，其中 iterator 是返回的迭代器，function 是指定的操作规则，list 是传入的可迭代集合，一般是列表

这里需要注意的是，我们直接调用 map 返回的是一个迭代器类型，如果我们想得到列表，可以使用 list() 进行强转，该函数支持把迭代器元素逐个写入列表

filter 函数

通过调用 filter() 函数可以遍历传入的可迭代集合中的每一个元素，根据筛选规则，过滤掉不符合要求的元素，保留符合要求的元素，并且返回一个迭代器，使得调用者可以通过该迭代器逐个获取筛选后的每一个元素，原集合不会被改变

语法：iterator = filter(function, list)，其中 iterator 是返回的迭代器，function 是指定的筛选规则，list 是传入的可迭代集合，一般是列表

具体调用规则同 map() 函数

reduce 函数

通过调用 reduce() 函数，对给定的可迭代集合进行求解，遍历其中的每一个元素，根据给定的规则，返回一个唯一的结果

语法：result = reduce(function, list)，其中 result 是返回的结果，function 是指定的规则，list 是传入的可迭代集合，一般是列表，值得注意的是，这里传入的 function 规则一般是有两个参数的一个函数，因为 reduce() 函数会根据给定的 function 规则，依次从传入的 list 中取两个元素进行运算（先运算前 2 个，得到的结果再和第 3 个运算，得到的结果再和第 4 个运算……）

注意：使用 reduce() 函数时需要导入from functools import reduce，因为 reduce() 并不是 python 中最基础的内置函数

装饰器

设计装饰器的目的：假设我们有若干个函数，这些函数在执行过程中，都会在最开始向用户端打印一个 “hello”，那么我们可以把这个共同的逻辑提取出来，写在一个公共部分，而不是在每个函数体中都写一个 print(“hello”)，这段公共部分的代码就称为装饰器

我们在设计函数的时候，一般会先设计其个性的功能，最后再考虑其共性的功能（比如日志，计时，权限检查等），那么从这个角度来讲，装饰器的功能可以看成是：在不修改原函数代码的情况下，为函数添加新的功能

基本实现

装饰器的本质是一个高阶函数：它会接收一个函数 A 作为参数，这个函数是执行具体个性功能的函数，同时装饰器会在内部再定义一个函数 B，这个函数 B 会先执行装饰器内定义的共性功能，接着再调用函数 A，最后装饰器会返回其内部定义好的函数 B

现在我们有 my_func1 和 my_func2 这两个函数，我们希望他们在执行前都输出一下 “welcome”，因此我们可以写一个装饰器，这个装饰器接收 my_func1 和 my_func2 作为参数，在其内部封装好带输出 “welcome” 的wrapper 函数后，把封装好的 wrapper 函数返回给 f1 和 f2

那么此时 f1 和 f2 就等价于有着输出 “welcome” 功能的 my_func1 和 my_func2 函数，其形参列表和 my_func1 与 my_func2 完全相同

而装饰器内部定义的 wrapper 函数使用可变参数列表，表示 wrapper 的参数实际上是由 my_func1 和 my_func2 所决定的，它自己只提供参数中转传递的功能

但是这么写还是有点麻烦，我们每次都要把真正执行功能的函数先当做参数传入到装饰器中，返回一个封装好的新函数以后，再执行这个新的函数，为此我们可以在所有需要装饰器优化的函数前，加上@+装饰器名称的关键字优化，这样我们直接调用原函数本身，就可以自动实现装饰器的功能

wraps 装饰器

现在我们尝试打印 my_func1 和 my_func2，会发现他们已经不再是原本的函数声明，而是变成了 wrapper，如果我们还想保留函数原本的类型，可以在 welcome 中加一行@wraps(fn)，这是 python 自带的装饰器，功能就是保留函数原本的类型

装饰器传参

如果我们想对装饰器函数也传参，也就是给我们的 welcome 函数中也传入参数，我们可以在装饰器的外层再套上一层普通函数，在最外层的函数中传入我们想给装饰器的参数

此时我们的 welcome 就是一个普通的函数，真正的装饰器实际上是 decorate 函数，我们要传给装饰器的参数都写入 welcome 的形参列表即可，此时我们写@welcome()关键字的时候，就可以直接传入参数了

生成器

生成器是 python 中用于生成某些数据的工具，它的本质可以看成一个迭代器，使用惰性计算原则，只在被调用的时候按需生成值，而不会一次性存储所有的值

yield 语句

以下所实现的就是一个生成器的实例，生成器本质上可以看成是一个迭代器，可以调用它的 next 方法，每调用一次 next 方法，它都会执行到下一条 yield 语句处停止，而不会一次性把整段代码都执行完

三次打印的结果分别是 1 2 3，其中 yield 语句和 return 语句的功能类似，会分别把 1 2 3 返回给 next 调用，但是 yield 执行以后不会退出函数，而是会把迭代器指针停留在当前执行的 yield 语句处，当下次再调用 next 语句时，会从上次执行完的位置开始继续向下执行，直到遇到下一个 yield 语句

正则表达式

正则表达式提供格式和符号来定义字符串的模式，通过模式可以对字符串的内容进行：

  1. 匹配验证
  2. 搜索

在 python 中，主要通过re模块实现正则表达式的功能

re 模块

match 方法

通过调用 match 方法，检测目标字符串是否以某个字符串作为开头，语法：re.match(匹配规则, 被匹配的字符串)

注意 match 方法只匹配开头位置，如果目标字符串没有出现在开头但是出现在了原串的内部，此时也会匹配失败

对于匹配结果，可以通过span()方法显示匹配成功的下标范围（左闭右开），通过group()方法显示匹配成功的内容

需要注意的是，match 方法在匹配的时候不会限制匹配完整单词，比如我们开头的单词如果是 python666，匹配目标是 python，该方法会去匹配 python666 的子串，也会显示匹配成功

如果我们想匹配完整的单词，即在 python666 中匹配 python 时会显示失败，此时我们可以使用 fullmatch 方法，该方法可以实现完全匹配，用法和 match 完全相同

match 方法在匹配失败时会返回 None，匹配成功时会返回匹配位置的下标范围和被匹配的内容

search 方法

通过调用 search 方法，可以在整个字符串中匹配目标子串，如果找到就返回第一个，并且不再继续向后查找，其余用法和 match 方法几乎相同，语法：re.search(屁匹配规则, 被匹配的字符串)

search 方法在匹配失败时会返回 None，匹配成功时会返回匹配位置的下标范围和被匹配的内容

findall 方法

通过调用 findall 方法，可以匹配原字符串中所有的目标子串，在整个字符串中匹配，获得全部匹配项，语法：re.findall(匹配规则, 被匹配的字符串)，其返回值是一个列表，即便没有找到匹配项，也会返回一个空列表
