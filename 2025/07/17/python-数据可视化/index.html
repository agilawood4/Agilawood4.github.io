json 数据格式

json 简介

json 是一种轻量级的数据交互格式，可以按照 json 指定的格式去组织和封装数据，而 json 本质上是一个带有特定格式的字符串

json 可以在不同编程语言之间流通，负责不同编程语言中的数据传递和交互

对于 python 而言，json 一般有两种格式，都可以和 python 已有的容器进行无缝切换：

  1. 列表格式的 json，其中每一个元素是字典
  2. 字典格式的 json

python 和 json 数据的转换

我们通过引入 python 中的 json 模块即可实现二者的相互转换：

  1. 通过调用 dumps 方法，可以把 python 数据转化为 json 数据
  2. 通过调用 loads 方法，可以把 json 数据转化为 python 数据

其中 json 格式的数据在 python 中直接以字符串形式存储即可，字符串的内容要符合 json 本身的格式

其中 ensure_ascii = False 的作用是使得中文可以被正确显示

基础折线图绘制

可以通过 pyecharts.charts 包中的 Line 功能进行折线图的绘制

配置选项

pyecharts 中有两个常见的配置选项：全局配置选项和系列配置选项

全局配置选项可以通过set_global_opts方法进行配置，可以用来配置图表的标题、图例、工具箱配置项等

全局配置项的设置：

全局配置项的设置需要导入若干包：

  1. 配置标题：TitleOpts，可以配置标题内容，左右相对位置，上下相对位置
  2. 配置图例：LegendOpts，可以配置图例是否显示
  3. 配置工具箱：ToolbxOpts，可以配置工具箱是否显示
  4. 配置视觉映射：VisualMapOpts，可以配置视觉映射是否显示

其他全局配置项的设置可以通过查阅官方文档获悉：

https://pyecharts.org
