函数

函数多返回值

如果一个函数想要实现多个返回值，只需要按照返回值的顺序，分别用多个变量进行接收即可，不同的返回值和用来接收返回值的变量之间都用逗号进行分隔

python 支持不同类型数据的 return

函数的传参方式

位置参数

位置参数就是我们最常见的函数传参方式，在调用函数时，根据函数定义的形参列表，在正确的位置上对应传入相应的实参即可

位置传参需要保证，对应位置上的形参和实参必须一一对应

关键字参数

函数调用的时候，可以通过形参 = 实参的形式进行传参，如果我们指明给函数的哪个形参赋值，在传参的时候就没有顺序和位置的限制了

未知参数和关键字参数可以混用，但必须保证位置参数在前，关键字参数在后

缺省参数

缺省参数也称为默认参数，在函数定义的时候，可以给一些参数提供默认值，调用函数的时候如果不进行传参，那么该参数就会采用默认值

注意：缺省参数必须定义在参数列表的最后，可以有多个缺省参数

不定长参数

不定长参数也称为可变参数，如果我们实现函数的时候，不确定调用者会传入多少参数（也可能不传参），就可以使用不定长参数，具体又分为位置传递和关键字传递

位置传递

函数形参使用*args来实现，调用者传入函数的所有参数都会被 args 变量收集，它会根据所传入参数的位置合并成为一个元组（tunple），args 就是元组类型

关键字传递

函数形参使用**kwargs来实现，调用者通过键 = 值的形式向 kwargs传入参数，其中 kwargs 是一个字典

匿名函数

函数作为参数传递

python 支持把一个函数作为另一个函数的参数，其实对标的就是 C 语言中的回调函数，此时我们传参传入的是计算逻辑，而不是数据

具体应用：

如果我们的 test_func 是一个排序函数（比如 bubble_sort），那么 compute 就可以设计成比较规则（升序、降序、字典序排列……），通过给排序主函数传入不同的比较规则，我们就可以实现基于不同规则的排序

lambda 匿名函数

在 python 中我们支持两种函数定义方式：

  1. def 关键字：定义有名称的函数，可以基于名称重复使用
  2. lambda 关键字：定义匿名函数，因为没有名称，只能临时使用一次

匿名函数的语法：lambda 形参列表 : 一行代码作为函数体，注意函数体只能是一行代码，不可以多行
