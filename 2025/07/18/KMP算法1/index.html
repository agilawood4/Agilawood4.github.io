KMP 算法（下）

本文用来补充上一篇文章中最后两个 DP 问题用矩阵快速幂优化的正解

P3193

思路

在之前的题解中，我们已经找到了 DP 的转移方程，假设我们当前文本串匹配的长度为 i，模式串匹配的长度为 j，第 i 个位置希望填写进文本串的字符为 ch，该状态所能贡献的状态由 j 和 ch 两个参数唯一确定，和 i 无关，换言之，也就是说只要我们在模式串中遍历到 j 长度，且打算在文本串中填写一个 ch 字符，不管这个字符填写在文本串的什么位置，其转移到的模式串的新位置 p 是唯一确定的，因此我们可以预处理出来所有的转移关系，也就是模式串的 j 位置转移到 p 位置的所有方案数，这个转移矩阵就是我们用到的基础系数矩阵

首先我们需要预处理出来 nxt 数组，因为从 j 位置向 p 位置的跳转，本质上是基于 nxt 数组进行转移的，对于我们当前遍历到模式串的 j 位置，它可能对哪些位置产生贡献，取决于当前我们文本串填写的字符而不是文本串填写的位置，因此我们根据当前 j 位置和填写的 ch 字符，进行 nxt 跳转，跳转的终点是 p 位置，也就是说 j 长度最后会对 p 位置有一次贡献

但是我们的系数矩阵维护的是 j 长度对 p 长度模式串的贡献，因此我们还要把 p 下标转长度，也就是 p++，那么系数矩阵的 j 行 p 列会因此获得 +1 的贡献

有了转移矩阵以后，我们还要考虑初始状态，也就是文本串长度为 0，dp [ 0 ] [ x ] 的值，只有 dp [ 0 ] [ 0 ] 的值为 1，其余状态都是非法状态（0 长度文本串无法匹配任何模式串），因此初始的答案矩阵也只有 0 行 0 列位置为1
