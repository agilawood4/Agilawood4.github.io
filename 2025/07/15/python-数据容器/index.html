数据容器

python 中的数据容器：一种可以容纳多份数据的数据类型，容纳的每一份数据称为一个元素，可以是任意类型的数据，如字符串，数字，布尔等

常见的数据容器种类：

  1. 列表（list）
  2. 元组（tuple）
  3. 字符串（str）
  4. 集合（set）
  5. 字典（dict）

列表

列表是 python 中最基本的容器，可以认为，列表模拟的是 C++ 中的数组

列表的基本定义

列表使用中括号定义

其中 list4 的初始化方式可以用来模拟 C++ 中的下标数组，列表中的元素可以是不同的数据类型，同时也支持嵌套

列表的下标索引

列表同时支持正向和反向的下标索引，正向索引从 0 开始，表示第一个元素；反向索引从 -1 开始，表示最后一个元素

嵌套列表的索引方式和 C++ 中二维数组的索引方式相同

列表的常用操作（方法）

查询元素下标

通过调用 index 方法可以查询指定元素在列表中的下标，如果不存在元素，会报错 ValueError

修改下标对应元素的值

和 C++ 中的数组完全相同

插入元素

通过调用 insert 方法，在 list 指定的下标位置插入指定的元素（使用 insert 方法可以有效避免越界，比如 list 中现在只有 3 个元素，但是我们在第 5 个位置 insert，实际上会插入在第 4 个位置）

也就是说，list 执行 insert 操作的时候不会留空

追加元素

通过调用 append 方法，在 list 尾部追加一个元素（传入单个元素）；通过调用 extend 方法，在 list 尾部追加批量元素（传入容器）

删除元素

通过调用 del 方法，在 list 指定下标位置删除元素；通过调用 pop 方法，在 list 指定下标位置删除元素，需要注意的是 del 的用法和其他方法不一样，是以 del list [ 指定下标 ] 的方式实现的

此外，通过 pop 方法删掉的元素，可以作为返回值被直接接收获得

还可以通过调用 remove 方法，传入想删除的元素值，此时 list 会从前向后搜索，删除第一个匹配到的目标元素

如果想删除所有匹配的元素，可以把 remove 的 () 换成 {}，此时会删除匹配到的所有目标元素

清空列表

通过调用 clear 方法，清空 list 中的所有元素

统计元素数量

通过调用 count 方法，统计 list 中指定元素的数量

统计列表中元素个数

直接调用 len 函数即可（len 不是 list 的方法，对于所有容器都可以使用 len 求容器长度）

元组

列表是可以被修改的，如果我们希望存储的信息不能被修改，就可以使用元组

元组的功能和列表一样，可以封装多个不同类型的元素在内，但是元组一但定义完成，就不可修改，可以认为元组是一种只读的列表

元组的定义

元组通过小括号定义

需要注意的是，即便元组中只有一个元素，也要在这个元素后边加上逗号，否则 python 可能不会将其解释成元组

元组也是支持任意元素类型和嵌套的，这点和列表相同

元组的下标索引

和列表完全相同

元组的常用操作（方法）

查找指定元素的索引

通过调用 index 方法获得指定元素在元组中的下标索引

获取指定元素的数量

通过调用 count 方法获得元组中指定目标元素的数量

获取元组中元素总数

通过 len 函数，获取元组中元素的总数

元组的遍历

for 循环遍历

注意

元组本身的内容是不支持修改的，但是如果元组中嵌套了一个列表，列表内部的内容是可以修改的

字符串

一般来讲，字符串既可以作为数据类型，也可以作为数据容器出现，字符串通过 “” 来定义

字符串的常用操作（方法）

字符串的替换

通过调用 replace 方法，把原串中全部的子串1 都替换成子串2 的内容，需要注意的是，replace 方法并不会修改原字符串，而是会返回一个修改后的新的字符串

字符串的分隔

通过调用 split 方法，把原字符串按照指定的分隔字符串，划分成多个子串，并且存入一个列表中，该方法同样不会修改原串，而是返回一个列表容器

需要注意的是，字符串和元组一样是不支持修改的，因此我们如果想对字符串进行修改，一般都需要先转化成列表，再对列表进行修改操作

字符串的规整操作

通过调用 strip 方法，对原字符串进行规整操作（默认是去掉前后空格，也可以传入一个 str，去掉前后指定的字符串 str）

注意在下面这个例子中，我们传入的 str 是 “12”，会按照单个字符的形式，把前导和后缀中的 1 和 2 这两个字符都去掉

统计子字符串的出现次数

通过调用 count 方法，统计原字符串中某个子字符串的出现次数

统计字符串长度

通过使用 len 函数，即可获得字符串的长度（可以对标 C++ 中的 strlen 函数）

总结

数据容器的切片

序列

序列指的是：内容连续，有序，可以使用下标索引的一类数据容器，列表、元组、字符串都可以看成是序列

序列的切片

所有的序列结构都支持切片操作，也就是从原序列中，取出一个子序列

表示从序列的起始下标位置开始，依次取出元素，到结束下标位置，得到一个新的序列

  1. 起始位置：可以留空，留空默认从 0 位置开始
  2. 结束位置：可以留空，留空默认直接取到序列结尾，如果填写一个具体的结束位置，则这个位置不会被取到，只会取到它的前一个位置
  3. 步长：表示取元素的间隔，步长写成 N 表示每次间隔 N-1 个元素进行选取，如果步长为负数则表示反向选取，注意此时起始位置和结束位置也要反过来，可以留空，留空默认表示步长为 1

对序列的操作都不会改变序列本身，而是会返回一个新的序列

集合

已知列表，元组等都支持重复元素，如果想对元素去重，则可以使用集合

集合的定义

集合通过 {} 进行定义

集合的特点：去重（即便初始化的时候有重复元素，也会自动去重），无序，因此集合是不支持下标索引访问的

集合的常用操作（方法）

向集合中添加元素

可以通过调用 add 方法向集合中添加元素，添加重复元素不报错，但会自动去重

从集合中移除元素

可以通过调用 remove 方法从集合中移除指定的元素

从集合中随机取出元素

可以通过调用 pop 方法，从集合中随机取出某个元素，并且取出的元素会直接返回，可以被接收，同时被取出的元素会从集合中删除

清空集合

可以通过调用 clear 方法，清空集合

取两个集合的差集

可以通过调用 difference 方法，取两个集合的差集，语法为 set1.difference(set2)，表示 set1 - set2，返回一个新集合，其内容为在 set1 中但是不在 set2 中的元素

消除差集操作

可以通过调用 difference_update 方法，去掉一个集合中，在另一个集合内页存在的元素，语法为 set1.difference_update(set2)，表示在 set1 中去掉其在 set2 中也存在的元素，set1 会被更新，set2 不变

集合合并操作

可以通过调用 union 方法，合并两个集合，语法为 set1.union(set2)，其中 set1 和 set2 都不变，返回一个新集合，新集合元素的内容是两个集合元素的并集（会自动去重）

集合的遍历

因为集合不支持下标索引，所以不能用 while 循环遍历，但可以用 for 循环遍历

for 循环遍历

字典

python 中的字典可以对标 C++ 中的 map，用来维护键值对的映射关系

字典的定义

字典和集合一样，同样使用 {} 定义

字典不支持下标索引，只能通过 key 值索引到对应的 value 值，索引方式为my_dict[key]

字典也是支持嵌套的，其中 key 不能是除元组外的其他容器，可以为一般数据类型，value 没有任何限制

字典的常用操作

新增元素 & 更新已有元素

如果给出的 key 不存在，就相当于是新添加了一个键值对，否则就是修改已有的键值对

删除元素

通过调用 pop 方法，给出待删除键值对的 key，直接返回这个 key 对应的 value，对应键值对在字典中会被删除

清空字典

通过调用 clear 方法，可以清空整个字典

获取字典所有的键

可以通过调用 keys 方法，获得字典中所有的键，并且由此遍历整个字典

总结

通用的容器操作

最大 & 最小元素

通过调用 max 和 min 方法，可以返回容器中的最大 / 最小元素（此时容器中的元素类型最好相同），字典会返回最大 / 最小的键，和值无关；字符串中会返回最大 / 最小的字符

容器的通用转换

通过调用 list()，set()，str()，tunple() 函数可以实现把五大类容器分别转换成列表，集合，字符串，元组

list()

其中元组、集合转列表很平凡，字符串转列表会把字符串中的每一个字符作为列表中的元素，字典转列表会把字典中的每一个键作为列表中的元素

str()

字典转化成元组、列表、集合的时候，其对应的 value 值都会丢失，但是转化成字符串的时候，value 会保留

排序函数

通过调用 sorted 方法，可以对容器进行排序，该方法设有两个参数，分别是待排序的容器，以及默认的从小到大排序，如果是从小到大排序是不需要写第二个参数的，从大到小排序则需要加 reverse = True 参数

sorted 方法会返回一个有序的列表，不会修改原容器，如果是对字典排序，则会丢弃值，只保留键，返回键排序的结果

其实 python 中为我们提供了两个排序函数，sorted 和 sort 函数

  1. sorted()：内置函数，可以对任何能够迭代的对象进行排序
  2. sort()：只能对列表排序，且为原地排序，会修改列表本身，返回值为 None，也可以通过设置 reverse = True 实现降序，默认升序排序
