神经网络基础

通过阅读本章内容，您可以大致了解神经网络的基础概念和简单的代码实现

神经网络原理

单个神经元

神经网络由神经元组成，从单个神经元来看，其就是一个简单的线性模型：y = f(∑wx + b)，这里求和只对 wx 求和，并不包含 b

在本例中，输入 x 是 n 维的（有 n 个特征），因此对应就有 n 个权重系数 w 和 1 个截距 b，其中 f 是激活函数，用于对线性求和进行非线性变换但从其中一个维度的 wx+b 来看，神经元本质就是线性模型

该表达式也可以写成矩阵的形式：y = f(WX + b)

其中神经网络又称为多层感知机模型（MLP - Multi-Layer Perceptron），而单个神经元也被称为感知器（Perceptron），二者对应的是同一个概念

神经网络的结构

许多神经元前后相连所组成的网络结构，就称为神经网络，一般而言神经网络至少要有三层组成：输入层 X（样本本身，样本有多少个维度，输入就需要几个维度），隐藏层 H（神经网络的核心），输出层 Y（对样本的分类标签，有几种分类，输出就有几个维度）

其中隐藏层可以由很多层神经元组成，随着隐藏层深度的增加，深度学习也由此得名

其中连接输入层和隐藏层的是一个 2×50 维度的矩阵 W1，连接隐藏层和输出层的是一个 50×4 维度的矩阵 W2，他们各自对应一个截距常数 b1 和 b2

激活函数层

如果我们在神经网络的各个层之间都只使用线性变化，形如 y1 = w1x + b1，y2 = w2y1 + b2，…，我们知道，多个线性方程可以进行合并，合并的结果整体上还是线性变换，因此如果我们各个层之间使用的全都是线性变化，本质上和进行一次线性变化没有区别

因此要想真正展示神经网络的性能，我们需要在每一个隐藏层之后增加能够提供非线性变化的激活函数层，把线性变为非线性

常见的激活函数如下：

阶跃函数：输出有二值，会发生跳变

Sigmoid 函数：输入的绝对值较大时，会造成梯度消失

Relu 函数：目前神经网络中最常用的激活函数

每一层隐藏层的计算之后，都需要加一层激活层，否则该层计算会失去对应的意义

输出层

再加上激活函数以后，我们的神经网络就如下图所示，现在我们希望输出的 y 值能够变成概率的形式，也就是 [0, 1] 范围内的数值，通过比较各个类别的概率大小判断分类标准

我们只需要引入 yi = e^i^ / ∑ j e ^j^ 即可，也就是我们先把每个输出以 e 为底求指数，对于第 i 个输出，我们用它自己的指数，除以所有的指数之和，即可得到每个输出对应的概率

除了交叉熵损失之外，我们还有很多其他的损失函数，常见的如对数损失，均方差损失（E = ∑1/2(y预测 - y实际)^2^ 该损失在理论课中多有涉及）

我们模型训练的目标就是使得损失函数能够得以最小化

反向传播

先前我们所提到的数据流，都是由输入 X 正向得到输出 Y，接着对 Y 进行概率化和损失计算的过程，而得到损失以后，我们要根据损失值继续调整我们模型的参数，即修改优化每一层的 w 和 b 值，这一过程就称为反向传播

神经网络的神奇之处就在于，其可以自动进行 w 和 b 值的优化，在这一过程中，我们一般采用梯度下降法，找到使得损失函数下降最快的那一组 w 和 b，进行模型参数调整

多层感知机

引入非线性变换的必要性

核心问题：线性模型的线性组合仍然是线性模型，而线性组合无法表示复杂的非线性数据

我们把 W1W2 看成一个新的系数矩阵，b2+b1W2 看成一个新的偏置常数，那么该线性模型的组合仍然是线性模型，而现实中的分类问题的边界往往都无法用单一的线性模型较好的拟合，所以需要引入非线性的激活函数

加入非线性变换 f 以后：h = f(Whhpre + bh)，激活函数的本质是对线性空间进行变形扭曲，使得原本线性不可分的元素在新空间下线性可分（可以参考 SVM 的核函数）

其中每一层隐藏层会在原本的基础上加上激活函数 f，一般各个层之间的激活函数都是相同的，即 f1 f2 f3 f4 都是同一个激活函数，在分类网络中，输出层的 fy 一般选用 Softmax 作为激活函数

激活函数

激活函数（Activation Function）可以对线性空间进行扭曲翻转

常用的激活函数如下所示：

Sigmoid函数

本质上就是高中生物里讲的 S 形曲线，能够把输出映射到 [0, 1] 范围内的空间，-inf 接近 0，inf 接近 1，在二分类问题中的使用效果较好，常用于逻辑回归中

缺点：输出接近 0 或者 1 位置时，容易造成梯度消失现象，同时模型以 0.5 为对称中心，而不是以 0 为对称中心

Tanh

该函数是 Sigmoid 函数的改进版，在循环神经网络、长短时神经网络中仍有较广泛的应用

对应的映射空间变为 [-1, 1]，同时以 0 作为中心，输入为 0 的时候输出也为 0，相比于 Sigmoid 函数有了更大的映射空间，函数更加陡峭，对应更快的收敛速度，在梯度下降中有重要作用

缺点：输入绝对值较大时仍然可能造成梯度消失

ReLu函数

ReLu 函数是目前神经网络中使用最广泛的激活函数，相比于 Sigmoid 类函数，它的导数只有 0 和 1 两个值，不会造成梯度消失现象，同时不涉及指数运算，计算速度较快，同时计算也较稳定

缺点：输入为负数的时候输出恒为 0，导致网络某些参数难以训练

Softmax函数

该函数可以把一组输入值映射为一组概率值进行输出，各个概率的累加和为 1，该函数在多分类问题中（神经网络的最后一层输出层）进行使用，使得输出变为概率值，具有更好的可解释性

前向传播和反向传播

前向传播

神经网络模型中最常见的就是深度前馈网络（Deep Feedforward Network）与前馈神经网络（Feedforward Neural Network），上文中我们提到的感知机模型就是前馈神经网络中的一种

该网络由输入层，输出层与中间的若干个隐藏层所组成，前馈的含义是数据只能前向传播（可以跨越某些隐藏层）而不能反向传播或再同一层的神经元之间传播，具体的表达式为：

  • 隐藏层之间：h = f(Whhpre + bh)
  • 隐藏层到输出层：y - fy(Wyhm + by)

损失函数

我们用损失函数来评估模型预测结果和真实结果之间的差距，用来反映神经网络性能，我们希望通过不断的训练使得该误差不断变小

常用的损失函数为均方误差（Mean Squared Error），其表达式为：MSE = 1/n ∑ i (y预测i - y真实i)^2^
