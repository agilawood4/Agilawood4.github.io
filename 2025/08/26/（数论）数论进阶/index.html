原理解析

首先是在线性筛模版中，我们可以保证每个合数 m，都是被他的最小质因子给筛掉的，也就是 i × p [ j ] 得到的这个合数被标记 vis 为 1 的时候，我们能保证 p [ j ] 就是这个合数的最小质因子，这是线性筛的基本原理，在线性筛的基础上，我们维护以下两个数组：

  1. a [ i ] : 数字 i 最小的质因子出现的次数
  2. d [ i ] : 数字 i 的约数个数

对于每一个质数 i，很显然，d [ i ] = 2，a [ i ] = i

对于每一个合数 m = i × p [ j ]，我们根据 p [ j ] 是否为 i 的约数进行分类讨论

  1. i % p [ j ] == 0 时：说明 p [ j ] 是 i 的质因子，且一定是 i 的最小质因子，因为如果不是的话，那么 i 一定还有一个更小的质因子，那么这个质因子也一定是 m 的最小质因子，那么 p [ j ] 就不是筛掉 m 的最小的那个质因子了，这和线性筛的规则相矛盾，那么也就是说 p [ j ] 同时是 i 和 m 的最小质因子，也就是说，m 和 i 比起来其他的质因子个数都没有变，就是 p [ j ] 这个因子的个数多了一个，所以 a [ m ] = a [ i ] + 1，d 值用素数唯一分解定理展开可知，只有和 p [ j ] 有关的这一项的次数多了 1 个，原本是 a [ i ] + 1 现在变成 a [ m ] + 1，所以我们乘一项再除一项即可
  2. i % p [ j ] != 0 时：说明 p [ j ] 不是 i 的质因子，但它是筛掉 m 的最小质因子，也就是说 m 和 i 相比，它多了一个质因子 p [ j ]，那么 a [ m ] 目前就是 1，只有 p [ j ] 自己，d [ m ] 相对于 d [ i ] 还是只改变了 p [ j ] 这一项，因为它只有一次，在约数个数的贡献中就是 2，所以乘 2 即可

考点：约数个数 + 前缀和

思路

首先本例有以下结论，把正整数 n 分解成两个整数的乘积（考虑第一个乘数和第二个乘数的顺序），所有分解的结果就是 n 的因子个数，比如 6 的因子有 1 2 3 6，那么它的分解即可表示为 6 = 1 × 6 = 2 × 3 = 3 × 2 = 6 × 1

那么本例我们可以先考虑 x + y == n 的情况，所得到的所有解就是 x 和 y 对应的因子数的乘积，也就是 dx × dy，接下来我们考虑把等号推广到小于等于的情况，这里我们常用的推广技巧就是前缀和优化，举一个很简单的例子，假设我们有 x + y == 5，现在我们给定 x == 2，那么 y == 3 是确定的，而如果变成 x + y <= 5，那么给定 y 的时候，对应的 x 就会变成 1 2 3 这三种情况，也就是从单点 3 值，变成了 1 到 3 的前缀和，因此本例我们也用如上的思路进行考虑：

我们枚举 x + y <= n 中 x 的贡献，也就是从 1 到 L 枚举 x 的值，剩下的 y 的最大范围就是 L - x，也就是我们需要维护好 d 数组从 1 到 L - x 的前缀和，那么对于每一个 x，我们得到的贡献就是 d [ x ] × sum(d[1] + d[2] + …… + d[L-x])

生成函数

生成函数：某个序列 a 的生成函数是一种形式幂级数，其每一项的系数可以提供关于这个序列的信息，根据问题的不同，我们可以构造不同的生成函数

所谓形式幂级数，对于 a~i~x^i^ 而言，我们只关注 x 的系数 ai 和 它的幂次 i 的含义，而并不关心 x 具体的值是多少，这样的幂级数表示就称为形式幂级数

生成函数包括普通生成函数，指数生成函数，狄利克雷生成函数等，这里我们主要介绍的是普通生成函数

普通生成函数

（1）定义

普通生成函数形如 F(x) = ∑ ~n>=0~ a~n~x^n^ 的形式，其中 a~n~ 可以是有穷序列，也可以是无穷序列，例如：

  1. 序列 <1,2,3> 的普通生成函数为：1x + 2x^2^ + 3x^3^
  2. 序列 <1,1,1,…> 的普通生成函数为：1x + 1x^2^ + 1x^3^ + … = ∑ ~n>=0~ x^n^
  3. 序列 <1,2,4,8…> 的普通生成函数为：1x + 2x^2^ + 4x^3^ + 8x^4^ + … = ∑ ~n>=0~ 2^n^x^n^

（2）加减运算

F(x) + G(x) = ∑ a~i~ x^i^ + ∑ b~i~ x^i^ = ∑ ~n>=0~ (a~n~ + b~n~)x^n^，也就是说 F(x)+G(x) 可以看成是序列 <an + bn> 的普通生成函数

（3）乘法运算

生成函数之间的乘法运算又称为卷积运算，可以表示为：

F(x)G(x) = ∑ ~n>=0~ x^n^ ∑ ~n>=0~ a~i~ b~n-i~

例如当 n=3 的时候，F(x)G(x) 表达式中，x^3^ 的系数就是 a0b3 + a1b2 + a2b1 + a3b0，也就是说 F(x) 在和 G(x) 相乘的时候，F(x) 中的每一个系数 ai 都会和 G(x) 中所有的系数 bi 做一次乘法，对应 x 的幂次合并同类项以后，所有的 a~i~b~n-i~ 项会被合并到一起，形成一个全连接的神经网络层

由于该结构非常像卷积神经网络中的卷积层，因此又称之为卷积操作

多重集组合数

普通生成函数可以用来解决多重集组合数问题，所谓多重集组合数，就是有 n 种物品，每种物品有 ai 个，问从中取 m 个物品的组合数？

我们可以用普通生成函数的思想来解决，对于第 i 种物品，最多可以选 ai 个，最少可以选 0 个，则我们可以构造生成函数 (1 + x^1^ + x^2^ +…+ x^ai^)，则我们把 n 个物品所有的生成函数相乘，也就是做卷积运算，最后 x^m^ 对应的系数，就是取 m 个物品的方案数

思路

我们构造每个生成函数为 (x^ai^ + x^ai+1^ + x^ai+2^ + … + x^bi^) 即可，然后令 n 个生成函数相乘，找到对应的 x^m^ 的系数即为最终的购买方案数

思路非常好想，关键是具体代码的实现，普通生成函数求解多重集组合数的问题都可以使用下面的代码模版来实现：

  • 我们逐个处理生成函数，假设对于当前这一轮，我们已经处理好了前 i-1 个生成函数，此时对应的 x^0^ 到 x^m^，他们对应的系数存在 c 数组中，分别是 c~0~ 到 c~m~
  • 此时我们考虑处理第 i 个生成函数，也就是把 ai 到 bi 的幂次对应的系数贡献，和 c~0~ 到 c~m~ 进行叠加，比如 c~j~ 和 a[k] 对应的项叠加，就表示我们已经维护好的 x^j^ 和 x^ak^ 幂次的相乘，对应的 j+ak 幂次的系数加 1，则得到的新的系数我们存入 d 数组中，经过一轮操作以后，d 数组中即存了处理完前 i 个生成函数以后，对应 x^0^ 到 x^m^ 的系数
  • 最后把 d 数组复制回 c 数组，接着处理第 i+1 个生成函数即可

整除分块

整除分块就是我们常说的数论分块，是对 ∑ ~k~ n/k 类式在 O(√n) 复杂度内快速求解的一类算法

算法模型

引理：假设现在我们要求 ∑ ~i~ n/i 的值，i 的范围从 1 到 n，其中 n/i 下取整，如果用暴力枚举复杂度为 O(n)，如果引入整除分块，则复杂度可以优化到 O(√n)

如果我们纯按照暴力来做，则需要依次枚举 1 到 n 的所有 i 值，分别计算每个 n/i 的结果，在枚举过程中我们发现，可能对于很多的 i，n/i 的结果都是相同的，于是我们想到按照 n/i 的取值结果对 i 的枚举过程进行分块，也就是把 n/i 取值相同的那些 i 统一计算（比如有 2 3 4 三个 i 值对应的 n/i 的结果都是 2，我们就把这三个值看成一块，直接用 2 × 3 求解），这就是整除分块的基本思想

整除分块中两个重要的性质：

  • 分块的块数 <= 2√n（该性质用于保证算法复杂度）
  • 数字 i 所在的块的右端点为 n / (n / i)，可以简单证明，我们假设 i 所对应的块的数值为 k = n/i，则有 k <= n/i，则有 n/k >= n/n/i = i，则有 i <= n/k，把 k 代入即为对应的结果（该性质用于快速枚举块的左右端点）

和式变换

所谓和式变换，就是对若干层 ∑ 的式子进行等价变形，从而降低求和复杂度的一种变形手段（比如对 ∑~i~∑~j~[gcd(ij)==k] 的化简），一般让我们通过和式变换化简的求和，都会和最大公约数 gcd 有关

公式符号说明：[] 表示判断是否成立，如果成立则取 1，不成立则取 0，比如 [ d|i ] 则表示 d 是 i 的约数时为 1，d 不是 i 的约数时为 0

和式变换公式

常见的和式变换公式如下（以下都默认 n <= m 成立）：

  • 条件替换式：该变形会增加复杂度，因为我们最内层的和式枚举变成了暴力枚举 1 到 n 的所有 d 值，检验是否满足 d 是 i 和 j 的最大公约数，而 d | gcd(i, j) 就等价于 d | i 且 d | j 成立，但该变形能够使得我们的处理更加方便
  • 指标变量替换式：我们把循环变量变为 ik 和 jk，则 gcd(ik, jk) == k 可以同时约分掉一个 k，变成对 gcd(i, j) == 1 的判断，而 ik 和 jk 的整体范围分别是 1 到 n 和 1 到 m，则对应还原后 i 和 j 此时的范围就是 1 到 n/k 和 1 到 m/k，因此该式子最终化简为 ∑~n/k~∑~m/k~[gcd(i, j) == 1]，可以降低枚举的复杂度
  • 变量分离式：如果 A 只和 i 有关，B 只和 j 有关，我们可以把乘积的求和变成求和的乘积，复杂度直接从 O(mn) 骤降到 O(m+n)，类似于级数中的找规律

思路

本例在上例的基础上，增加了一个 gcd∈prim 的限制，因此需要额外进行一步换元，把对应的质数限制给去掉，具体的推导过程如下：

对于这种带有 gcd 的求和式，我们常用的转换步骤不外乎以下几步：

  • 看到 gcd(i, j) ∈ prim，想到多一层暴力枚举，令 gcd 等于某个具体的 k 值，在让 k 值 ∈ prim，也就是变成 ∑ [gcd(i, j) = k] [k ∈ prim]
  • 看到 gcd(i, j) = k，想到换元消 k，也就是令 i = ik，j = jk，因此 gcd(ik, jk) = k 就可以同时约掉一个 k 值变成 gcd(i, j) = 1，对应前面 i 和 j 在求和中的枚举也从 1 到 n 变成从 1 到 n/k
  • 经过前两步转换，我们就得到了 [gcd(i, j) = 1] 的条件式，这里使用莫比乌斯反演中的黄金代换，即可把其代换为 ∑ ~d|gcd(i,j)~ μ(d)
  • 进一步，我们把这个和式转化为暴力枚举 d 的形式，也就是枚举从 1 到 n/k 范围内的 d，使得 d | gcd(i, j)，又因为 d | gcd(i, j) 这个条件等价于 d|i 且 d|j，因此可以变形为 ∑ [ d|i ] [ d|j ] μ(d)，也就是坐左起第五行的式子，这里我们以退为进，把初始的两重循环升到四重循环
  • 我们调换式子的结构，这里看到 ∑~d~ ∑~i~ [ d|i ] 的结构和 ∑~d~ ∑~j~ [ d|j ] 的结构，我们要立刻想到可以消掉一重和式，内层和式可以等价写成 i_max/d 和 j_max/d（以前者为例，i 从 1 到 i_max 中有多少个 i 是 d 的整数倍，就等价于用 i_max/d 下取整得到的结果），而这里的 i_max = n/k，j_max = m/k，代入则得到右起第一行的式子
  • 如果是上例（没有对 k ∈ prim 的限制），此时已经可以求解，因为 ∑~d~ n/kd m/kd 已经满足整除分块的标准格式，只需要对 μ(x) 求前缀和即可，而本例中因为还有 k ∈ prim 的限制，因此需要继续处理
  • 这里我们主要考虑，能否把 kd 在分母上看成一个变量，对其整体用一次整除分块，也就有换元 T = kd，换元后相应地 d = T/k，代入求和范围中 d 本来的范围，得到 T 应该是 k 到 n，但是 T<k 对应的 μ 值都是 μ(0) = 0，因此不会产生影响，我们直接让 T 从 1 到 n 即可，这样主要是为了书写方便，于是就得到了最后一行的式子
  • 对于最后的式子，我们就可以对 T 使用整除分块，再额外维护一个函数 f(x) = ∑ ~k∈prim~ μ(x/k) 即可，而这个函数的求解也很有讲究，它是按照每个质数的贡献对 x 进行插值求解，而不是以 x 为主体进行逐项求解

思路

首先我们需要知道下面这个结论，把对两个数乘积的约数个数的求解问题，转化为最大公约数的条件数的求和问题

这个结论所表示的含义为，对于整数 i 和 j，他们乘积的约数个数，等于 i 和 j 所有的约数所形成的数对中，那些最大公约数为 1 的数对的个数，利用这个结论，我们即可把原问题转化为两重和式与gcd结合的问题，也就是我们熟悉的形式

下面是我们对本例的常见转化流程，其中前几步引入 gcd，把 gcd 条件式通过黄金代换代称 μ，把 ∑~d~ ∑~x~ [d|x] 代换成 ∑~d~ x_max/d 等都是上例中用过的常见变形：

值得注意的是左起最后一步的变形，这里按理说有 ∑~d~ 的枚举，有 ∑~x~ [d|x] 的枚举，可以直接把对 x 的枚举省掉，用 x_max/d 来代替，但是这里以内 x 在求和式的其他部分出现（也就是 n/x 项中出现），因此我们无法简单的把含有 x 的项直接舍弃，也就不能仿照上文使用这种化简方式

这里我们给出另一种常见思路，用来消除 [d|x] 的条件式，归根结底这个式子约束的就是 d 必须是 x 的因子，我们可以改变对 x 枚举的策略，我们在枚举 x 的时候只枚举 d 的倍数，那么这个条件式一定成立，值为 1 就可以省略，因此我们可以换元 x = dx，y = dy，都按照 d 的倍数进行枚举，对应化简完的和式即为右起第一个式子

此时 dx 和 dy 的范围分别是 1 到 n 和 1 到 m，对应 x 和 y 的范围就是 1 到 n/d 和 1 到 m/d，此时我们即可对得到的结果进行整除分块，这是我们整除分块的另一种常见形式：分块嵌套分块

我们观察化简后得到的形式，其中对 x 和 y 的两个求和形式完全相同，只不过一个上界为 n/d 一个上界为 m/d，我们可以把这部分提取出来当成一个函数，记为 f(x) = ∑ ~i从1到x~ x/i，其中每个 f(x) 本身可以通过整除分块的形式求出，因为 f 此时当 x 确定的时候，本身就满足整除分块的格式

莫比乌斯反演

莫比乌斯函数

莫比乌斯函数一般用 μ(n) 表示，该函数为分段函数:

  • n=1 时，μ(n) = 1（积性函数的性质）;
  • n=p1×p2×…ps 时（p1 到 ps 是互不相同的质数），μ(n) = (-1)^s^，其中 s 是 n 所能拆分成的质数的个数;
  • n 包含相同的质因子时（根据唯一分解定理，n 一定可以由若干质数的乘积组成），μ(n) = 0
