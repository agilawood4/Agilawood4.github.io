基础循环神经网络

在前面的章节中，我们提到，多层感知机处理序列 / 时间相关数据的不足，而循环神经网络（RNN）则专门用于解决这类问题，通过阅读本文，您可以大致了解和循环神经网络有关的基础知识和代码实现

序列建模

常见序列数据

序列数据【时间】：不同时间上收集到的数据，描述现象随时间变化的情况（比如股票价格随时间的变化，天气随日期的变化，社会人口 / 就业量随月份 / 年份的变化）

时间序列分析是指从时间排序的数据点中抽取有价值的总结和统计信息，既包含对过去数据的诊断，也包含对未来信息的预测，自回归模型的应用，标志着时间序列分析作为一个独立的学科进行发展

序列数据【文本】：由一串有序的文本所组成的序列，每一个文本都有一个确定的位置，且后面的文本可能受到前面文本的影响，深度学习网络可以从文本中学习出重要的模式，解决如自然语言理解，文献分类，情感分类，问答系统等问题

  • 分词 / 向量化（tokenization）也称为词元化，由于深度学习模型并不能真正理解文本，所以我们需要现将文本转化为对应的数值，通过数值比较得到规律，这一过程就称为向量化

序列数据【图像】：由一组有序图像组成的序列，每一张图像都对应一个时间戳，后一帧图像可能受到前一帧图像的影响（如运动目标的检测，运动图像的分隔，运动参数的估计）

是否为图像序列主要看的是前后帧位上图像是否有关系，如手写数字的识别，就不是一个图像序列

自回归模型

自回归模型（Autoregressive model）是深度学习中的经典序列模型，对于当前时间 t，该模型假设 t 时刻输出只和前 t-1 个时刻的历史输出线性或非线性相关，用历史状态 x~1~ 到 x~t-1~ 来预测 x~t~ 的值，该墨西哥常用于时间序列预测

给定一个序列 {x}t=1T\{x\}_{t=1}^T{x}t=1T​，自回归模型假设：

xt=f(xt−1,xt−2,…,xt−p)+ϵtx_t = f(x_{t-1},x_{t-2},\dots,x_{t-p})+\epsilon_t xt​=f(xt−1​,xt−2​,…,xt−p​)+ϵt​

其中 p 是回溯的阶数，表示当前 x 和向前 p 个时刻的输出有关，ϵt\epsilon_tϵt​ 为噪声项（通常假设为高斯分布），其中 f(∗)f(*)f(∗) 通常取位线性函数，即：

xt=∑i=1pϕixi−1+ϵix_t = \sum_{i=1}^p\phi_ix_{i-1}+\epsilon_i xt​=i=1∑p​ϕi​xi−1​+ϵi​

在深度学习中的自回归模型，则会把条件概率进行分解：

P(x)=P(x1,x2,…,xT)=∏t=1TP(xt∣x<t)P(x)=P(x_1,x_2,\dots,x_T)=\prod_{t=1}^TP(x_t|x_{<t}) P(x)=P(x1​,x2​,…,xT​)=t=1∏T​P(xt​∣x<t​)

其中 RNN，LSTM，Transformer 解码器都是典型的自回归结构

隐变量自回归模型

隐变量自回归模型（Latent Autoregressive Model）是对自回归模型的改进版本，单纯的自回归模型只能利用过去的观测值，如序列中存在潜在因素（比如主题，语义等隐藏状态），单靠观测值难以建模，隐变量自回归模型在自回归框架中引入隐变量 hth_tht​，帮助模型解释数据背后的潜在结构

隐变量自回归模型同时可以解决自回归模型中 P(x~i~ | x~1~x~2~…x~i-1~) 表达式中 x~1~x~2~…x~i-1~ 的链路太长所造成的依赖的历史状态过多的问题，该模型对历史值进行总结，并进行迭代更新

由于模型中间层输出的 h~t~ 从来没有被观测过，因此被称为隐变量，而该模型也因此得名隐变量自回归模型

我们下面提到的循环神经网络（RNN）本质上就是隐变量自回归模型的一种，该模型是用深度神经网络解决序列建模的核心技术之一

文本数据预处理

中文文本需要用分词算法来完成分词；英文文本则需要拼写检查，词干提取和词形还原

文本处理的步骤

常见的文本数据预处理一般由以下几个步骤组成：

  • 获取原始文本
  • 去除噪声与停用词
  • 词元化（Tokenizer，分词）
  • 标准化
  • 词表
  • 特征提取
  • 建模

去除噪声 / 停用词

去除噪声（Text Clearing）指删除文本中不相关或无用的信息（标点符号，停用词，或者因为数据爬取而带有的 html 标记、广告等无关内容），使得文本中有意义的信息更加突出，提高文本处理的效率

停用词指的是在文本中出现频率过高或者不具有实际意义的词（常见的如：英文介词，代词，连词等，中文助词，量词，叹词等），这些停用词的去除可以通过建立停用词表，进而通过 Python 自带的正则表达式匹配来进行删除

其中停用词的去去除可以在词元化之前，也可以在词元化之后，没有固定的要求，一个朴素的停用词去除算法（基于循环思想）：先建立停用词表，接着遍历输入流中的每一个 token，如果它在停用词表中就删除，反之就保留，最后得到输出流即可

词元化

词元化（Tokenization）把输入的文本流，切分成若干子串，每个子串都具有相对完整的含义，方便模型进行后续学习

标准化

标准化（Normalization）主要是针对英文而言，由两组变形组成：

  • 词干提取（stemming）：抽取词的词干或词根形式（目标是使得单词尽可能变短，不考虑语义）
  • 词形还原（lemmatization）：把任何形式语言词汇还原为一般形式（复数变单数，动词变原型等）

对于某些单词而言，其名词形式最短，而有的单词则动词形式最短，在词干提取时，我们不考虑语义，而是一律用这个词最短的那个形式来代替这个词

词表

词表（Vocabulary）即文本中出现的所有词汇所组成的列表，通过查询词表，我们可以大致了解原文中词汇的组成和分布情况，下面给出一个简单的词表建立算法：

  1. 遍历原文本，统计词频
  2. 过滤高频词（可能是停用词）和低频词（没什么代表性），保留中频词（最能体现文本特征）
  3. 为每个词分配一个编号，以之作为索引建立词表

有时候如果我们加载预训练的词向量模型时，词表已经被建立好，不需要我们再手动建立

特征提取

特征提取（Feature Extraction）是文本预处理中最重要的步骤，其核心目的是把单词进行向量化 / 数字化编码，便于计算机进行处理

词袋模型

词袋模型（Bag of Words）能够根据词表中单词在句子中出现的次数，将其转化为向量，该算法不考虑单词顺序，只考虑其出现次数

对于上面的例子，我们先把两句话中所有出现的单词列成词表（这里暂时不去除停用词），接着我们用和词表长度相同的向量来分别表示这两句话，向量对应位置的值即为词表在该位置的单词在某句话中出现的次数

比如 football 在词表中的索引为 1，这个单词在 s1 中出现 0 词，在 s2 中出现 1 词，因此 s1[1] = 0, s2[1] = 1，以此类推

N-gram模型

N-gram 模型是一种基于概率的判别式语言模型，区别于词袋模型，该模型可以通过 N 值的设置，在不同程度上考虑文本中各个单词的位置关系，其假设第 N 个词的出现只与它前面的 N-1 个词相关，在句子划分时，会把每 N 个词划分为一组

词嵌入模型

在传统的文本处理中（比如 N-gram），每个单词都看成一个独立的词元，比如“猫”、“狗”、“沙发”，模型在处理这些词元时，无法理解它们之间的任何关系，即对于模型来说：“猫” 和 “狗” 的相似度，与 “猫” 和 “沙发” 的相似度没有任何区别

词嵌入模型（Word Embedding） 的核心思想在于：将词汇表中的每个词，映射到一个高维空间（通常是几百维）中的一个实数向量（Vector），这个向量不再是随机的，而是包含了这个词的语义信息

  • 语义相似的词，它们的向量在空间中的距离也更近
  • 词与词之间的关系（如 “国王” - “男人” + “女人” ≈ “女王”）可以通过向量的算术运算来体现

简单来说，Embedding 就是一个 “单词 → 向量” 的查找表（Lookup Table），只不过在映射过程中增加了对单词本身含义和词与词之间关系的考虑，从而让模型能够更地学习和处理

循环神经网络

循环神经网络（Recurrent Neural Network - RNN）即为面向序列数据分析的专用网络模型

基本思想

神经网络本质上可以看成是通过大量数据训练后，可以用于拟合某个函数 y=f(x)y=f(x)y=f(x) 的黑盒子，其内部各个中间层+激活函数叠加的效果，就等价于 fff 的映射

全连接网络结构（Linear）结构上本身没有和顺序相关的处理模块，因为其是一个一个处理输入的，每输入一个 x 得到一个 y，输入之间没有关系，换言之，我们把输入改变顺序后重新给到模型，模型的输出除了顺序对应改变外，内容上本质 不会有太大区别

而对于我们先前提到的文本，时间，视频等数据，顺序对于预测结果的影响其实是很大的，因此我们需要在现有神经网络的隐层（Hidden）中加入和顺序处理有关的网络层级

展开计算图

计算图是形式化一组计算结构的可视化图表，而对于循环神经网络，由于内部涉及到循环结构，为了使得可视化更加直观，我们引入展开计算图，展开（unfolding）计算图导致的深度网络结构中的参数共享

在上图中，右图即为左图展开的结果，对于当前的输入 x，它和 h 上一个时间点的输出共同参与当前 h 中的计算，计算结果又作为下一个时间点 h 的输入，也就是 h~t~ 的输入同时来源于 x~t~ 和 h~t-1~，而它的输出又和 x~t+1~ 共同作为 h~t+1~ 的输入，如此循环往复，形成时间序列

所谓参数共享，指的是每一次 h~t-1~ 和 x~t~ 计算得到 h~t~ 的过程使用的参数都完全相同，同时计算使用的激活函数 fff 也完全相同，可以看成是左图中的结果复制了若干份，把相同的网络结构在不同的时间戳下多次运行的结果

网络结构

所谓循环神经网络，就是在隐藏层中增加了一个循环层，用于把上一个时间戳对应的隐层输出，作为下一个时间戳的隐层输入，和当前时间戳下隐层本身的输入 X 共同作用，得到本次隐层的输出

其中输入层和隐藏层之间的参数矩阵为 U，隐藏层和输出层之间的参数矩阵为 V，隐藏层内部循环使用的参数矩阵为 W

我们把隐藏层间进行链接形成循环神经网络，模型因此就具备了记忆能力，每个时刻的输出不仅和当前输入有关，还受到上一个时刻输出的影响，需要注意的是，循环神经网络并没有增加网络层的结构，只是在计算当前时间戳对应的隐层输出时，加入了对上一个时间戳的考虑

在左图中，三个灰色的水平网络表示的是同一个全连接层对应的不同时刻，其中从粉色的 X 输入到灰色的输出都表示一个完整的全连接层，而橙色的竖直网络则表示隐层之间的联系，t-1 时刻的隐层输出作为 t 时刻的隐层输入，t 时刻的隐层输出作为 t+1 时刻的隐层输入

在右图中给出了循环神经网络的参数设置，对于一般的全连接层网络：

S=f(UX+b)S = f(UX+b) S=f(UX+b)

在加入隐层之间的循环连接以后：

St=f(UX+WSt−1+b)S_t=f(UX+WS_{t-1}+b) St​=f(UX+WSt−1​+b)

其中 StS_tSt​ 是本时刻隐层的输出，St−1S_{t-1}St−1​ 是上一时刻隐层的输出，注意在随时间进行记忆的是网络隐层的输出，而不是最终输出，隐层输出经过参数矩阵 V 变换以后，才能得到真正的网络输出

  • U：输入层到隐藏层输入之间的系数矩阵

α(t)=b+Wh(t−1)+Ux(t)\alpha^{(t)} = b + Wh^{(t-1)}+Ux^{(t)} α(t)=b+Wh(t−1)+Ux(t)

o(t)=c+Vh(t)o^{(t)} = c + Vh^{(t)} o(t)=c+Vh(t)

y^(t)=softmax(o(t))\hat{y}^{(t)} = softmax(o^{(t)}) y^​(t)=softmax(o(t))

Teacher Forcing模型

上面这两种模型都有一个共性的问题，那就是无法并行化训练，当前时刻 t 对应的隐层 h 必须要在上一个时刻的隐层训练完成（得到 h(t−1)h^{(t-1)}h(t−1) 或 o(t−1)o^{(t-1)}o(t−1) 以后）才能进行训练，这可能导致网络训练时间过长，收敛变慢

为此我们提出导师驱动模型（Teacher Forcing），即做出如下的改进：直接用每个时刻对应的数据真实输出 / 标准答案 yyy 作为下一个时刻隐层的输入，也就是用 y(t−1)y^{(t-1)}y(t−1) 来代替 h(t−1)h^{(t-1)}h(t−1) 和 o(t−1)o^{(t-1)}o(t−1)，因为训练集的真实输出是全局可知的，这样每个时刻的隐层的训练都不再需要前面时刻的输出，即都可以独立训练，得到的网络结构如下：

此时假设我们有前后依赖的两个输入 x(1)x^{(1)}x(1) 和 x(2)x^{(2)}x(2)，我们要计算这两轮训练的全局损失 L(y(1),y(2)∣x(1),x(2))L(y^{(1)},y^{(2)}|x^{(1)},x^{(2)})L(y(1),y(2)∣x(1),x(2))，使用对数似然即可进行下面的拆分：

log⁡p(y(1),y(2)∣x(1),x(2))=log⁡p(y(2)∣y(1),x(1),x(2))+log⁡p(y(1)∣x(1),x(2))\log p(y^{(1)},y^{(2)}|x^{(1)},x^{(2)})\\ =\log p(y^{(2)}|y^{(1)},x^{(1)},x^{(2)}) + \log p(y^{(1)}|x^{(1)},x^{(2)}) logp(y(1),y(2)∣x(1),x(2))=logp(y(2)∣y(1),x(1),x(2))+logp(y(1)∣x(1),x(2))

这种训练模式直接把上一个时刻的标准输出作为下一个时刻隐层的输入，虽然极大提高了模型的收敛速度，但由于测试的时候数据的真实输出未知，可能导致模型泛化能力不想，因此需要慎重使用

RNN反向传播

循环神经网络的反向传播称为 BPTT（Backpropagation Through Time）算法，该算法将序列数据的每个时间步看做一层，然后在每个时间步上使用标准 BP 算法来计算梯度

只是值得注意的是，对于 h(t)h^{(t)}h(t) 隐藏层梯度的求解，因为 h(t)h^{(t)}h(t) 层的输入有两个去处，其一是最终作为本层的最终输出 y(t)y^{(t)}y(t)，这一级梯度即为网络层常规输出的梯度求解，其二是作为一下时间戳隐层 h(t+1)h^{(t+1)}h(t+1) 的输入，进而参与下一时间戳网络层最终输出 y(t+1)y^{(t+1)}y(t+1) 的输出

而由此类推，本层 h(t)h^{(t)}h(t) 的输出，由于循环神经网络时间依赖的特性，会向后作用于从 h(t+1)h^{(t+1)}h(t+1) 到最后一级时间戳 h(T)h^{(T)}h(T) 输出的全体，也就是说在求解梯度时，f(t+1)f^{(t+1)}f(t+1) 到 f(T)f^{(T)}f(T) 的输出都会反向传播影响到当前 h(t)h^{(t)}h(t) 的梯度，这是一个需要递归求解的计算过程

这一过程的数学推到较为复杂，Pytorch 中已经封装好了现有模块，我们只需要知道 RNN 中的梯度求解来自两个部分（区别于一般的神经网络）即可

RNN长期依赖问题

长期依赖

长期依赖：当前系统的状态，可能依赖很长时间之前的系统状态，比如在上下文文本预测时，当前待预测的词，可能和文本中很长篇幅之前的内容有关，显然依赖关系随着链条长度的增加会逐渐变得脆弱

因此 RNN 面临一个很严重的问题：随着序列长度的增加，RNN 会因为长期依赖现象的存在，导致长期记忆实现，进而使得训练变得困难，很难收敛

长期记忆失效

我们以最原始的 RNN 模型为例（每个时刻依赖于上一时刻隐层的直接输出），且不考虑激活函数，只考虑线性层变换，进行说明：

从 t=0 时刻推导 t=1 时刻：

h(1)=b+Wh(0)+Ux(1)h(1) = b+Wh(0)+Ux(1) h(1)=b+Wh(0)+Ux(1)

从 t=1 时刻推导 t=2 时刻：

h(2)=b+Wh(1)+Ux(2)=b+W[b+Wh(0)+Ux(1)]+Ux(2)=b+Wb+W2h(0)+WUx(1)+Ux(2)h(2)=b+Wh(1)+Ux(2)\\ =b+W[b+Wh(0)+Ux(1)]+Ux(2)\\ =b+Wb+W^2h(0)+WUx(1)+Ux(2) h(2)=b+Wh(1)+Ux(2)=b+W[b+Wh(0)+Ux(1)]+Ux(2)=b+Wb+W2h(0)+WUx(1)+Ux(2)

我们观察到，在 t=2 时刻对应的表达式中，h(0)h(0)h(0) 对应的系数变成 W2W^2W2，而简单推导可知，在 t=T 时刻对应的表达式中，h(0)h(0)h(0) 对应的系数会变成 WTW^TWT，而假设 ∣W∣<1|W| < 1∣W∣<1，则 h(0)h(0)h(0) 的信息在 t=T 时刻的贡献就会微乎其微（此时 WTW^TWT 会变成一个极小的值），这一现象类似于神经网络中的梯度消失，在 RNN 中我们称之为长期记忆失效

而与之相对的，如果 ∣W∣>1|W| > 1∣W∣>1，那么 WTW^TWT 可能会变成一个极大的值，这会导致网络最终的输出结果对输入过于敏感，可能 t=0 时刻的信息发生任何一点微小的改变，都会导致网络最终输出的结果天差地别

对于 RNN 的长期依赖问题，我们提出了各种各样的网络模型变体进行解决，在下一章我们会分别进行详细介绍
