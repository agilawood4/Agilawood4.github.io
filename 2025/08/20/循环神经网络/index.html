循环神经网络

通过阅读本文，您可以大致了解和循环神经网络有关的简单知识和代码实现

序列建模

常见序列数据

序列数据【时间】：不同时间上收集到的数据，描述现象随时间变化的情况（比如股票价格随时间的变化，天气随天数的变化）

序列数据【文本】：由一串有序的文本所组成的序列，每一个文本都有一个确定的位置，且后面的文本可能受到前面文本的影响（自然语言理解，文献分类，情感分类，问答系统）

  • 向量化（tokenization）也称为词元化，由于深度学习模型并不能真正理解文本，而是将其转化为对应的数值，通过数值比较得到规律，这一过程通过向量化实现

序列数据【图像】：有序图像组成的序列，每一张图像都对应一个时间戳，后一帧图像可能受到前一帧图像的影响（运动目标的检测，运动图像的分隔）

序列模型

处理序列数据需要特定的统计工具和网络模型，用于处理和预测序列数据的模型就称为序列模型（Sequence Model）

自回归模型

自回归模型（Autoregressive model）是深度学习中极为经典的序列模型，对于当前时间 t，该模型假设当前输出只和历史输出线性相关，用历史状态 x1 到 xt-1 来预测 xt 的值

隐变量自回归模型

隐变量自回归模型（Latent Autoregressive Model）是对自回归模型的改进版本，为了防止自回归模型中 P(xi | x1x2…xi-1) 表达式中 x1x2…xi-1 的链路太长（依赖的历史状态太多），该模型对历史值进行总结，并进行迭代更新

我们下面提到的循环神经网络（RNN）本质上就是隐变量自回归模型的一种，该模型是用深度神经网络解决序列建模的核心技术之一

文本数据预处理

中文文本需要用分词算法来完成分词；英文文本你需要拼写检查，词干提取和词形还原

文本处理的步骤

常见的文本数据预处理一般由以下几个步骤组成

去除噪声 / 停用词

去除噪声（Text Clearing）指删除文本中不相关或无用的信息（标点符号，停用词等），使得文本中有意义的信息更加突出，提高文本处理的效率

常见的停用词：英文介词，代词，连词等，中文助词，量词，叹词等，这些停用词的去除可以通过建立停用词表，进而通过 Python 自带的正则表达式来实现

词元化

词元化（Tokenization）把输入的文本流，切分成若干子串，每个子串都具有相对完整的含义，方便模型进行后续学习

标准化

标准化（Normalization）由两组变形组成：

  • 词干提取（stemming）：抽取词的词干或词根形式（把词尽可能变短）
  • 词形还原（lemmatization）：把任何形式语言词汇还原为一般形式（复数变单数，动词变原型等）

词表

词表（Vocabulary）即文本中出现的所有词汇所组成的列表，通过查询词表，我们可以大致了解原文中词汇的组成和分布情况，下面给出一个简单的词表建立算法：

  1. 遍历原文本，统计词频
  2. 过滤高频词（可能是停用词）和低频词（没什么代表性），保留中频词（最能体现文本特征）
  3. 为每个词分配一个编号，以之作为索引建立词表

有时候如果我们加载预训练的词向量模型时，词表已经被建立好，不需要我们再手动建立

特征提取

词袋模型

词袋模型（Bag of Words）能够根据词表中单词在句子中出现的次数，将其转化为向量，该算法不考虑单词顺序，只考虑其出现次数

对于上面的例子，我们先把两句话中所有出现的单词列成词表（这里暂时不去除停用词），接着我们用和词表长度相同的向量来分别表示这两句话，向量对应位置的值即为词表在该位置的单词在某句话中出现的次数

比如 football 在 s1 中出现 0 词，在 s2 中出现 1 词，因此 s1[1] = 0, s2[1] = 1，以此类推

N-gram模型

区别于词袋模型，该模型可以通过 N 值的设置，在不同程度上考虑文本中各个单词的位置关系，其假设第 N 个词的出现只与它前面的 N-1 个词相关，该模型师基于概率的判别式语言模型

词嵌入模型

在传统的文本处理中（比如 N-gram），每个单词都看成一个独立的词元，比如“猫”、“狗”、“沙发”，模型在处理这些词元时，无法理解它们之间的任何关系，即对于模型来说：“猫” 和 “狗” 的相似度，与 “猫” 和 “沙发” 的相似度没有任何区别

词嵌入（Word Embedding） 的核心思想在于：将词汇表中的每个词，映射到一个高维空间（通常是几百维）中的一个实数向量（Vector），这个向量不再是随机的，而是包含了这个词的语义信息

  • 语义相似的词，它们的向量在空间中的距离也更近
  • 词与词之间的关系（如 “国王” - “男人” + “女人” ≈ “女王”）可以通过向量的算术运算来体现

简单来说，Embedding 就是一个 “单词 → 向量” 的查找表（Lookup Table），只不过在映射过程中增加了对单词本身含义和词与词之间关系的考虑，从而让模型能够更地学习和处理

循环神经网络

循环神经网络（Recurrent Neural Network - RNN）即为面向序列数据分析的专用网络模型

基本思想

全连接网络结构（Linear）结构上本身没有和顺序相关的处理模块，因为其是一个一个处理输入的，每输入一个 x 得到一个 y，输入之间没有关系，换言之，我们把输入改变顺序后重新给到模型，模型的输出除了顺序对应改变外，本质上不会有太大区别

而对于我们先前提到的文本，时间，视频等数据，顺序对于预测结果的影响其实是很大的，因此我们需要在现有神经网络的隐层（Hidden）中加入和顺序处理有关的网络层级

展开计算图

计算图是形式化一组计算结构的可视化图表，而对于循环神经网络，由于内部涉及到循环结构，为了使得可视化更加直观，我们引入展开计算图，展开计算图回事额的深度网络结构中的参数共享

右图即为左图展开的结果，对于当前的输入 x，它和 h 上一个时间点的输出共同参与当前 h 中的计算，计算结果又作为下一个时间点 h 的输入，也就是 ht 的输入同时来源于 xt 和 ht-1

所谓参数共享，也就是每一次 ht-1 和 xt 计算得到 ht 的过程使用的参数都完全相同，可以看成是左图中的结果复制了若干份，把相同的网络结构在不同的时间戳下多次运行的结果

所谓循环神经网络，就是在隐藏层中增加了一个循环层，用于把上一个时间戳对应的隐层输出，作为下一个时间戳的隐层输入，和当前时间戳下隐层本身的输入 X 共同作用，得到本次隐层的输出

网络结构

我们把隐藏层间进行链接形成循环神经网络，模型因此就具备了记忆能力，每个时刻的输出不仅和当前输入有关，还受到上一个时刻输出的影响

在左图中，三个灰色的水平网络表示的是同一个全连接层对应的不同时刻，橙色的竖直网络则表示隐层之间的联系，t-1 时刻的输出作为 t 时刻的输入，t 时刻的输出作为 t+1 时刻的输入

基础模型

我们把具体的循环网络结构进行展开，如下图所示，对应的权重系数矩阵如下（这里看上去有若干个隐层，实际上是同一个隐层结构对应了不同的时间戳，因此这里的 U V W 参数在不同时间是完全相同的）：

  • U：输入层到隐藏层输入之间的系数矩阵
  • W：上一个时刻隐藏层输出到当前时刻隐藏层输入之间的系数矩阵
  • V：隐藏层输出到网络输出的系数矩阵

各层输入输出的计算公式如下：

  • a(t) = b + Wh(t-1) + Ux(t)：隐藏层输入计算公式
  • h(t) = tanh(a(t))：隐藏层输出激活
  • o(t) = c + Vh(t)：网络输出计算公式
  • y'(t) = softmax(o(t))：网络输出激活，用于和真实的 y 值通过 L 函数进行误差分析

损失函数定义如下：

  • L = ∑L(t) = -∑logp(y(t)|x(1),x(2),...,x(t))

上面这两种模型都有一个共性的问题，那就是无法并行化训练，当前时刻 t 对应的隐层 h 必须要在上一个时刻的隐层训练完成（得到 h(t-1) 或 o(t-1) 以后）才能进行训练，为此我们可以做如下的改进：直接用每个时刻对应的真实值 y 作为下一个时刻隐层的输入，也就是用 y(t-1) 来代替 h(t-1) 和 o(t-1)，这样每个时刻的隐层都可以独立训练，得到的网络结构如下：
