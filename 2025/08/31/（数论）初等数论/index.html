初等数论

  1. 线性筛
  2. 质数分解定理

我们考虑用两遍线性筛，对于一个合数 n，它一定存在至少一对因子，其中一个因子 x 小于根号 n，另一个因子大于根号 n，且可以保证较小的因子是质数（考虑每个合数都一定会被它最小的质因子筛掉），而我们要确定它是合数，没必要找到它的所有因子，只需要找到这个不超过根号 n 的质数因子即可，而当 n 为 109 的时候，取根号也不会超过 50000，也就是说我们只需要预处理出来不超过 50000 的所有质数，那么 n 如果是合数，它就一定有一个质因子在 50000 以内，我们就可以用这个质因子筛掉它

约数问题

基础知识

（一）约数分解定理：一个整数一定可以分解成若干个质数的幂次的乘积的形式，即 x = p1a1 × p2a1 × …… × pkak 这样的形式，其中所有的 pi 都是质数

那么有以下结论：

  1. 这个数的所有因子的个数（包括质因子和非质数因子）为：(a1+1)(a2+1)……(ak+1)
  2. 这个数的所有因子的和（包括质因子和非质数因子）为：(1+p1+p12+……+p1a1)(1+p2+p12+……+p2a2)(1+pk+pk2+……+pkak)

（二）有关约数的几个常识：

  1. int 范围内所有的数字中因数个数最多的那个数，因数不会超过 1600 个
  2. 1 - N 范围内的质数个数不会超过 N / lnN 个
  3. 1 - N 范围内的所有数字的约束个数之和的量级大概在 N logN 个

（三）求解多个数字的因数的两个大方向：

  1. 遍历所有待求解的数字，求解每个数字的因数都有哪些
  2. 遍历所有可能出现的因数，求解当前这个因数都是哪些数字的因数，也就是有哪些数字是这个因数的倍数

因数和倍数可以看成一组对偶运算，我们常常需要转换思维，把求由原数字求因数的问题转化成由因数求倍数的问题，一般可以简化求解的时间复杂度

欧拉函数

基础知识

欧拉函数 phi(N)：1 - N 中和 N 互质的数的个数

欧拉函数公式：phi(N) = N × (1 - 1/p1) × (1 - 1/p2) × (1 - 1/p3) …… (1 - 1/pk)，其中 p1，p2，……pk 是 N 进行质因数分解得到的所有质因子

单点欧拉函数求解

对于一个数的欧拉函数，我们直接用试除法求解即可

线性筛法求欧拉函数

如果 x 是质数，那么 phi(x) 就是 x-1，除了 x 自己，1 - x 的其他数都和 x 互质

如果 x 是合数，它当前表示为 i × prime[ j ]，又分成两种情况

  1. 如果 p = prime[ j ] 是 i 的因子，那么 x 和 i 相比只多了 p 这一个因子，而且这个因子是 i 本来就有的，相当于没有增加新的因子，因此 phi(x) = phi(i) × p
  2. 如果 p = prime[ j ] 是 i 的因子，那么 x 和 i 相比只多了 p 这一个因子，而这个因子是 i 本来没有，因此在 x 的 phi 的表达式中会多一项 (1 - 1/p) 出来，这一项和前面的 p 合并之后会得到 (p - 1)，则此时 phi(x) = phi(i) × (p-1)

其中因为 1 不是质数，则 phi [ 1 ] 的初始化需要根据题目具体而定

10e(qx+r) ≡ 1 mod(C) 和 10e(x) ≡ 1 mod(C) 同时成立，则我们在前者中去掉含有 x 的项，得到 10er ≡ 1 mod(C)，而 r 是 phi(C 关于 x 的余数，因此 r 一定比 x 小，所以这和 x 是最小的合法答案矛盾，因此合法答案一定是 phi(C) 的一个因子

最后我们得到了本例的全部思路：

  1. 首先通过对条件变形，由 L 值得到 C，先判断 C是否为 2 或者 5 的倍数，若是，则 C mod 10 一定不可能余数为 1，直接返回 0
  2. 接着求出 C 对应的欧拉函数
  3. 最后用试除法遍历 phi(C) 的每一个因子，如果有满足条件（10ex ≡ 1 mod(C)）的，则更新答案的最小值

本例中在计算 10ex 的时候，单步乘法的结果会爆掉 longlong，还来不及取模就会溢出，此时我们需要用龟速乘来完成单步的乘法运算，而 10ex 本身又可以用快速幂来完成，因此本例的所有取幂运算都需要通过龟速乘优化的快速幂来实现

中国剩余定理

（一）定理内容：给出若干同余方程如下：

x ≡ a1 (mod m1)

x ≡ a2 (mod m2)

x ≡ a3 (mod m3)

……

x ≡ ak (mod mk)

其中 m1 m2 …… mk 是两两互质的（注意朴素的中国剩余定理必须有这个限定！），现在要求求出满足这些方程约束下的最小的一个 x

具体求法：我们记 M = m1×m2……×mk，Mi = M / mi，ti 为 Mi 关于 mi 的逆元，也就是满足 Mi × ti ≡ 1 (mod mi)，则有结论

同时，因为 x 最大不会超过所有 m 的乘积，所以在运算过程中需要时刻对 M 取模，以及最后我们得到的 x 有可能是负的，这是因为我们解出来的 ti 有可能是负数，所以最后还需要对 x 取一次最小非负解

（二）逆元：a 关于 b 的逆元为 c，则有 a × c ≡ 1 (mod b)，换言之，如果 c 满足上面的同余方程，则称 c 为 a 关于 b 的逆元，因为逆元也是关于同余方程给出的，所以我们同样可以用扩展欧几里得来求解，移项则有

逆元

基础知识

定义：一般来讲，除法是不支持取模运算的，即 (a / b) % c != a % c / b % c，因此我们就不能像加法或者乘法一样，在每一步过程中都进行取余，逆元就是用来解决除法取余计算的，也就是我们把 (a / b) % p 这个式子在对 p 取模的意义下转化成为 a × c % p，那么 c 就称为 b 在模 p 的意义下的逆元（逆元的存在性必须是相对于某个模数而言的，模数不同逆元也因此不同，不谈模数而取逆元没有意义），而当 a 是 p 的整数倍的时候，我们认为此时 a 相对 p 的逆元不存在

费马小定理：已知 p 是质数，a 和 p 互质，则有 ap-1 ≡ 1 mod(p) 成立，当 a < p 的时候，这个式子对于任意的 a 都成立，这是我们求解逆元的基础

逆元的求解：我们这里只给出相对于一个质数作为模数情况下的逆元，也就是 p 为质数的情况，我们对费马小定理进行变形则有，ap-2 ≡ a-1 mod(p)，也就是说此时 a 相对于模 p 意义下的逆元就是 ap-2，因此可以用快速幂的方式求出
