Pytorch入门

通过阅读本文，您可以大致了解 Pytorch 的基本概念与语法

Pytorch 的基本概念：

  1. Tensor 张量（标量 -> 向量 -> 矩阵 -> 张量）
  2. Variable 变量
  3. nn.Module 网络结构

本章中，我们主要学习 Tensor 张量这一部分的概念与相关操作

Tensor张量

相比于标量，向量，矩阵而言，标量具有更好的泛化能力，可以描述三维及以上的概念，在 Pytorch 中可以代替 Numpy 数组发挥更好的作用

常见的 Tensor 类型有 float，int，bool 等几种，他们都以torch.*作为前缀开头，我们需要了解的大致也只有浮点，整型，布尔这三种常用的类型

Tensor算术运算

四则运算

Tensor 的四则运算要求参与运算的两个张量维度必须相同，运算时会对每个 Tensor 各个对应位置的元素，做相应的四则运算，运算的结果会返回一个相同维度的新张量

四则运算中还有一类*-()形式的运算，这里可以类比 C++ 中的 +=，-=，*=，/= 等操作，这些运算不会返回新的张量，而是会在原张量上进行运算，直接修改原张量的值

矩阵乘法

对于二维 Tensor 的矩阵乘法，我们就当成矩阵相乘即可，有三种方式进行实现，分别是torch.matmul()，torch.mm()，@，他们都会返回一个新的矩阵，参与运算的两个矩阵必须是 n × m 和 m × k，运算结果为 n × k 矩阵

对于高维 Tensor（dim > 2），定义其矩阵乘法为：仅在其最后的两个维度上进行矩阵乘法，其他前面的维度必须保持一致，且此时只有torch.matmul()这一个函数可以实现

所有的矩阵运算都没有*_()的下划线形式，因为矩阵乘法的结果可能和参与运算的两个矩阵形状都不相同，没法返回到原矩阵上

幂运算

Tensor 的幂运算可以实现对 Tensor 中的每个元素取相应的幂

还有一种常见的取幂方式，即 e^n^ 形式，也就是对一个 Tensor 取以 e 为底的指数，该运算会对 Tensor 中的每一个元素，把他变成自己以 e 为底的指数

开方运算

Tensor 的开方运算会对其中的每个元素进行开平方操作

对数运算

Tensor 的对数运算可以对 Tensor 中的每一个元素取对数，常见的有以 e 为底log()，以 2 为底log2()，以 10 为底log10()

广播机制

广播机制：两个张量在进行运算的时候，如果张量形状不同，张量参数可以自动扩展为相同大小

广播机制需要满足的条件：

  1. 每个张量都必须至少有一个维度
  2. 满足右对齐原则
  3. 对齐+填充后，对应位置要么维度相同，要么有一个为 1，才能进行运算

比如我们有torch.rand(2, 1, 1) + torch.rand(3)，这两个张量形状不同，系统会尝试进行广播机制，对齐方式为右对齐

从右向左看，3 和 2对齐，那么张量 2 缺少的维度都会默认被填上 1，也就是变成

此时我们依次比较每一组位置，要么这两个位置上数字相同，要么这两个位置上有一个位置是 1，才可以进行运算，上例是可以进行运算的

在运算过程中，为 1 的位置会进行维度扩展：

  1. 第一维：T2 为 1，T1 为 2，T2 会进行扩展，从（1,1,3）扩展为（2,1,3）
  2. 第二维：T1 和 T2 都是 1，不需要扩展
  3. 第三维：T1 为 1，T2 为 3，T1 会进行扩展，从（2,1,1）扩展为（2,1,3）

因此最后广播后相加得到的张量形状为（2,1,3）

Tensor边界判定

torch 支持对某个 Tensor 进行有界性 / 无界性 / 是否含有非法数据的判定，返回结果是一个与输入 Tensor 相同形状的 mask 张量，对应位置为输入 Tensor 中每个位置的值是否为 finite / inf / nan 的标记

这三个函数通常用于数据清洗，用于筛选出不合法的数据，供后续操作

torch.mask_select()

我们通过生成一个和原张量形状相同，且对应位置只有 True 和 False 两种值的 mask 张量，借助这个张量完成数据筛选，因为两者位置是一一对应的，则 True 位置的值会被保留，而 False 位置的值会被丢弃

变形操作

变形操作用于改变张量的形状，Pytorch 中常见的张量变形函数如下：

torch.reshape()

torch 中的 reshape 和 numpy 的功能完全相同，可以直接改变 Tensor 的形状

torch.t()

该函数仅可以用于矩阵（二维张量）的转置操作，也就是行列互换

该函数可以用于交换高维张量指定的两个维度，因此该函数也可以用于矩阵转置

该函数用于删除张量中所有为 1 的维度，按序保留剩余的维度

torch.flip()
