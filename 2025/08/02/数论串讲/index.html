初等数论

通过阅读本文，您可以大致了解初等数论的内容，数论题目的输入输出范围一般都比较大，因此本章会涉及较多卡常、取模有关的内容

基本概念

整除：对于两个整数 b 和 a（b <= a），如果 a = k × b，那么我们记作 b | a

带余除法：对于两个整数 a 和 b，一定存在一个整数 r ∈ [0, b) 和整数 k，使得 a = r + k × b

算术基本定理 / 唯一分解定理：对于任何一个大于 1 的整数 x，都可以表示成它的若干个质因子的乘积，这个表示法是唯一的

试除法进行唯一分解：该算法可以在 O(√n) 复杂度内完成对 n 的素数唯一分解

扩展欧几里得

扩展欧几里得用来求解不定方程 ax + by = c 的解（x 和 y），在数论中，一般用来求解同余方程，因为同余方程可以变形成为不定方程，对于 ax ≡ b (mod p)，我们把该式写成带余除法的形式，也就是 ax = pk + b，我们移项即可得到 ax + pk = b，因此可以转化成不定方程的形式进行求解

需要注意的是，ax + by = c 有解的前提是 gcd(a, b) | c，也就是要求 c 必须是 a 和 b 公约数的倍数，当 a 和 b 互质的时候，方程是一定有解的，在方程有解的前提下，我们解的原始方程是 ax + by = gcd(a, b) 对应的 x 和 y，根据 c 和 gcd(a, b) 的倍数关系，我们得到的 x 和 y 也要进行对应的缩放，才是真正的解

威尔逊定理

判定数字 p 是质数，等价于以下式子成立： (p-1)! ≡ p-1 (mod p)，也可以写成 (p-1)! ≡ -1 (mod p) 的形式

欧拉函数

（1）我们一般用 phi(x) 表示 x 的欧拉函数：

phi(x) 表示所有小于 x 的数中，与 x 互质的数字的个数之和，特别规定 phi(1) = 1

（2）欧拉函数的一般表示：

phi(x) = x × (1 - 1/p1) × (1 - 1/p2) ×……× (1 - 1/pk)，其中 p1，p2，pk 等为 [1, x) 内与 x 互质的数字

（3）欧拉函数的性质：

  1. 如果 a 和 b 互质，那么 phi(a × b) = phi(a) × phi(b)，该性质称为积性
  2. 对于整数 n 而言，它的所有因子 d 的欧拉函数之和恰好为 n 本身，也就是 sum( phi(d) )d|n = n
  3. 对于任意质数 p 而言，phi(p^k^) = p^k^ - p^k-1^，简单证明：因为 p 是质数，那么和 p^k^ 不互质且小于它的数只有 p，2p，……，p^k-1^p，恰好有 p^k-1^ 个这么多

（4）利用线性筛求 1 ~ n 范围内所有的欧拉函数，对于一个数字 i 而言，设 p 是它的最小质因子，有三种情况：

  1. i 为质数，phi(i) = i - 1
  2. p 和 i/p 互质，利用积性则有 phi(i) = phi( i/p ) × phi(p)
  3. p 是 i/p 的质因子，phi(i) = phi( i/p ) × p，简单证明，我们令 i = p^k^q，其中 k 极大使得 p 和 q 互质，那么 phi(i) = phi( p^k^q ) = phi(p^k^) × phi(q)，其中 phi(p^k^) = p^k^ - p^k-1^，我们提出一个 p 来则有，phi(i) = p × (p^k-1^ - p^k-2^) × phi(q) = phi(p^k-1^) × phi(q) × p = phi( i/p ) × p，推导的关键是提出一个 p 以后，逆用性质三即可

逆元

（1）逆元：逆元用来实现同余意义下的整除操作

在实数域中，x × 1/x = 1，我们说一个数除以 x，就等价于乘以 1/x

而在模 p 同余的意义下，对于每个 x，如果能找到一个整数 x^-1^ 在 [1, p) 之间，使得任何数除以 x 都等价于乘以 x^-1^，那么 x^-1^ 就称为 x 在模 p 意义下的逆元

显然逆元需要满足性质 x × x^-1^ ≡ 1(mod p)，比如在 mod 7 意义下，2 的逆元是 4，因为 2 × 4 = 8，8 % 7 = 1 恰好成立

（2）逆元存在性定理：

x 在模 p 同余下存在逆元，当且仅当 gcd(x, p) = 1，也就是 x 和 p 互质，比较常见的情况，就是模数 p 为质数的时候，此时 [1, p-1] 中的数字在模 p 意义下一定存在逆元，其中 0 没有逆元

（3）逆元的性质：

  1. 模 p 同余下，一个整数 x 的逆元如果存在，则必定唯一，简单证明：使用反证法，假设 x 存在两个不同逆元 i1 和 i2，那么均满足 xi1 = 1 mod p，xi2 = 1 mod p，由同余传递性知，xi1 = xi2 mod p，两边同时左乘 i1，则有 (i1x)i1 = (i1x)i2 = mod p，那么也就是 i1 = i2 mod p，而 i1 和 i2 均小于 p，此时 mod p 同余等价于 i1 = i2
  2. 模 p 同余下，[1, p-1] 范围内的逆元互不相同，也就是 1^-1^，2^-1^，……p-1^-1^ 都是互不相同的，简单证明：使用反证法，假设 x 和 y 有相同的逆元 i，则有 xi = yi mod p，两边同时右乘 i^-1^，则有 x = y mod p，则 x = y 矛盾
  3. 一个数的逆元的逆元等同于这个数本身

（4）扩展欧几里得求逆元：

因为 x × x^-1^ ≡ 1(mod p) 本身就满足同余方程的形式，我们将其展开则为 x × x^-1^ = k × p + 1，简单变形则有 x × x^-1^ + k × p = 1，其中 x 和 p 已知，分别看成方程中的 a 和 b，x^-1^ 和 k 则是我们要求的系数 x 和 y，也就是同余方程的解，最后我们只需要把求得的 x^-1^ 模进 [1, p) 范围内即可

同时，对于同余方程 ax ≡ 1(mod p) 的最小整数解 x 就是 a 在模 p 意义下的逆元，因为我们可以首先用扩展欧几里得求出 a^-1^ 的一个值，然后把他模进 [1, p) 范围内，根据逆元的唯一性定理，该范围内有且仅有一个逆元，因此逆元 a^-1^ 就是我们要求的方程最小整数解

欧拉定理

欧拉定理：对于任意正整数 a 和 m，如果 a 和 m 互质，则有 a^phi(m)^ ≡ 1 (mod m)

费马小定理

其中当 m 为质数的时候，该定理退化为费马小定理，因为 phi(m) 此时为 m-1，因此则有：当 m 为质数时， a^m-1^ ≡ 1 (mod m)

因此我们可以用费马小定理来求逆元，当模数 p 本身是质数时，则有 a^p-1^ ≡ 1 (mod p) 成立，而 a^p-1^ = a × a^p-2^，因此此时 a 在模质数 p 意义下的逆元即为 a^p-2^，这种情况下，我们可以用快速幂求逆元

中国剩余定理

中国剩余定理：给定 n 个同余方程，满足 x ≡ ai (mod mi)，其中 m 两两互质，i ∈ range(1, n)，求有关 x 的通解，该问题称为 Crt 问题，如果不要求 m 两两互质的限定，则称为 exCrt 问题

该问题可以通过构造法求解：记 M = ∏mi，Mi = M / mi，ti 为 Mi 在模 mi 意义下的逆元，则方程的唯一通解可以表示为：x ≡ ∑ai × ti × Mi (mod M)

这里 ai 是每一组的余数，ti 是 M / mi 对应模 mi 的逆元，Mi 是 M / mi 的结果，最终答案为这三部分相乘后，对 i ∈ range(1, n) 遍历求和即可
