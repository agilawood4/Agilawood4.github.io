AI 中数据的本质

灰度图片的本质

灰度图会被转化成二维矩阵，A×B 的灰度图表示图像由 A×B 个像素点所组成，其中每个像素点对应 A 行 B 列矩阵中的一个格点，其中每个格点上数值的取值范围为 [0, 255]，取值越接近于 0，则表示该像素点偏白色；取值越接近 1，则表示该像素点偏黑色

彩色图片的本质

彩色图会被转化为三维矩阵，其中第一个维度表示 RGB 通道数，通道数一般为 3（R - 红色，G - 绿色，B - 蓝色），后两个维度和灰度图相同，A×B 的彩色图表示每个通道所对应的二维矩阵都由 A×B 个像素点组成，对应一个 A 行 B 列的矩阵

每个通道矩阵格点的取值范围也是 [0, 255]，其中越接近 0 表示该格点偏白色，越接近 255 表示该格点偏彩色（在 R G B 三种颜色中取对应的颜色），直观来讲，彩色图可以看成是由三张 “灰度图” 加权叠加得到的结果

字符的本质

我们通常用到的字符一般为英文字母，中文汉字，其他国家常用语言字符，标点符号，阿拉伯数字等，具体有以下四种表示方式：

  • 扩展 ASCII 码：传统的 ASCII 码只能表示 128 个字符，主要用于英文字符，其余国家将其扩展到 256 或更多个字符，如中国的 GB2312
  • Unicode 和 UTF-8：Unicode 是一种包含全世界所有字符的编码系统，每个字符都会映射得到一个唯一的数值，UTF-8 是 Unicode 的一种具体实现，它是一种变长的编码方式，用 1 到 4 个字节来表示一个字符
  • One-hot 编码：在早期机器学习中，每个汉字被转化为一个只有一个元素为 1，其余元素均为 0 的向量，向量的长度等于汉字的总数量，为 1 的位置表示该汉字的索引
  • 词嵌入：把每个字或词映射到一个高维向量，这个向量就称为词嵌入，可以捕捉到字或者词的语义信息

表示学习和嵌入

表示学习和嵌入

表示学习（Representation Learning）和嵌入（Embedding）是密切相关的概念，它们被视为在不同领域下对同一概念的描述

表示学习：通过学习算法自动地从原始数据中找到一种表示形式或特征表示，这种表示形式能够更好地表达数据的重要特征和核心结构

嵌入：表示学习的一种形式，通常用于把高维数据映射到低维空间中的表示形式，嵌入可以是词嵌入，图像嵌入，图嵌入等

  • 在自然语言处理中，词嵌入把词语映射到低维向量空间，以捕捉词语之间的语义和句法关系
  • 在图像处理中，图像嵌入把图像映射到低维向量空间，以表示图像的视觉特征

词嵌入和语言模型的关系

Word Embedding：词嵌入通常用于生成词的向量表示，这通常是一个静态的过程，即一旦训练完成，每个词的向量表示就确定下来，词嵌入的主要目标是捕获单词或短语的语义和语法信息，并且把这些信息用向量的形式表示出来，无法理解上下文信息，即相同的词在不同的上下文中可能有不同的含义，但词嵌入无法区分这些含义

Language Model：语言模型是用于预测词序列的概率模型，这通常是一个动态的过程，会根据输入的上下文进行变化，语言模型的主要目标是理解和生成文本，包括对上下文的理解，词的预测，语句生成等

直观来讲，词嵌入可以看成是语言模型的一部分或者输入，语言模型使用词嵌入得到的向量化信息，进行更深层次的语义理解和文本生成，现在也存在一些更加先进的模型，如 BERT，GPT 等，它们可以进行上下文相关的词嵌入，一定程度上弥补了传统词嵌入模型的不足
