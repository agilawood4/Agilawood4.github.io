LLM LLM（Large Language Model，大语言模型）是指在海量无标注文本数据上进行预训练得到的大语言模型，通过学习大规模语料中的语言规律，模型能够理解、生成和推理自然语言文本，典型代表包括 GPT 系列，LLaMA 系列，Qwen 系列 这里之所以使用无标注数据即可进行预训练，是因为 LLM 的预训练过程通常是根据上下文预测下一个 token，因此给定原始语料即可训练 大模型的大主要体现在以下三个方面： 参数量大，模型通常拥有数十亿到上万亿的参数（1B 即一亿参数） 预训练数据量大：涵盖多语言、多领域的庞大语料 能力强大：具备跨任务的泛化与生成能力，能够在问答、摘要、翻译、编程等多种场景下展现出类似人类的语言理解和表达水平
